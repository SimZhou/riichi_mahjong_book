# 三色

曾几何时，被视为手役花形的三色同顺，  
到了现代麻将里，已经成了配角。

不过它毕竟还是相对容易做成的 2 番役，  
实战中也经常会去狙击。  
关于三色有很多偏门手筋，  
但真正会在实战中经常用到的其实不多。

比起“刻意去做出三色”，  
我觉得三色同顺更重要的，其实是“不要漏看三色”。  
先把最基本的手筋记住就够了。

## 1. 滑移

为了让牌变成三色，  
把万子、筒子、索子的各组面子朝同一个数字带去滑动，  
就是三色的基本。
