# 断幺九

断幺九最大的卖点，说到底就是：**副露后也能和牌。**

尤其是在赤牌很多的自由桌规则里，食断的威力非常大。`断幺九 + 宝牌 + 赤牌` 的高打点副露手非常常见，甚至常常会出现为了防一手食断而被迫退守的场面。

从现代规则感来看，断幺九几乎已经可以算是最强势的常用手役之一。

## 1. 要按会变成断幺九的方向打牌

断幺九本来就容易做，又很容易和其它手役复合，因此实战里必须随时考虑断幺九的可能性。

## 3. 从副露角度考虑

### 例 5

<span class="b">例 5</span><img src="../images/001.gif" width="472" height="503" />

这手若单论最大受入，最宽的切法是摸切 `2s`。

但实战中，这题选择的是切 `6m`。

原因在于：这是南四局、自己是亲家、而且当前排在第四，必须想办法先和一手。既然局况要求的是“无论如何都要和出来”，那就必须把食断副露路线放进判断里。

若选择切 `2s`，之后即使碰到 `6m` 或 `2p`，也可能变成片和；而切 `6m` 后，这些副露路线都会自然得多。

也就是说，这题不是在追求绝对最大受入，而是在追求“更容易鸣、也更容易和”的实战路线。

2. 但若局况要求你尽快鸣牌和出，食断的副露价值就可能压过那一点点眼前宽度

## 这一页的结论

1. 断幺九最大的强点，是副露后也能和牌，而且很容易和赤牌、宝牌复合
2. 幺九牌如果只是边缘拖牌，应主动考虑把它们换成中张路线
3. 当断幺九一旦确定，就会明显提升副露效率时，可以主动把它锁定
4. 断幺九的价值不只是一番，还包括“获得副露资格”
5. 特定局况下，为了更快副露和出，可以接受小幅牺牲表面手广
