# 断幺九

断幺九最大的卖点，说到底就只有一个：  
**就算副露了也能和牌。**

特别是在现在那种<span class="b">赤牌一大堆</span>的自由场里，食断的威力极大。  
“断幺九宝牌赤赤，8000！”这种事经常发生。  
被这种手打中当然很烦，甚至还会出现那种很没出息的“为了防一手食断而弃和”的局面。  
有些店里赤牌还带祝仪，所以<span class="b">“食断正在变成最强手役”</span>。

## 1. 朝着能做成断幺九的方向打

断幺九既容易做，又容易和其他役复合。  
所以打牌时，必须时刻去想它的可能性。
