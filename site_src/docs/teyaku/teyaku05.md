# 断幺九

断幺九之所以强，不是因为它番数高，而是因为它实在太实用：

1. 容易做
2. 能副露
3. 和平和、三色、赤宝牌都容易复合
4. 在赤牌多、节奏快的环境里尤其高效

因此这页的重点不是解释“断幺九是什么”，而是：**哪些手牌该主动往断幺九靠，哪些又不能为了断幺九把速度打坏。**

## 第一部分：看到可行空间，就要主动往断幺九方向整理

断幺九是最值得提前放进判断里的高频役之一。

因为只要手牌允许，中张化通常同时带来两件事：

1. 役种成立机会增加
2. 牌形本身也更灵活

### 例 1：老头对子经常就是第一拆点

第一例里，`11m` 对子看似能当头，但它们对断幺九是硬障碍。此时拆掉这组老头对子，等于同时打开：

1. 断幺九
2. 平和
3. 三色等更高质量的中张路线

这种局面下，拆老头对子不是“为了做役牺牲牌理”，而是牌理和手役方向本来就一致。

### 例 2：把无效幺九换成中张，是典型的断幺九思路

第二例更直接。把 `9p` 换成 `6p`，表面上只是中间调张，实质上是在提前为之后摸 `2m`、`4m` 之类中张改良留出口。

实战里要形成的习惯是：

1. 当幺九牌只是拖在边缘，没有关键结构职责时
2. 要优先考虑能不能把它换成中张位

## 第二部分：某些局面里，应当让断幺九“确定下来”

断幺九不只是可以“顺便有”，在一些牌姿里还值得主动把它锁定。

### 例 3：面子超载时，直接拆掉会破坏断幺九的边张块

第三例是典型面子超载题。

如果继续保留 `23s`，那一旦摸进 `1s`，最终很可能只剩立直而无额外手役；而直接拆掉这组带幺九的块，则断幺九当场确定，还顺带保留后续副露空间。

这页的重要原则之一就是：

**当断幺九一旦锁定，就能显著提升副露与打点价值时，可以主动让它确定下来。**

### 例 4：只看眼前受入，不如看“确定役 + 可鸣牌”

第四例里，切 `6m` 可能眼前更宽，但普通实战更应先切 `9m`。因为一旦 `9m` 走掉，断幺九立刻成立，之后：

1. 可以吃碰
2. 还有三色空间
3. 整体打法更主动

这也说明断幺九和许多手役不同。它常常不是纯打点加成，而是**行动权限加成**。

## 第三部分：副露视角下，断幺九更强

### 例 5：为了“能鸣且能和”，可以牺牲一点表面手广

第五例是整页里最贴近实战的一题。

如果只按最大受入，切 `2s` 似乎更自然；但放到“亲家、南四、必须先和”的局面里，切 `6m` 更好，因为这样一来 `6m`、`2p` 等牌的碰吃都会更自然，断幺九副露路线被提前打通。

也就是说：

1. 常规牌理里手广更重要
2. 但当局面要求“尽快鸣牌和出”时，断幺九的副露价值会压过一点点眼前宽度

## 这一页的结论

1. 断幺九是高频、易做、可副露、复合性强的核心手役
2. 幺九牌若只是边缘拖牌，应主动考虑把它们换成中张路线
3. 当断幺九一旦确定，就能显著提升副露效率时，可以主动锁定它
4. 断幺九的重要价值不只是多一番，更是让你更早获得“能鸣牌”的资格
5. 特定局况下，为了更快副露和出，可以接受小幅牺牲表面手广
