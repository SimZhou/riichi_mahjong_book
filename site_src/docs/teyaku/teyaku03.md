# 役牌

役牌（番牌）的优点，就在于几乎没有什么限制。  
断幺九和平和当然也是容易和牌的役，但它们会对 14 张牌全部施加限制。

只靠 3 张牌，就能满足麻将的一番缚，这样的役没有几个。  
而且它还能鸣。

实际上，麻将里最常出现的手役之一，就是番牌役。  
如果能更好地处理字牌，并和鸣牌结合起来，和牌率就会提高。

## 1. 保留番牌

起手之后，大家通常都是按“客风 -> 役牌 -> 不要的数牌”的顺序切牌。  
但如果想提高番牌役的利用率，只要条件允许，就应该尽量改成“不要的数牌 -> 役牌”的切牌顺序。  
当然，数牌的有效牌始终更多，所以前提是你真的得认准哪些数牌确实没用。

因为副露时，断幺九更容易鸣，而且也更容易和其他役复合，比如平和。

## 3. 关于鸣牌

基本上，役牌可以一鸣就碰。  
常有人说“役牌要等二鸣”，但我觉得这是比较老的战术了。

第一，哪怕只是便宜和牌，只要和掉，就等于掐掉了别人的机会，  
它的价值不止牌面点数而已。

第二，如果是赤牌麻将，就算已经鸣了，  
后面摸进赤牌，手牌还是可能变高。

而最重要的是，现在大多数人都在强调速度。  
等二鸣，很多时候已经来不及了。

实际上，就连上级者也有很多一鸣的场面。  
不要再被“看起来不美”“还该再贪手役”这种旧价值观绑住。  
<span class="b">现代麻将里，一鸣就是基本。</span>

不过，像例 5 这种门清已经很够打的手，  
或者像例 6 这种手牌实在太散的手，  
就不该从第一张起立刻鸣。
