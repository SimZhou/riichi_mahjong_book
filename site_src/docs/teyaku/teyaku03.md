# 役牌

役牌是最基础、也最实用的手役之一。

它的强大不在于番数高，而在于限制极少：

1. 只要凑出三张就有役
2. 副露后依然成立
3. 对整手其它十三张牌几乎没有额外束缚

从实战角度看，这几乎是最适合和速度结合的一类役。

## 为什么役牌这么强

平和、断幺九当然也都是高频役，但它们会对整手牌产生持续限制；而役牌只要求你处理好那组三张字牌，其余部分依旧可以按最快的牌形去做。

也正因为如此，役牌一直是麻将里最常见的役种之一。谁更会处理字牌、谁更会判断什么时候保留役牌，对整体和牌率影响很大。

## 第一部分：起手阶段，尽量先切真正没用的数牌

大多数人起手会按“客风 -> 役牌 -> 弱数牌”的顺序切。但如果你想提高役牌手的利用率，就不能总是这么机械。

更好的顺序通常是：

1. 先识别哪些数牌是真的没用
2. 只要不明显损害整体牌形，就优先保留单张役牌多看几巡

### 例 1：能先切边张弱牌，就别急着拆役牌

第一例里，`9s` 的独立价值明显偏低，因此完全可以先从它下手，而不是立刻把 `白`、`发` 当成无用字牌扔掉。

这背后的逻辑不是“役牌一定比数牌强”，而是：

1. 你手里已经有足够多更有工作能力的数牌块
2. 那张边张孤张又难延伸
3. 那就给役牌多一轮重叠机会

### 例 2：当块数已经够了，单张役牌的抽奖价值会上升

第二例更明显。

手里已经有不少稳定搭子，这时再多留一张普通单牌，边际收益没那么高；反而是保留 `发`，等待它重叠成役牌刻子，更可能带来实际增益。

这类题的关键判断是：

1. 我现在缺不缺块
2. 如果不缺，单张役牌的潜在收益是否高于那张单牌数牌

## 第二部分：役牌和断幺九怎么选

从整体效率上看，断幺九通常比役牌更占优。因为：

1. 断幺九更容易和其它役复合
2. 副露空间更广
3. 对牌河的适应性也更强

但这不等于“有役牌对子时永远拆役牌”。关键在于：**断幺九是否已经足够稳定。**

### 例 3：断幺九路线已经稳，就拆役牌

第三例里，断幺九方向已经成型得很清楚，这时继续抱着 `发` 不放，只会拖慢速度，因此应当果断拆役牌，直接走断幺九路线。

### 例 4：断幺九还没定下来时，确定役比不确定役更值钱

第四例则相反。虽然也有断幺九的影子，但还没有稳定到可以放心全押。这时保留 `发` 的对子，就是保留一个确定能成立的役。

这一页在这里给出的结论非常实用：

**番数相同时，确定役通常优于不确定役。**

## 第三部分：役牌该不该一鸣就鸣

原页这部分非常鲜明：在现代麻将里，**一鸣役牌是常规，不是羞耻。**

过去常有人说役牌要等二鸣，想再贪更高一点的手役或更完整的牌形。但在现在的快节奏环境里，这种想法经常来不及。

一鸣的价值主要来自三点：

1. 先和先赚，还能打断别人机会
2. 有红牌、宝牌时，便宜副露手也可能很值钱
3. 现代对局整体更重视速度，等到二鸣常常已经错过窗口

### 什么时候不急着一鸣

当然也不是所有役牌都该见牌就碰。

原页给了两个保留场景：

1. 手牌本身已经很完整，闭门推进明显更强
2. 手牌实在太散，就算先碰了也未必来得及做成

换句话说，役牌一鸣是默认值，但不是死规则。你仍然要问：

1. 碰了以后，这手是不是更快
2. 不碰的话，闭门路线是不是明显更强

## 这一页的结论

1. 役牌强在限制少、可副露、成役成本低
2. 起手阶段应尽量先切真正没用的数牌，给单张役牌更多重叠机会
3. 断幺九通常整体更强，但若断幺九尚未稳定，确定役牌往往更值得保留
4. 现代麻将里，役牌一鸣是基本思路
5. 只有在手牌已经很完整，或手牌过散导致先碰也没用时，才更适合暂缓鸣牌
