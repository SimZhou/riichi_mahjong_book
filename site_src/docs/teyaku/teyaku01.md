# 手役思路（1）

麻将的魅力之一，就是做役。  
无论是役满，还是那些难度高、很稀有的役，只要做出来，谁都会高兴。

比如这手牌，不能因为能看见 `234` 三色，就去切 <img src="../hai/sou7m.gif" width="19" height="26" class="hai" />。

只要摸进 <img src="../hai/man1m.gif" width="19" height="26" class="hai" /> 或 <img src="../hai/pin5m.gif" width="19" height="26" class="hai" />，三色就会散掉。

切这种不上不下的 <img src="../hai/man5m.gif" width="19" height="26" class="hai" />，也不太好。  
还不如拆掉索子的坎张，去看断幺九平和更划算。

初学者常说自己不会做役，  
但我反而觉得，更常见的是“做役做过头”而失败。  
尤其很多人会硬往混一色、对对和上靠。  
副露了三次还没听牌，这种情况并不少见。

麻将首先讲的是“先有形”。  
先去考虑“怎样做出四面子一雀头”，  
然后只在不牺牲速度的范围内去做役。

麻将里大约有将近 `40` 种役，  
但真正经常出现的，也就三分之一左右。  
剩下那些，几乎可以当成<span class="b">不太实用的役</span>。  
其中最典型的就是三杠子。

这种役的难度和回报完全不成比例。  
更糟的是，还可能因为开杠多翻出几张宝牌，  
把别人的立直一番手直接送成倍满。  
这种吃亏的役，根本没必要勉强去追。

实际来看一下各种手役的出现率。  
网络麻将站点 [天凤](http://tenhou.net/) 上，就有统计数据。

麻将的役虽然很多，但真正常做的役其实很有限。  
这张表里，出现在 `B` 级以上的手役，才是真正重要、需要练会的。  
至于那些出现率不到 `0.5%` 的稀有役，基本可以忽略。

### 理论小结

<span class="green">手役的难度和点数，并不一定是平衡的。<br />
没有必要勉强去追那些难度很高的手役。</span>
