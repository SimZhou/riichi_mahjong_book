# 麻将役种

这一栏开始进入“做成以后算什么番”的部分，也就是手役。

但第三章的重点，不是让你去背一大堆罕见役，而是先建立一条更实战的顺序：

1. 先把手牌做快
2. 再判断当前牌形自然会长成什么役
3. 只有在不明显拖慢速度时，才主动让手役介入

换句话说，手役不是脱离牌形独立存在的愿望清单，而是建立在速度、进张和副露条件上的实际选择。

## 本栏内容

1. [手役思路（1）](./teyaku01.md)
: 先把“不要为稀有役上头”这件事讲清楚，建立役种难度、收益与速度之间的基本尺度。

2. [手役思路（2）](./teyaku02.md)
: 继续说明手役通常是顺势形成的，不该预设终局强行锁死路线，同时引入“手役的价值之一是让你能鸣牌”。

3. [役牌](./teyaku03.md)
: 从最常见也最实用的役种开始，讨论役牌为什么灵活、什么时候应当保留、什么时候该一鸣就鸣。

4. [平和](./teyaku04.md)
: 讲清平和为什么本质上就是朝两面和顺子方向做牌，以及役牌对子为何常常该拆。

5. [断幺九](./teyaku05.md)
: 说明断幺九为何既是高频手役，也是副露加速的关键工具。

6. [染手（1）](./teyaku06.md)
: 先处理混一色/清一色的起手判断，区分“该染的好材料”和“不该强染的普通好牌”。

7. [染手（2）](./teyaku07.md)
: 继续讲混一色手筋，重点是留转身空间的切法、先打伪装以及复杂同色待牌的处理。

8. [三色](./teyaku08.md)
: 归纳三色最常用的滑移、天秤和余牌利用，不让高频两番役从眼前溜走。

9. [对子手（1）](./teyaku09.md)
: 先解决七对子与对对和的分叉，按对子数量、鸣牌难度和打点条件来判断。

10. [对子手（2）](./teyaku10.md)
: 补完七对子和面子手的分岔，并总结七对子单骑待该如何按场况去挑。
