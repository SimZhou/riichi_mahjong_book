# 安全牌

例如，面对下面这个立直，  
有哪些牌切出去以后**绝对不会被荣和**？

<img src="../images/27.gif" width="396" height="420" />

根据振听规则，**只要对方自己打过的牌，就不能荣和。**  
所以对手自己河里的牌，全部都是安全牌。

在这张图里，符合这一点的有：  
![2m](../hai/man2.gif)![8m](../hai/man8.gif)![9m](../hai/man9.gif)![1p](../hai/pin1.gif)![4p](../hai/pin4.gif)![9p](../hai/pin9.gif)![1s](../hai/sou1.gif)![4s](../hai/sou4.gif)![9s](../hai/sou9.gif)![南](../hai/nan.gif)![白](../hai/haku.gif)![中](../hai/tyun.gif)

另外，立直以后如果和牌张被别人打出来却见逃了，那么之后也不能再荣和那张牌。  
所以图里对家打过的 ![7p](../hai/pin7.gif)，以及下家打过的 ![5m](../hai/man5.gif)，也都不会放铳。

像这样，由于振听禁止荣和而**明确绝对不会被荣和**的牌，就叫做**现物**。

---

除了现物之外，还有一种 `100%` 安全的牌。

那就是现在刚摸进来的 ![发](../hai/hatu.gif)。

因为这张牌在场上已经出现了三张，所以它已经不可能再构成：

1. 双碰待
2. 单骑待

同时这里也不可能是国士无双，所以它就是一张绝对不会被荣和的**完全安全牌**。

---

## 完全安全牌的四种情况

完全安全牌大致有下面四类：

1. 其他三家都各自打过一张的牌（也就是三家的现物）
2. 上家刚刚打出的牌（同样也是三家的现物）
3. 在国士无双听牌不可能成立时，场上第四张字牌
4. 完全无机会牌

最后这一条，会在后面的  [壁牌](mamori05.md)里再详细解释。

---

在这个例子里，安全牌已经很充足了。只要你想避免放铳，其实很容易就能做到。

但问题在于，实战里并不是每次都有这么多安全牌。  
正因为如此，后面才需要继续学习“筋”和“壁”。
