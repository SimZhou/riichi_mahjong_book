# 安全牌

防守真正开始落地时，首先要解决的就是一个很具体的问题：

**面对立直，我现在有哪些牌一定不会放铳。**

这页讲的，就是防守里最基本、也最重要的一类牌：安全牌。

## 第一部分：现物，是最基础的绝对安全牌

### `27.gif` 这张图的重点，是先学会从河里直接找“绝对不会荣和”的牌

原页开头先给了一个立直局面图。

<img src="../images/27.gif" width="396" height="420" />

这张图要说明的，是麻将最基本的防守原理之一：

1. 若对手自己已经打过一张牌
2. 那么根据振听规则，他不能再荣和这张牌
3. 因此这张牌对他来说就是绝对安全的

这类牌就叫现物。

所以在图示局面里，对手自己河里的牌，例如 `2m`、`8m`、`9m`、`1p`、`4p`、`9p`、`1s`、`4s`、`9s`、`南`、`白`、`中` 等，都是现物。

原页还补充了另一个容易忽略的点：立直后若有人打出和牌张而被见逃，这张牌从那一刻开始也变成了现物。因此像图里别家在立直后打过的 `7p`、`5m` 之类，也同样安全。

## 第二部分：有些牌不仅是现物，还是完全安全牌

### 场上已经确定“不可能构成任何荣和形”的牌，安全度比普通现物还更强

原页在现物之外，又举了一个更强的概念：完全安全牌。

例如图里新摸到的 `发`，因为场上已经出现了三张。此时可以直接确定：

1. 不可能是双碰待
2. 不可能是单骑待
3. 国士无双也已经被排除

那么这张 `发` 就不只是“对当前这个立直者安全”，而是从结构上已经不可能成为和牌张的牌。

这类牌的优先级甚至高于普通现物，因为你不需要再借助“对手河里有没有打过”这种条件，它本身就是死安全。

## 第三部分：原页给出的四类完全安全牌，可以直接记忆

原页把完全安全牌概括成四类，这里可以直接记：

1. 其他三家都各自打过一次的牌
2. 上家刚打出的牌
3. 在国士无双不可能成立时，场上第四张字牌
4. 完全无筋、完全无搭子的“纯壁安全牌”

其中第四类会在后面的壁牌章节展开，这页先知道它存在即可。

这四类牌的共同点是：不只是“通常安全”，而是从规则或牌型结构上已经几乎不可能放铳。

## 第四部分：为什么先学现物和完全安全牌，而不是一上来学筋读

原页最后也说得很清楚：图示这类局面里，如果安全牌够多，防守其实很简单，你只要一直打这些绝对安全牌就行。

真正困难的，是：

1. 现物不够了怎么办
2. 完全安全牌也快打光了怎么办

这时候才轮到筋和壁这些“高概率安全牌”上场。

也就是说，防守学习的顺序必须是：

1. 先学绝对安全牌
2. 再学相对安全牌
3. 最后才学读牌与更细的危险度比较

如果这个顺序反了，很容易出现一种常见错误：明明手里还有现物，却先去打“我觉得可能安全”的筋牌，结果白送放铳。

## 这一页的结论

1. 现物是最基础的绝对安全牌：对手自己打过、或立直后见逃过的牌都属于现物
2. `27.gif` 的作用，是教你先从河里直接找绝对不会荣和的牌
3. 场上已确定无法构成任何荣和形的牌，属于完全安全牌，优先级高于普通现物
4. 原页给出的四类完全安全牌可以直接记忆，后续再展开纯壁安全牌的细节
5. 防守学习顺序必须是：先绝对安全，再相对安全，最后才是更细的读牌
