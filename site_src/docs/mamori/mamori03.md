# 筋牌（1）

面对立直时，最值得先提防的等待类型，是两面听牌。原因很简单：正常玩家只要能把坎张、边张、双碰整理成两面，多半都会整理完再立直。

因此原页先给出的基础数据非常重要：

**立直的等待里，大约三分之二是两面。**

而专门用来防两面的基础理论，就是筋。

## 第一部分：先把六种两面等待彻底记住

麻将里两面等待其实只有六种：

1. `1-4`
2. `2-5`
3. `3-6`
4. `4-7`
5. `5-8`
6. `6-9`

这六种等待把所有两面都覆盖完了，没有别的例外。原页强调这一点，是因为后面所有筋判断，都是建立在这六种等待基础上的。

## 第二部分：六种两面，其实又能归成三个筋组

把六种等待再往上整理，就会变成三个很重要的分组：

1. `1-4-7`
2. `2-5-8`
3. `3-6-9`

原页把这一步看得很重，是对的。因为你真正实战里，不可能每次都重新把所有两面穷举一遍，必须把它压缩成这种可快速识别的三大筋组。

记住这三个分组，后面的筋判断速度会快很多。

## 第三部分：筋的本质，是“若某张牌已被舍出，则与之对应的两面荣和会振听”

这部分的逻辑原页拆得很清楚，本质上只有两步：

1. 立直者若自己打过某张牌，对那张牌的荣和必然振听
2. 两面等待只有那六种，因此能反推出某些牌不可能以两面荣和命中

这就是为什么立直者打过 `4p` 以后，`5p6p` 两面就被排除了。因为如果他真的在等 `5p` 或 `6p`，那么 `4p` 所在的对应两面结构会让它构成立直者自己的振听。

## 第四部分：`28.gif` 这张图的作用，是把“筋如何落到实战河里”演示出来

<img src="../images/28.gif" width="396" height="420" />

这张图是整页里最关键的示例。它不是单纯告诉你“某几张牌是筋”，而是在示范：

1. 立直者已经打过 `4p`
2. 因此 `7p` 至少不会点到两面 `5p6p`
3. 立直者又打过 `2m` 与 `8m`
4. 因此 `5m` 两侧的两面 `34m` 与 `67m` 也都因振听被排除

于是像图里别家在立直后选择打出 `7p`、`5m`，并不是拍脑袋，而是在用筋来规避最常见的两面放铳。

原页同时也很克制，没有把它讲成“绝对安全”。因为筋能排除的，只是**两面放铳**。若对手是双碰、单骑、坎张、边张，筋依然可能照样中枪。

所以这一页真正要形成的结论是：

**筋不是绝对安全，而是“至少不点最常见的两面听牌”。**

## 这一页的结论

1. 立直等待里，两面占比最高，因此防守第一优先级就是先防两面
2. 两面等待只有六种，可再整理成 `147 / 258 / 369` 三组筋
3. 筋的本质，是利用振听规则排除某些两面荣和可能
4. `28.gif` 的作用，是演示如何从立直者舍牌反推出哪些牌至少不会点两面
5. 筋只能防两面，不能把双碰、单骑、坎张、边张一起排除，因此它只是“相对安全”
