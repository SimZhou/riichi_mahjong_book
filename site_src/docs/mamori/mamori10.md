# 对副露的防守

很多人面对立直还知道要防，但面对副露却常常两种极端：

1. 对方一鸣牌就立刻过度害怕，白白放弃很多本可继续推进的手
2. 完全不去读副露内容，直到对方和出来才发现那手其实一眼就该提防

原页这一页讲的，就是副露防守的第一层：

**先判断对方有没有听牌，再判断大概值多少钱，最后才决定要不要退。**

## 第一部分：先判断对方是不是已经听牌

和立直不同，副露不会自动告诉你“我已经听牌”。所以对副露防守的第一步不是看危险牌，而是先判断对方现在大概在哪个向听数。

原页先给了几个很实用的经验线：

1. 三副露通常应视为大概率已听牌
2. 混一色手一旦开始余出本色牌，往往已接近成牌
3. 很少看别人河的人，突然开始认真盯河，常常是因为已经听牌
4. 节奏一向很慢的人突然开始无停顿摸切，也往往意味着手牌已固定成听牌

这几条都不是绝对规则，但作为第一层筛查已经足够好用。尤其在牌桌上，能不能感觉到“对方突然切换了行为模式”，往往比死记几条公式更实战。

## 第二部分：再判断这手副露是高是低，别对所有副露都一律弃和

原页这一段是在纠正很多初学者的另一个毛病：对方一副露就紧张，但其实副露手之间点数差距非常大。

例如例 1 那种双吃断幺路线，如果宝牌不在那条色里，极可能只是 `1000` 点上下的小手。这类牌不该被过度高估。若你自己是胜负手，完全可以无视继续推进。

而像例 2 那种：

1. 役牌已碰
2. 宝牌明确参与
3. 亲家或场风役已经成立

就明显不同了。原页这里的逻辑很直接：这种副露至少已经在 `5800` 以上，继续无脑对推就很容易把自己送进大失点。

再往上，染手、副露宝牌、役牌叠加这些，原页都把它们归到高危副露。尤其清一色、带宝牌的混一色，几乎天然就是满贯甚至跳满候选。

## 第三部分：副露防守真正要看的，是“哪些牌最危险”

原页后半给了几类最该立刻提高警戒的牌型：

1. 大三元、役牌重叠时，剩下那张役牌会变成极高危牌
2. 对对和模样很浓时，生张字牌和生张刻子候选会非常危险
3. 后付式副露时，场上未见的役牌要重点盯防
4. 染手时，对应花色整门都要提高警戒

这部分最实用的地方在于，它不是让你“什么都怕”，而是让你把危险缩到少数几类具体牌上。这样你既不会无脑弃和，也不会完全没读到对手意图。

## 第四部分：对副露防守不能只靠形式，要结合点棒与对手身份

原页虽然没展开成一整套局势章，但意思已经很清楚：

1. 若对方只是 `1000` 点小副露，而你自己是机会手，不必轻易收手
2. 若对方是你当前点棒上的直接竞争对手，他的副露价值会被放大
3. 若对方副露后已经进入高打点、高速度状态，就要更果断切回防守

也就是说，对副露防守永远不是“看到两副露就停”，而是要先回答两件事：

1. 他现在大概有没有听牌
2. 即使和出来，大概会和多大

这两个问题答完，才有资格谈该不该退。

## 这一页的结论

1. 对副露防守的第一步，是先判断对方大概有没有听牌，而不是一看到鸣牌就乱怕
2. 三副露、染手余牌、本来不看河的人突然看河、节奏突然固定，都是高价值听牌信号
3. 副露手之间点数差异很大，不能把 `1000` 点断幺和役牌宝牌满贯看成同一等级
4. 真正要重点防的是役牌剩张、对对和生张、后付役牌、染手同色牌这些高危区域
5. 是否退让，最终仍要结合自己的手牌价值和点棒关系一起判断
