# 壁牌

寻找更安全牌的另一种重要手段，就是**壁**。

---

## 无机会壁

例如，如果对手是等 ![1p](../hai/pin1.gif) - ![4p](../hai/pin4.gif) 的两面，  
那么他手里至少得有一张 ![2p](../hai/pin2.gif) 和一张 ![3p](../hai/pin3.gif)。

也就是说，如果 ![2p](../hai/pin2.gif) 或 ![3p](../hai/pin3.gif) 其中某张在场上已经见了四张，  
对手就不可能是 ![1p](../hai/pin1.gif) - ![4p](../hai/pin4.gif) 两面待。

因为麻将里同一张牌总共就只有四张，这当然不可能再成立。

因此，![1p](../hai/pin1.gif) 就会变得相当安全。

这里要特别注意的是：**![4p](../hai/pin4.gif) 并不能因此说安全。**  
因为 ![4p](../hai/pin4.gif) - ![7p](../hai/pin7.gif) 两面仍然完全可能存在。

下面这张表可以直接记：

| 四张已见的牌 | 因“壁”而相对安全的牌 |
| --- | --- |
| `1` | 无 |
| `2` | `1` |
| `3` | `1` 与 `2` |
| `4` | `2` 与 `3` |
| `5` | `3` 与 `7` |
| `6` | `7` 与 `8` |
| `7` | `8` 与 `9` |
| `8` | `9` |
| `9` | 无 |

---

## 一机会壁

如果某张牌只见了三张，就叫做**一机会**。  
因为最后还剩一张没见，所以立直者依然有可能正好把那一张拿在手里。

这和无机会壁不同，仍然完全可能点进两面。

所以一机会只能理解成：

- 早巡还比较有参考价值
- 终盘基本靠不住

如果在牌局前半，某张牌已经见了三张，那么最后那一张埋在山里的可能性仍然不低，  
这时一机会壁还有一定可信度。

但越到后半巡，这种可信度就越差。

另外，例如立直者自己已经打过 ![3s](../hai/sou3.gif)，  
而场上又因为大家弃牌形成了一机会，那么这种一机会很可能是**人为制造出来的假安全感**。

尤其当三家都在弃和、最后那一张 ![3s](../hai/sou3.gif) 迟迟不露面时，  
反而要怀疑：那最后一张很可能就在立直者手里。

再比如，明明形成了壁，但外侧那张牌迟迟一直不出现，  
这时也必须怀疑双碰待。

这点不仅对一机会成立，对无机会壁也一样成立。

总之，一机会并不是特别可靠的东西。

### 理论

**一机会在早巡还可以参考，到了终盘就不能再信。**

---

## 双重一机会

一机会还有一种加强版。

例如 ![7p](../hai/pin7.gif) 和 ![8p](../hai/pin8.gif) 都已经见了三张时，  
对手同时把这两张都拿在手里的概率就会明显变低。

这种情况就叫**双重一机会**。在这种局面下，![9p](../hai/pin9.gif) 就会相当安全。

安全度大致可以记成：

**无机会 ＞ 双重一机会 ＞＞ 一机会**

---

## 利用壁找安全牌

接下来实际看一手怎么利用壁找安全牌。

<img src="../images/h009.gif" width="500" height="500" />

这里切立直者的现物 ![9m](../hai/man9m.gif) 当然也不差，  
但这题更好的答案是切 ![9p](../hai/pin9m.gif)。

因为场上 ![8p](../hai/pin8m.gif) 已经四张全见，所以 ![9p](../hai/pin9m.gif) 不可能点到两面。

而且在这个局面里，![9p](../hai/pin9m.gif) 本身也已经是第四张。  
这样一来，双碰待、单骑待也都一起被排除了。

再加上这里当然也不可能是国士无双，  
所以这张 ![9p](../hai/pin9m.gif) 就成了 **100% 会通过的牌**。
