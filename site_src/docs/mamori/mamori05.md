# 壁牌

除了筋之外，防守里另一条非常重要的相对安全判断，就是壁牌。

筋是靠“对方自己打过什么牌”来排除两面；壁则是靠“场上已经见了多少张同牌”来排除两面。

所以这页真正要学的，是：

1. 什么叫无机会壁
2. 什么叫一机会壁
3. 为什么一机会壁只能早巡用，不能终盘迷信

## 第一部分：无机会壁，是由“四张全见”直接排除某类两面

### 若 `2p` 或 `3p` 四张全见，那么 `1p-4p` 两面不成立

原页开头先把原理说得很直白。若对手想做 `1p-4p` 两面，手里至少得持有 `2p` 与 `3p` 之一。那如果 `2p`、`3p` 之中有某张已经四张全见，这类两面自然就不可能成立。

所以像：

1. `2` 四张全见，则 `1` 相对更安全
2. `3` 四张全见，则 `1` 和 `2` 相对更安全
3. `8` 四张全见，则 `9` 相对更安全

原页还给了一张总表，这张表其实很值得直接记住，因为它把所有壁安全牌的对应关系都列完了。

这里最重要的一句，是它特别提醒：

**能因为壁变安全的是外侧牌，不是内侧牌。**

例如 `2p`、`3p` 全见，会让 `1p` 相对安全，但不会让 `4p` 也跟着安全，因为 `4p-7p` 两面仍然完全可能成立。

## 第二部分：一机会壁，只是“相对还行”，绝不是硬安全

### 三张已见时，只能说还剩一张机会，因此叫一机会

若某张牌只见了三张，还剩最后一张没见，就会形成一机会壁。此时相关外侧牌仍然相对安全，但远没有无机会壁那么扎实。

因为只要最后那一张恰好就在立直者手里，相关两面依旧成立。

所以一机会壁与无机会壁的差别必须记清：

1. 无机会壁：对应两面从结构上已经被切断
2. 一机会壁：只是“只剩最后一张机会”，危险度下降，但仍可能中枪

## 第三部分：一机会壁越到后巡越不可信

### 早巡能信，终盘不能迷信

原页对一机会壁的警告非常重要。它说得很明确：一机会壁只有在早巡才有较高参考价值。

因为：

1. 牌局前半，最后那一张更可能还埋在山里
2. 牌局越往后，最后那一张在立直者手里的概率越高
3. 若立直者自己又打过形成该壁的牌，一机会壁还可能是被人为做出来的假安全感

所以一机会壁不能当成“晚巡救命稻草”。越到终盘，越应把它视为弱参考，而不是决定性安全依据。

## 第四部分：`h009.gif` 这张图的作用，是示范如何把壁直接转成完全安全牌

这张图是整页里最关键的实战演示。原页并不是单纯说“这里可以打 `9p`”，而是在完整展示判断链：

1. 场上 `8p` 已经四张全见
2. 所以 `9p` 不可能点到两面 `7p8p`
3. 同时图中 `9p` 自身也来到第四张
4. 因而连双碰、单骑都一起被排除
5. 国士无双也被排除后，这张牌就从“壁安全牌”升级成了真正的完全安全牌

也就是说，这张图要传达的重点不是“壁有时挺安全”，而是：

**壁可以和张数信息叠加，直接把某张牌推到 100% 安全。**

这就是为什么图里比起单纯打现物 `9m`，更推荐打 `9p`。因为后者不是普通安全，而是完全不会放铳。

## 这一页的结论

1. 壁牌是靠“场上同牌已见张数”来排除某类两面等待
2. 无机会壁很强，可以直接显著降低外侧牌的两面放铳风险
3. 一机会壁只是在早巡有一定参考价值，终盘不能迷信
4. 壁的安全收益主要落在外侧牌，不会自动让内侧牌一起安全
5. `h009.gif` 说明的重点，是壁与张数信息叠加后，有时能把某张牌直接推成完全安全牌
