# 弃和步骤（3）

前两页把弃和的原则和排序讲完了。这一页把它们放进具体局面里，告诉你遇到复杂实战时，应该怎样一巡一巡地按顺序退。

## 第一部分：`31.gif` 这张图展示的是最标准的单立直彻底防守顺序

<img src="../images/31.gif" width="472" height="502" />

这张图的局面很典型：

1. 两侧对手看起来都已经在退
2. 你的任务只剩对这一家立直彻底防守
3. 手牌里既有绝对安全，也有相对安全，还有最后不得不拆的对子

原页给出的顺序是：

1. 先打接近完全安全的 `1m`
2. 再打老头筋 `9p`
3. 再打相对更危险一点的 `6m`
4. 最后如果还没结束，只能拆 `7p` 对子

这个顺序背后的逻辑非常完整：

1. 先把最稳的牌用掉
2. 再用不会点两面的老头筋
3. 再用会吃坎张、但仍比无筋好一些的中张筋
4. 真走到最后，才拆对子买后续两巡

原页还特别解释了为什么 `9p` 必须先于 `6m`：因为 `6m` 不仅会中坎张，还有断幺九价值，放铳后的平均打点更高。

所以 `31.gif` 的意义不只是“给出答案”，而是在示范：

**弃和不是单看安全度，还要连放铳后的平均伤害一起考虑。**

## 第二部分：`32.gif` 这张图讲的是双立直时，共通安全牌比单家现物更重要

<img src="../images/32.gif" width="465" height="499" />

第二个例子更困难：场上已经出现双立直，而且时间还偏晚。

这时候原页的核心思路是：

1. 不要被“这张是其中一家的现物”迷惑
2. 先找对两家都相对能过的牌
3. 尤其要重视距离宣言牌较远的外筋

图中原页推荐的是连续打 `1s`。原因不是它对某一家绝对安全，而是：

1. 对亲家比较稳
2. 对另一家从早切 `2s` 来看也较像能过
3. 在双立直局面里，这种兼顾两边的退路比单家现物更值钱

原页同时顺手补了一条很实用的小经验：

**越远离立直宣言牌的那张外筋，通常越安全。**

因为若对手很早就把某张附近牌切掉，往往说明那一组牌在听牌前就已经被放弃了；而若是贴着宣言牌附近才出现，则更可能是听牌过程中的关键整理信息。

## 第三部分：弃和时，不要把“看起来像现物”误当成真实安全

第二个例子里，原页还顺带反驳了几种常见误判：

1. “先行立直的现物，后发立直应该也不会用吧”
2. “三段目的一机会壁还能信吧”
3. “宝牌附近的筋，应该也总比正无筋好吧”

这些在普通局面都未必成立，到了双立直、终盘场面更不能信。原页这里其实是在提醒你：

**局面越糟，越要回到最硬的依据，而不是靠模糊感觉。**

## 第四部分：弃和步骤真正想训练的，是“按顺序退”而不是“凭感觉退”

三页弃和步骤合起来，真正要训练的并不是记住每一张例牌，而是形成一套动作顺序：

1. 绝对安全先打
2. 再打只命中极少数形的牌
3. 再打不命中两面的牌
4. 最后才拆对子、拆暗刻续命

一旦这个顺序变成习惯，你面对立直时就不会每巡都重新恐慌，而是能很快知道：“我这一巡手里最该打的是哪一层牌”。

这就是原页三连篇真正的价值。

## 这一页的结论

1. `31.gif` 展示的是标准单立直弃和顺序：绝对安全、老头筋、中张筋、对子拆牌依次后移
2. `32.gif` 展示的是双立直时，共通安全与双边可过牌优先于单家局部安全
3. 弃和时不仅比较安全度，还要比较一旦中枪后的平均伤害
4. 双立直、终盘局面下，模糊线索会迅速失真，必须更依赖最硬的安全依据
5. 三页弃和步骤真正要养成的是“按顺序退”的习惯，而不是每巡凭感觉乱选
