# 防守

防守不是麻将里的附属技能，而是决定你能不能稳定上分的核心能力。

很多人把麻将理解成“只要会做快牌就行”，但真正拉开长期成绩差距的，往往不是胡牌有多华丽，而是：

1. 该冲的时候能冲
2. 不该冲的时候能及时收手
3. 在别人已经听牌时，知道哪些牌能打、哪些牌绝不能碰

所以这一章不会只讲几个零散安全牌口诀，而是从基础开始，把防守拆成完整体系。

## 本栏内容

1. [防守基础](./mamori01.md)
: 先建立最基础的防守观，明确麻将里的守备主要是对立直的应对，以及常见的几种攻守策略。

2. [安全牌](./mamori02.md)
: 从现物、完全安全牌开始，建立“绝对不会放铳的牌”这一最基本的安全概念。

3. [筋牌（1）](./mamori03.md)
: 先把筋的六种两面和三大分组彻底讲清，建立“筋只能防两面”的基本认识。

4. [筋牌（2）](./mamori04.md)
: 继续比较筋之间的安全度，重点讲没有“远筋”以及宝牌附近的筋反而可能更危险。

5. [壁牌](./mamori05.md)
: 处理无机会壁、一机会壁与双重壁，并说明壁如何进一步推导完全安全牌。

6. [弃和步骤（1）](./mamori06.md)
: 先建立彻底防守的基本顺序，重点是绝对安全牌先打、共通安全牌后留。

7. [弃和步骤（2）](./mamori07.md)
: 继续处理安全牌耗尽后的梯度退牌，把相对安全牌按命中形态分层排序。

8. [弃和步骤（3）](./mamori08.md)
: 用具体单立直、双立直场面演示如何把前两页的退牌顺序真正落到实战。

9. [迂回打法](./mamori09.md)
: 处理既不该全押也不该立刻放弃时的绕打，重点是对子落与搭子落的适用边界。

10. [对副露的防守](./mamori10.md)
: 先学会读副露是否听牌、值多少钱，再决定要不要对一副露手退让。

11. [扣牌（1）](./mamori11.md)
: 先建立扣牌的意义、成本与适用前提，避免把它误用成无差别拖牌。

12. [扣牌（2）](./mamori12.md)
: 用三个实战例子说明，什么时候该为压制对手副露而牺牲自己的通常牌效率。

13. [避免放铳](./mamori13.md)
: 从提前保留安全牌、终盘改打更安全听牌，到配牌弃和，系统整理降低放铳率的基本思路。

14. [读立直（1）](./mamori14.md)
: 先纠正“猜准确待牌”的误区，改成读取危险区，并讲清五的早切、间四间与搭子落筋。

15. [读立直（2）](./mamori15.md)
: 继续从切牌顺序里找隐藏安全牌，重点处理安全牌手打立直、逆切筋，以及对双碰和坎张的否定。
