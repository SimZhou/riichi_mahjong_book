# 读立直（1）

所谓“读立直”，并不是把对手到底在等哪一张准确猜出来。  
真要做到这一点，是极其困难的，甚至说几乎不可能也不过分。

归根到底，想去“猜中待牌”这件事本身就是错方向。  
真正该读的，不是待牌本身，而是：

- **哪些牌特别危险**
- **哪些牌相对安全**

这才是读立直的第一步。

### 理论小结

**立直的待牌不可能准确读出，也没必要去读。**  
真正该掌握的，是面对有特征的立直舍牌时，哪些牌会特别危险，哪些牌会相当安全。

以下开始看“危险度高的牌”的例子。

---

## 五的里筋

关于里筋，很多地方都讲过，这里不展开。  
真正在实战里危险度特别高的，主要也就两类：

1. 五的里筋
2. 间四间

先说五的里筋。

“某门 `5` 早切，则该门 `1-4`、`6-9` 值得警戒”  
这是很老的经验。虽然我没有统计数据，但从体感上讲，确实值得实战里认真留意。

而且 ![9p](../hai/pin9.gif) 已经在河里了，所以我会认为 ![1p](../hai/pin1.gif) 到 ![4p](../hai/pin4.gif) 的危险度明显偏高。

实际手牌如下：

![1m](../hai/man1.gif)![1m](../hai/man1.gif)![3m](../hai/man3.gif)![4m](../hai/man4.gif)![5m](../hai/man5.gif)![2p](../hai/pin2.gif)![3p](../hai/pin3.gif)![2s](../hai/sou2.gif)![2s](../hai/sou2.gif)![3s](../hai/sou3.gif)![3s](../hai/sou3.gif)![4s](../hai/sou4.gif)![4s](../hai/sou4.gif)

当然，也完全可能最后是索子或万子的待牌。  
但至少对筒子来说，![1p](../hai/pin1.gif) 到 ![4p](../hai/pin4.gif) 这条线会是重点危险区。

关键就在于：**![5p](../hai/pin5.gif) 是早切。**  
正因为利用价值很高的 `5` 在序盘就被切掉了，  
才会让人去怀疑，对手手里更像是已经有 `23` 或 `78` 这样的形。

如果 `5` 是接近立直时才切掉，那就完全不同了。  
它可能是浮牌，也可能是 `57` 摸 `8` 后不要了，也可能是从 `455` 这种形里固定出来的。  
可能性太多，就不值得特别去警戒它的里筋。

所以，哪怕例 2 里后来也切出了 ![5s](../hai/sou5.gif)，  
也不能据此认为 ![6s](../hai/sou6.gif) 到 ![9s](../hai/sou9.gif) 就比别的无筋更危险。

### 理论小结

**面对某门 `5` 早切的立直，要特别警戒这一门的 `1-4`、`6-9`。**

## 间四间

间四间一共有下面 `4` 种模式：

| 条件1 | 条件2 | 危险牌 |
| --- | --- | --- |
| `3` 早切 | 河里有 `8` | `4 / 7` |
| `4` 早切 | 河里有 `9` | `5 / 8` |
| `6` 早切 | 河里有 `1` | `2 / 5` |
| `7` 早切 | 河里有 `2` | `3 / 6` |

这两张相关牌相隔 `5` 格，所以也叫四间筋。  
而这里的关键词同样是：**早切。**  
因为只有利用价值高的 `3` 到 `7` 在序盘就被切掉，间四间才成立。

像 A、B 这种形，都会因为嫌弃二度受，而把 ![3p](../hai/pin3.gif) ![2p](../hai/pin2.gif) 这组拆掉。

那条筋通常是危险的。

### 理论小结

**看到两面搭子落立直，要先怀疑二度受。**

## 坎张落的跨筋

这条读法条件更复杂，但危险度也非常高。

**如果是离得较远的“外 -> 内”切巡，再形成坎张落立直，那么跨筋很危险。**

但其中有一条筋，危险度明显更高：

**![2m](../hai/man2.gif) 到 ![5m](../hai/man5.gif)**

理由如下。

这里看得出 ![1m](../hai/man1.gif) 和 ![3m](../hai/man3.gif) 形成了坎张落。  
但中间又隔着 ![5s](../hai/sou5.gif)、![东](../hai/ton.gif) 这些无关牌，所以它不可能只是单纯“原来有个坎张，后来不要了”。

如果只是拿 ![东](../hai/ton.gif) 当安全牌，那它更早就该被留下，不会是这种顺序。  
而如果是为了摸 ![4m](../hai/man4.gif) 做两面，也不太合理，因为同阶段里 ![5s](../hai/sou5.gif) 明显更适合先处理。

因此，更合理的解释往往是：

![3m](../hai/man3.gif)![3m](../hai/man3.gif)![4m](../hai/man4.gif)

这一类形里，切掉 ![3m](../hai/man3.gif) 立直。  
这样一来，![2m](../hai/man2.gif) 到 ![5m](../hai/man5.gif) 的跨筋就会变得特别危险。

其实就算没到这么完整的条件，  
只要看到宣言牌离得较远、并且是外后内的坎张落，  
就应该先认知到：这条跨筋往往很危险。

读立直时，危险牌能读到这个程度，我认为其实已经很够用了。  
比起继续往下猜，更重要的是下一页要讲的“安全模式”。
