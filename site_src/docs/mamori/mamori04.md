# 筋牌（2）

这一页继续把筋往前讲深一点。  
筋是麻将里最重要的概念之一，但也最容易被误用。

---

## 远筋？

这是初学者里很常见的一种误解。

例如，面对下面这个立直：

![1s](../hai/sou1.gif)![白](../hai/haku.gif)![西](../hai/sha.gif)![8s](../hai/sou8.gif)![6m](../hai/man6.gif)![赤6p](../hai/ypin6.gif)

因为 ![6m](../hai/man6.gif) 已经过河，所以 ![3m](../hai/man3.gif)、![9m](../hai/man9.gif) 是筋。  
也就是说，它们至少不会点进两面，算是稍微安全一点。

但即使 ![8s](../hai/sou8.gif) 已经过河，也不能说 ![2s](../hai/sou2.gif) 和 ![5s](../hai/sou5.gif) 就安全。  
偶尔会看到有人喊着“远筋”就把牌打出去，但那其实毫无意义。

不过，同样都是无筋，![5p](../hai/pin5.gif) 和 ![5s](../hai/sou5.gif) 的危险度还是会有差别。

![5p](../hai/pin5.gif) 可能命中：

- ![3p](../hai/pin3.gif)![4p](../hai/pin4.gif)
- ![6p](../hai/pin6.gif)![7p](../hai/pin7.gif)

这两种两面。

而 ![5s](../hai/sou5.gif) 因为其中一边的 ![6s](../hai/sou6.gif)![7s](../hai/sou7.gif) 两面已经被否定，所以相对还稍微好一点。

下面把筋整理成一张表，直接对照记忆：

| 弃牌 | 筋 |
| --- | --- |
| `4` | `1` 与 `7` |
| `5` | `2` 与 `8` |
| `6` | `3` 与 `9` |
| `1` 与 `7` | `4` |
| `2` 与 `8` | `5` |
| `3` 与 `9` | `6` |

## 更安全的筋

**＜对手立直的舍牌＞**

![1s](../hai/sou1.gif)![白](../hai/haku.gif)![西](../hai/sha.gif)![8s](../hai/sou8.gif)![6m](../hai/man6.gif)![赤5p](../hai/ypin5.gif)　宝牌 ![1p](../hai/pin1.gif)

就用这个立直来想。

### 例1

![3m](../hai/man3.gif)![3m](../hai/man3.gif)![9m](../hai/man9.gif)![9m](../hai/man9.gif)![1p](../hai/pin1.gif)![2p](../hai/pin2.gif)![3p](../hai/pin3.gif)![6p](../hai/pin6.gif)![7p](../hai/pin7.gif)![2s](../hai/sou2.gif)![2s](../hai/sou2.gif)![3s](../hai/sou3.gif)![4s](../hai/sou4.gif)　自摸 ![2s](../hai/sou2.gif)

因为 ![6m](../hai/man6.gif) 已经过河，所以 ![3m](../hai/man3.gif) 和 ![9m](../hai/man9.gif) 都是筋。  
那么该先拆哪一张？

- ![3m](../hai/man3.gif) 可能点到：![3m](../hai/man3.gif) 单骑、![3m](../hai/man3.gif) 和别的牌的双碰、以及 ![1m](../hai/man1.gif)![2m](../hai/man2.gif) / ![2m](../hai/man2.gif)![4m](../hai/man4.gif)
- ![9m](../hai/man9.gif) 可能点到：![9m](../hai/man9.gif) 单骑、![9m](../hai/man9.gif) 和别的牌的双碰

由于 ![9m](../hai/man9.gif) 不会再吃到边张、坎张那部分风险，所以它明显更安全。  
因此这里应该切 ![9m](../hai/man9.gif)。

有些人会怀疑：`3m` 单骑这种东西真的会有吗？

![1m](../hai/man1.gif)![1m](../hai/man1.gif)![1m](../hai/man1.gif)![3m](../hai/man3.gif)　![2m](../hai/man2.gif)![2m](../hai/man2.gif)![2m](../hai/man2.gif)![3m](../hai/man3.gif)　![3m](../hai/man3.gif)![4m](../hai/man4.gif)![4m](../hai/man4.gif)![4m](../hai/man4.gif)

像这种复合形里，当然完全可能出现。

### 理论

**老头牌没有坎张、边张风险，所以通常比其他数牌更安全。**

**＜对手立直的舍牌＞**

### 图2

![2m](../hai/man2.gif)![3m](../hai/man3.gif)![7m](../hai/man7.gif)![8m](../hai/man8.gif)![2p](../hai/pin2.gif)![2p](../hai/pin2.gif)![4p](../hai/pin4.gif)![6p](../hai/pin6.gif)![8p](../hai/pin8.gif)![8p](../hai/pin8.gif)![9p](../hai/pin9.gif)![9p](../hai/pin9.gif)![9p](../hai/pin9.gif)　自摸 ![4m](../hai/man4.gif)

这时自己手里有 ![2p](../hai/pin2.gif) 和 ![8p](../hai/pin8.gif)，两张都算筋。

- ![2p](../hai/pin2.gif) 可能点到：![2p](../hai/pin2.gif) 单骑、![2p](../hai/pin2.gif) 与别的牌双碰、![1p](../hai/pin1.gif)![3p](../hai/pin3.gif)
- ![8p](../hai/pin8.gif) 可能点到：![8p](../hai/pin8.gif) 单骑、![8p](../hai/pin8.gif) 与别的牌双碰、![7p](../hai/pin7.gif)![9p](../hai/pin9.gif)

但这里 ![9p](../hai/pin9.gif) **已经四张全见（包括宝牌指示牌）**，所以坎 ![8p](../hai/pin8.gif) 待其实不可能存在。

反过来，因为宝牌是 ![1p](../hai/pin1.gif)，所以坎 ![2p](../hai/pin2.gif) 待反而非常像会出现的牌。

如果别人手里拿着 ![1p](../hai/pin1.gif)![3p](../hai/pin3.gif)![5p](../hai/pin5.gif) 这种两坎结构，往往就会切 ![5p](../hai/pin5.gif) 立直。

像下面这种形也完全可能：

![1p](../hai/pin1.gif)![1p](../hai/pin1.gif)![1p](../hai/pin1.gif)![3p](../hai/pin3.gif)　![1p](../hai/pin1.gif)![3p](../hai/pin3.gif)![3p](../hai/pin3.gif)![3p](../hai/pin3.gif)

所以 ![2p](../hai/pin2.gif) 根本不是“有筋所以可以打”，而是**相当危险的牌**。

从图 2 里，正确答案是切 ![8p](../hai/pin8.gif)。

### 理论

**宝牌旁边，就算是筋也很危险。筋引挂立直也很多。**
