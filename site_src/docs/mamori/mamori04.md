# 筋牌（2）

上一页先讲了最基础的筋：它只能帮你排除两面放铳。这一页继续把筋往实战方向推进，重点有两条：

1. 不存在所谓“远筋就安全”
2. 同样是筋，安全度也有先后顺序

## 第一部分：没有所谓“远筋更安全”这种规则

### `8s` 过了，不代表 `2s`、`5s` 就安全

原页先处理了一个非常常见的误解：有人看到 `8s` 已经过河，就会说 `2s`、`5s` 是“远筋”，因此也能打。

这个理解是错的。

筋只在明确对应关系成立时才有效，例如：

1. `6m` 过了，则 `3m`、`9m` 是筋
2. `4s` 过了，则 `1s`、`7s` 是筋

但如果只是 `8s` 过了，你并不能顺手推出 `2s`、`5s` 安全。它们依然可能命中：

1. `23s`
2. `45s`
3. 双碰
4. 单骑等别的等待

所以原页这里的批评很直接：把“远筋”当安全依据，本身没有理论支撑。

## 第二部分：同样是无筋，危险度仍然会有差别

### 例子里为什么 `5s` 比 `5p` 稍微好一点

原页虽然否定了“远筋”，但没有把所有无筋牌都看成完全一样危险。它举的例子是：

1. `5p` 可以命中 `34p` 与 `67p` 两种两面
2. `5s` 因为一侧两面已被排除，只剩另一边更现实

这说明真正该看的不是“远不远”，而是：

**这张牌还保留多少种实际可能的两面命中方式。**

也就是说，危险度比较必须回到具体两面结构本身，而不是靠口头术语偷懒。

## 第三部分：同样是筋，老头牌往往比中筋更安全

### 例 1：`3m` 和 `9m` 都是筋时，`9m` 更安全

原页第一个正式例题非常重要。若 `6m` 已经过河，那么 `3m` 和 `9m` 都算筋。但它们并不等价。

因为：

1. `3m` 仍可能命中坎张、边张、单骑、双碰
2. `9m` 则少了边张、坎张这类一部分常见结构

换句话说，幺九牌之所以比中张更适合防守，不是因为它们“神秘地安全”，而是因为它们天然少了一部分可成立的坏形组合。

所以在这题里，正确选择是先拆 `9m`，不是 `3m`。

### 这条经验可以直接记成一句话

**同样是筋，幺九牌通常比中张更安全。**

这是因为它们不会再吃到某些坎张、边张形的放铳风险。

## 第四部分：靠宝牌附近的筋，往往反而很危险

### 例 2：宝牌旁边最容易出现“筋引挂”与贪打点听牌

第二个例题则是反过来的高危场面。

原页给的局面里，`2p` 与 `8p` 都看上去是筋，但由于：

1. `9p` 已经全见，`8p` 一侧的坏形风险反而明显减少
2. 宝牌是 `1p`，于是 `2p` 周边特别容易成为高打点手的关键区
3. 这类时候，很多人会用宝牌周边做筋引挂立直

所以这里真正安全的是 `8p`，而 `2p` 不仅不是“筋所以还行”，反而是很危险的牌。

这也是原页最后一条结论：

**宝牌附近，即使是筋，也不能轻信。**

因为打点欲望会让别人更愿意把危险形听在宝牌边上。

## 这一页的结论

1. 不存在“远筋就安全”的规则，筋必须按真实对应关系来判断
2. 同样是无筋，仍可按“还剩几种两面命中方式”比较危险度
3. 同样是筋时，幺九牌通常比中张更安全，因为少掉部分坎张、边张风险
4. 宝牌附近的筋需要特别警惕，筋引挂和高打点贪留会显著抬高其危险度
5. 筋不是背术语，而是持续缩小“这张牌还可能命中哪些实际听牌形”的过程
