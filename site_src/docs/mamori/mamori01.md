# 防守基础

麻将里理想的防守，当然是只停掉对手真正的和牌张，其余全部照打。但现实里这几乎不可能做到。就算你会读牌，真正能把对手和牌张压缩到非常小的范围，命中率也远没有想象中高。

所以防守的第一步，不是幻想“读出对手所有等待”，而是先抓住最可靠的事实：

**只有当对手已经听牌时，放铳风险才真正成立。**

而在实际麻将里，最明确告诉你“我已经听牌了”的动作，就是立直。因此多数防守技术，首先都是围绕立直展开。

## 第一部分：防守的核心对象，是已经明确听牌的对手

和没有听牌的人对打时，你切什么都不会放铳。问题只会在对手已经听牌以后产生。所以原页一开头就把重点放在“立直对策”上，这个切入很准确。

因为立直者已经把最重要的信息公开给你了：

1. 他已经听牌
2. 接下来每一张打出去的牌都存在真实放铳风险
3. 你不能再按平时那种只看自己手牌的节奏继续推进

也就是说，防守本质上是从“只考虑自己如何成牌”，切换到“先控制损失”。

## 第二部分：麻将里的攻守，大致可以分成五种策略

### `26.gif` 这张图的重点，是帮你先建立完整的攻守坐标系

原页在这里给出了一张总览图，把攻守分成五类。

<img src="../images/26.gif" width="233" height="147" />

这张图要表达的不是复杂术语，而是一个很实际的框架：

1. 最保守的一端，是彻底弃和的 `彻底防守`
2. 最激进的一端，是继续按原计划全力推进的 `全押`
3. 中间还存在绕打、回转、差入等更细的过渡带

对初学者来说，不需要一开始就把五种策略都运用得很细。原页说得很务实：先学会两头就够了。

也就是：

1. `全押`
2. `彻底防守`

因为很多中级以下玩家的主要问题，根本不是不会中间技术，而是连什么时候该彻底停手都做不到。

## 第三部分：彻底防守，是所有防守技术的基础形态

### 彻底防守的意思，是先放弃和牌目标，把“不放铳”放到第一位

所谓彻底防守，就是放弃自己这一手的和牌与手变，专门挑最安全的牌打出去。它并不花哨，但它是最可靠、最容易学习、也最能直接改善成绩的防守方式。

因为对很多玩家来说，输分的主要来源不是“不会做大牌”，而是：

1. 该收的时候不收
2. 明明局面已经危险，却还抱着“说不定还能摸回来”的幻想硬推
3. 最后把一手本来只会流局的小损失，推成了大放铳

所以原页强调“先学会彻底防守”，这是非常正确的顺序。

## 第四部分：回转打法更难，但不是初学者的优先课题

原页也提到中间地带的策略，例如边守边做、尽量兼顾听牌等。这类打法确实更高级，但门槛也高，因为它要求你同时处理：

1. 牌山与危险度判断
2. 手牌未来的可改良空间
3. 安全牌的剩余数量
4. 当前局况是否值得继续争

所以对当前阶段来说，更实用的顺序是：

1. 先学会彻底防守
2. 再逐步学会哪些场面可以少量回转
3. 最后才处理那些很细的攻守平衡题

## 第五部分：防守之所以重要，是因为它最直接提升长期成绩

原页最后一段其实是在强调一个很容易被忽视的事实：防守虽然不像和牌那样“爽”，但它对成绩的影响往往更直接。

因为多数玩家输分，不是输在和不到，而是输在：

1. 本可不放的铳放了
2. 本可小输的局输成大失点
3. 本可保住名次的终局因为一铳直接崩盘

所以如果目标是稳定变强，防守是无法绕开的训练项。尤其彻底防守，是最先值得练到本能化的部分。

## 这一页的结论

1. 防守的核心对象，是已经明确听牌的对手，尤其是立直者
2. `26.gif` 的意义，是先把麻将攻守分成完整的五种策略框架
3. 对初学者来说，先掌握“全押”和“彻底防守”两端就够了
4. 彻底防守的本质，是放弃这手牌的和牌目标，把不放铳放到第一位
5. 防守虽然不华丽，却是最容易直接改善长期成绩的核心技术之一
