# 完全弃和的步骤（1）

完全弃和有明确的步骤。  
只要严格按步骤退，放铳就会明显减少。

---

## 完全弃和的基本想法

**从安全度更高的牌开始打。**

这就是最基本的原则。  
手牌被拆坏也不用在意。

因此，只要手里还有：

- 现物
- 场上已经三张见切的字牌这类完全安全牌

就应该优先把这些牌打出去。

关键在于：**先打确定能过的牌。**  
因为一巡之间，现物还可能继续增加；  
也可能在你下一次出牌之前，别人已经放铳，或者立直者已经自摸。

明明手里还有安全牌，却先去打“看起来好像也能过”的牌，  
从防守角度看就是吃亏。

---

## 现物之间的优先级

就算同样都是现物，还是有顺序。  
原则是：**先打将来危险度更高的现物。**

### 例

<img src="../images/h006.gif" width="500" height="500" />

这手已经散得很厉害了，  
从这里去跟亲家的立直硬拼，几乎等于自杀。

手里现在的安全牌有：

- ![5m](../hai/man5m.gif)
- ![6s](../hai/sou6m.gif)
- ![西](../hai/sham.gif)

这里不管先切 ![5m](../hai/man5m.gif) 还是先切 ![6s](../hai/sou6m.gif) 都没问题，  
但先切 ![西](../hai/sham.gif) 就是顺序错误。

因为如果这时又有人追立，手里就会一下子变成没有任何安全牌。  
而图里下家已经突然开始切无筋，所以接下来再冒出立直，完全不奇怪。

因此，这种局面就该毫不犹豫地先把 ![5m](../hai/man5m.gif)、![6s](../hai/sou6m.gif) 拔掉。

### 理论

**当现物有多张时，共通安全牌要温存。**
