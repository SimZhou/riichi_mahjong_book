# 碰与吃（1）

你有意识到这两种鸣牌的区别吗？

## 碰是四倍速，吃是二倍速

碰可以从三家那里鸣到，吃却只能从上家那里鸣到。

也就是说，碰比吃更容易实现。

## 如果是为了消一发，就用吃

如果鸣牌只是为了消掉立直的一发，理论上优先用吃。

因为如果用碰来消一发，就会一下子失去两张对立直者的现物。

如果一发是消掉了，结果却轮到自己放铳，

那就不知道是为了什么才去鸣了。

无论是吃来消一发，还是碰来消一发，

都只该在你确信自己鸣完以后还能安然撤退时才去做。

### 理论总结

<span class="green">用“碰”来消一发，风险很高。  
如果只是为了消一发而鸣，用“吃”会更稳妥。</span>
