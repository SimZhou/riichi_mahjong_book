# 对对和

对对和是很多初学者最喜欢的副露役之一，因为它看上去很直接：只要不停碰牌，把整手做成刻子就行。

但这也是最容易“看起来能做，实际越做越死”的役种之一。

这一页的重点不是教你见对子就冲，而是先建立一个判断：

**什么样的手是真正适合走对对和，什么样的手只是硬凑。**

## 第一部分：四组对子起步，通常还不够支撑你硬做对对和

### 例 1：材料不足时，早早从无役字牌开碰，往往只是把自己锁进苦手

第一例最典型。手里只有四组对子，就有人会连不是役牌的 `北` 都先碰出去。

这类打法的问题很集中：

1. 对对和材料其实还没成熟
2. 中张对子如 `44m`、`77s` 并不好碰
3. 就算最后全碰出来，往往也是裸单骑、点数不高
4. 一旦中途被立直压到，既难前进，也难回头

所以这不是“积极进攻”，而是典型的强行仕挂。四组对子起手，通常还不够支撑你把整手直接固定成对对和。

## 第二部分：真正适合对对和的，是刻子与对子合计已经达到五组的手

### 例 2：材料已经齐全时，就应主动把对对和当主路线

第二例才是更标准的对对和手。

这里的关键不在于“对子多”，而在于：

**刻子与对子合计已经有五组。**

一旦达到这个数量，说明你离对对和只差把剩余块兑现出来，路线已经足够明确。这种手就不需要再等役牌碰出以后才开始动手，而是可以从 `3p` 这类普通对子先积极碰起。

因为在这种牌里，副露不是把结构打散，而是在把已经成熟的路线尽快兑现。

## 第三部分：五组对子时，要比较七对子和对对和哪条更现实

### 例 3 与例 4：同样是五组对子，能不能走对对和，要看剩余形状是否容易碰成

最微妙的，就是五组对子这类手。它们往往同时也是七对子一向听。

这时不能死背“对子多就做对对和”，而应比较两件事：

1. 剩余两块是不是容易碰
2. 若副露之后，残形会不会太苦

第三例适合转对对和，原因是：

1. `东`、`中` 这类字牌对子本来就容易碰
2. 还有老头牌、幺九牌与混全带幺九的复合空间
3. 即使不是门清七对子，对对和路线也明显更快

所以这手除了 `4m` 之外，基本都可以积极仕挂。

第四例则相反。虽然也是五组对子，但剩余部分更偏散、碰完以后残形会很僵，副露质量并不好。与其把牌越碰越苦，不如直接把它固定回七对子路线。

还有一条很实用：若场上已经出现两张同牌，让你手里某个对子变成“死对子”，那七对子路线会明显变差，这时也要重新提高对对和的优先级。

## 第四部分：四暗刻一向听时，通常也该先取现成的满贯听牌

### 例 5：不要为了追役满，放弃已经很值钱的碰听机会

第五例是很多人容易浪漫化的位置。牌面看起来是四暗刻一向听，这时若有人打出 `6p`，很多人会舍不得碰，想继续等役满。

但从实战角度看，更稳的处理通常是：

1. 就算第一张，也直接碰听
2. 先把现成的满贯级和牌率拿到手
3. 除非点棒情况明确要求逆转大牌，否则不必执着役满梦

原页这里的取向很明确：除非是落后很多、必须搏一击逆转的局面，否则不该为四暗刻幻想放弃已经现成可和的高价值手。

## 第五部分：对对和不能贪过头，已有足够打点时应优先和牌率

### 例 6：还有余地时，可以顺手兼顾对对和高目

第六例这种手，如果再摸进 `34p`，可以顺势做成对对和双碰待；若有 `8m` 打出，也可以碰进去形成更高目的等待。

这种“顺势兼顾高目”没有问题，因为它没有明显破坏当前和牌率。

### 例 7：但若为了贪对对和把等待大幅缩死，就已经过线了

第七例就不同。此时如果连 `1m` 都去碰，对对和是更近了，但等待张数会明显暴跌。

而且要注意另一个更现实的问题：

1. 你已经碰了三组牌
2. 这时再想让别人继续打进关键万子，本来就不现实
3. 原本手里已经有足够打点，再继续硬抬番，只是在牺牲和牌率

所以这类局面应记住一个底线：

**当手牌已经有 `3900` 以上、等待也不差时，应优先和牌，不要为了额外一番把整手贪坏。**

## 这一页的结论

1. 四组对子起手通常还不足以强做对对和，尤其不该从无役字牌乱碰起手
2. 真正适合对对和的，是刻子与对子合计已经达到五组的手
3. 五组对子时，要比较七对子和对对和哪条路线更快、更现实
4. 四暗刻一向听多数情况下应先取现成高打点听牌，不必执着役满
5. 对对和可以顺手兼顾，但已有足够打点时，不要为了贪更多番而大幅牺牲等待
