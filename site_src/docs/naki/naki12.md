# 后付（2）

上一页讲的是：后付并不一定比先碰役牌更差，而且很多时候还更便于保留守备。

这一页继续往前推一步，讨论更实战的三个场面：

1. 急所牌已经出现，不能再等
2. 可以同时保留别的役种变化时，后付会更强
3. 局况逼着你“只要先和就行”时，后付是直接可用的抢和手段

## 第一部分：急所牌已经出现时，后付应果断执行

### 例 1：关键牌只剩极少枚时，不能再指望自摸把形慢慢补齐

第一例里，如果上家打出 `7s`，这张牌就已经不是普通进张，而是整手的急所牌。

原因很明确：

1. `7s` 本身是连接宝牌块的关键张
2. 场上可见后，这张牌剩余枚数已经很少
3. 若错过这一次，再想靠自己连摸两次去补 `4s`、`7s`，成功率非常低

所以这里虽然是边张吃牌，看起来很像典型后付，但依然应该马上吃下，把速度先拿到手。

吃完以后，真正重要的收尾是：

1. 尽快把 `6m`、`4p` 这类与当前路线无关的牌处理掉
2. 最好留一张相对安全的牌在手里
3. 若后续再摸到役牌，也可以顺势保留，让役种更稳

这类仕挂看起来“很像在赌白牌会不会出”，但实战里大多并不会被人完全锁死。相反，别人若强行把白牌抱死，自己也会因此失去不少进攻空间。所以这种后付并不虚，反而是很标准的实战判断。

## 第二部分：带有双重役种变化的后付，比单纯等役牌更值得鸣

### 例 2：三色与白牌并存时，后付不再只有一条生路

第二例的好处在于，它不是“纯靠白牌活”。

如果 `3m`、`4p`、`3s` 这类关键牌出来，本来就应该积极吃牌，因为这手不止有白牌后付，还同时保留三色变化。

这意味着：

1. 你不是把所有希望压在一张役牌上
2. 还能靠另一条手役路线把牌完成
3. 所以这类后付的风险比表面看起来小得多

原页这段的意思，可以直接翻成实战话：

**后付最好不是单线赌博，而是同时保留另一条能和的路线。**

### 例 3：先碰也不怕，因为没和成另一边的风险其实没那么高

第三例更进一步。若 `1p` 打出来，应直接碰。

很多人会担心：万一发牌一直不出，这不是白开了吗？但实战里这类担心通常被高估了。因为这手不只是在等 `发`：

1. 还有刻子手变化
2. 还能往三暗刻、对对和方向扩展
3. 已经副露后，整体完成速度本来就快

所以这里更该看的，不是“有没有理论上的卡死可能”，而是“鸣完后整体和牌率是不是足够高”。答案是高，所以可以大胆碰。

## 第三部分：终局只求和牌时，后付就是正解

### 例 4：`41.gif` 这张图表达的不是排版，而是点差局面彻底改变了副露标准

这一页唯一的独立示意图，就是第四例的点数局面图。重点不在图本身，而在它表达的场况。

<img src="../images/41.gif" width="415" height="443" />

这张图说明的是典型的南场终盘接战：

1. 点差非常接近
2. 只要先和出 `1500` 左右就足以逆转或守住名次
3. 这时继续等 `南` 这种确定役牌再开门，已经太慢

所以原页这里给的判断非常坚决：

1. 能鸣的牌几乎都该鸣
2. 目标不是做漂亮，而是先把和牌抢到手
3. 只有像 `5m` 碰这种会明显把结构碰坏的动作仍然要避开

这张图真正要传达的是：

**在南四接战里，副露标准和东一局完全不同，后付会从“可考虑”直接升级成“必须考虑”。**

而且这类局面下，对手自己也在拼命找和，通常没有余裕长期把你的役牌死死扣住。正因为全桌都急，后付反而比平时更有效。

## 这一页的结论

1. 急所牌已经出现且剩张很少时，不要再犹豫，后付应立即执行
2. 最好的后付不是单线押役牌，而是同时保留三色、对对和等第二条完成路线
3. 只要鸣完后整体和牌率明显提高，就不必过度担心“万一役牌不出”
4. `41.gif` 说明的是南场终盘接战里，点差要求会直接改变副露标准
5. 在“只要先和就够”的局面里，后付不是妥协，而是最务实的抢和手段
