# 副露判断基础

上一页讲清了副露的收益和代价。这一页开始回答真正的实战问题：

**一张牌打出来时，到底该不该鸣。**

最重要的不是背答案，而是建立判断顺序。原页把它比喻成一个天平，这个比喻很好用：

1. 一边是鸣牌带来的速度和成役收益
2. 另一边是打点下降、守备变差、路线被锁死的代价
3. 哪边更重，就决定这手该不该鸣

关键在于：**这些砝码的重量会随局面变化。**

## 第一原则：鸣不鸣，不看条目数量，看权重大小

很多初学者会犯一个简单错误：

1. 把优点列三条
2. 把缺点列三条
3. 觉得“差不多”或者“优点更多”

这当然不对。

真正该比较的是：

1. 这次鸣牌到底能加快多少
2. 这次掉分到底痛不痛
3. 一旦别人先攻，你还有没有退路
4. 当前局况，是更需要快还是更需要值钱

所以这一页所有题，都是在训练一个基本动作：

**把速度、打点、守备、局况放到同一杆秤上称。**

## 例 1：早巡、便宜、牌形又差时，通常不鸣

第一例里，东一局子家，三巡目碰中。

表面上看，碰完就有了一番，似乎更容易和；但认真称量以后会发现：

1. 剩余牌形依然很差，就算鸣了也未必真能快很多
2. 一旦继续走副露路线，守备会大幅下降
3. 这手多数情况下也就 `1000-2000` 点
4. 东一局子家这种局况，本来就不值得为这点便宜收益过早锁死自己

所以原页结论是不鸣。

这题最重要的收获是：

**“有役了”不等于“就该鸣”。**

如果这副牌鸣完依旧又慢又穷，那速度收益并没有大到值得你交出守备和门清可能。

## 怎么称“速度”这一边

原页在例 1 里其实已经示范了速度判断方法。不是问“鸣了是不是快一点”，而是问：

1. 鸣完后这手距离听牌还有多远
2. 剩下的块是不是依然很坏
3. 未来是不是还要继续靠多次副露才能成型

如果副露后的牌依旧不好做，那速度收益往往比你想象中小。

## 怎么称“守备”这一边

守备不是抽象词，实战里可以具体问：

1. 鸣完以后，手里还剩多少张可回旋的牌
2. 这手未来大概率还要再鸣几次
3. 如果别人先立直，我还有没有安牌可退
4. 这手本来有没有宝牌、红牌、足够打点，值得我承担全押风险

如果答案普遍偏负面，那么守备损失的砝码就很重。

## 例 2：亲番、中巡、速度提升明显时，就该鸣

第二例几乎是例 1 的反面。

南一局亲家、六巡目、打出 `6s`。如果吃进去：

1. 速度会大幅提升
2. 原本门清路线也并不算特别容易和
3. 即使副露，手牌整体守备也没有惨到无法接受
4. 更重要的是，亲番和连庄价值会让“先和一次”的收益大幅上升

于是这次天平就明显倒向副露一边。

这题说明：

1. 同样是鸣牌，场况一变，答案就可能完全反过来
2. 亲番、需要连庄、已经进入中巡时，速度砝码会显著变重

## 第二原则：局况会直接改变砝码重量

原页最后特意强调：真正的高手，不是懂副露优缺点，而是**天平校准得更准**。

也就是同一手牌，在不同局况下，能迅速改写砝码大小。最典型的变化包括：

1. `东一子家` 和 `南场亲家` 对速度要求完全不同
2. 早巡和中后巡对守备损失的容忍度不同
3. 顶目、末位、需要和牌连庄、需要大牌逆转，这些都会改变打点权重

因此副露判断从来都不是静态教材题，而是：

**同一副牌，放在不同分差和巡目里，答案可能完全不同。**

## 这一页的结论

1. 鸣不鸣要用“速度 vs 打点与守备”的天平来判断
2. 不看优缺点数量，只看各自权重大小
3. 如果鸣完牌形依然很差，速度收益其实有限
4. 若鸣牌会严重压缩守备，而打点又低，这通常不值得
5. 亲番、中巡、需要连庄或急需先和时，速度砝码会明显变重
6. 真正的副露水平，体现在你能否随局况准确调整这杆秤
