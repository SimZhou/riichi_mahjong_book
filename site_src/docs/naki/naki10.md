# 混一色

混一色因为只要把一种花色和字牌集中起来，作业式地去收集就能做成，
所以非常受初学者欢迎。

实际上，混一色不但容易做，打点也高，
确实是最值得去狙击的役种之一。

混一色真正拉开差距的地方，还是在鸣法和对牌形的理解。

实战里经常能见到有人该鸣的牌不鸣，
也有人把不该鸣的牌鸣了，结果错失和牌。

## 不该鸣的手牌

混一色就算鸣牌也有 `2 番`，所以一般没必要强执着门清。

那么，什么情况下反而应该先不鸣？

### 例 1

![1m](../hai/man1.gif)![2m](../hai/man2.gif)![3m](../hai/man3.gif)![4m](../hai/man4.gif)![5m](../hai/man5.gif)![5m](../hai/man5.gif)![6m](../hai/man6.gif)![7m](../hai/man7.gif)![7m](../hai/man7.gif)![8m](../hai/man8.gif)![北](../hai/pei.gif)![北](../hai/pei.gif)![北](../hai/pei.gif)

像例 1 这种，门清本来就完全足以做到听牌。

如果一鸣就会变成廉价副露混一色，
甚至原本还带着一气通贯，
那就应该尽量拖到最后，先狙击门前混一色。

顺便一提，例 1 这手无论摸进哪张万子，
都会以两面以上的形听牌。

这当然是个偏极端的例子，但如果已经是门清混一色好形一向听，那就值得先靠门清努力看看。

像这种形，如果 ![东](../hai/ton.gif) 或 ![北](../hai/pei.gif) 在场上已经有 `2 张` 切出，变成了枯对子，这在实战里是很常见的。

很多人会摆出“这跟我没关系”的样子，照样一路碰下去。

但只要有死对子，这种时候更好的路线就是门前混一色七对子。

面子都还没怎么成型就先仕挂。

![4m](../hai/man4.gif)![1p](../hai/pin1.gif)![3p](../hai/pin3.gif)![5p](../hai/pin5.gif)![6p](../hai/pin6.gif)![7p](../hai/pin7.gif)![9p](../hai/pin9.gif)![白](../hai/haku.gif)![东](../hai/ton.gif)![北](../hai/pei.gif)　　　碰![西](../hai/sha.gif)![西](../hai/sha.gif)![西](../hai/ysha.gif)

这种时候，还是该先门清把形整理一段再说。

剩下来的形实在太苦了。

如果这时别人立直，你根本没有押回去的力量，
而且还失去了本来安全的 ![西](../hai/sha.gif)。

这个代价很大。

## 该鸣的牌、该放过的牌

混一色里鸣牌最重要的点，就是：

**不要把手牌分断。**

![2m](../hai/man2.gif)![5m](../hai/man5.gif)![8m](../hai/man8.gif)![8m](../hai/man8.gif)![9m](../hai/man9.gif)![发](../hai/hatu.gif)![发](../hai/hatu.gif)　　　碰![3m](../hai/man3.gif)![3m](../hai/yman3.gif)![3m](../hai/man3.gif)　碰![中](../hai/ytyun.gif)![中](../hai/tyun.gif)![中](../hai/tyun.gif)

一旦这样碰，![2m](../hai/man2.gif) 和 ![5m](../hai/man5.gif) 就会变得非常难用。

结果当然也有可能和成混一色对对和跳满，
但那也只是“碰巧”顺利而已。

更多时候，还是会把和牌逃掉。

像这样靠鸣牌把自己做成愚形的行为，要尽量避免。
