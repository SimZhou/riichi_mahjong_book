# 鸣牌技巧（1）

这一组开始进入更偏技巧层面的副露内容。它们不是“会不会做役”的问题，而是：

1. 怎么用一次吃牌把手牌结构直接做轻
2. 怎么用副露解除受入重叠这种笨重结构
3. 以及在鸣牌之前，怎样先把理牌摆好，避免把自己的等待暴露给全桌

## 第一部分：吃后延展，是用一次吃牌把一组牌延展成两组面子

### 例 1：混一色里最典型的吃后延展

第一例里，若上家打出 `3m`，直接吃成 `234m`，看似只是拿了一组顺子，实际收益更大：

1. 原来 `22m34m` 这一段很重
2. 吃完以后，万子下段直接展开成两组可用结构
3. 整手一下就轻了很多

这种“通过吃牌把原本一团挤在一起的牌延长、分流”的做法，就是吃后延展。

它在混一色里尤其重要，因为染手最怕牌块互相挤住、动不了。

### 例 2：机会手里，也要敢于用吃后延展换速度

第二例是更贴近赤牌实战的题。门清做下去，理论上甚至有跳满空间；但实战上如果上家开始切索子，更好的选择往往是立即吃牌，用吃后延展做成 `断幺九 + 宝牌 3` 的快手。

原页这里强调的不是“高打点不要门清”，而是：

1. 你当然可以继续做大
2. 但如果副露之后已经足够满贯
3. 那就应优先把速度兑现

因为从这副牌来看，副露后一向听能接受的关键牌明显更多，而且还保留直接吃听的可能。简单说：

**已经够贵的手，不必执着理论最高点，要先保证它真的能和出来。**

## 第二部分：双面子处理，是用鸣牌把受入重叠直接拆掉

### 例 3：关键牌一出现，就该用吃牌把笨重结构做轻

第三例的核心，是两组典型的笨重结构：

1. `3344m`
2. `5778p`

这两组牌虽然看起来都“有形”，但本质上都带着明显的受入重叠问题。也就是说，它们不是真的轻快，而是要多绕一步才能变成好用面子。

所以这里一旦上家打出关键牌，就应立刻鸣：

1. `2m-5m` 这段能把 `3344m` 直接拆轻
2. `6p` 能把 `5778p` 直接整理成更自然的结构

鸣完之后的价值，不只是“变成一向听”，而是：

1. 后续再吃牌更容易
2. 上家若再次摸到相关牌，也更可能继续打出
3. 整副手的推进节奏会明显比门清时轻快

这也是原页最重要的一个判断：

**当受入重叠能靠鸣牌直接解消时，应尽量解消，不要抱着笨重结构硬拖。**

## 第三部分：但这类技术非常怕理牌不当，把等待直接摆给别人看

### 例 4：鸣牌之前就要先想好，别人看得到什么

第四例讲的不是“该不该吃”，而是“你怎么摆牌”。

如果这手直接把 `1p` 吃出来，而原手牌摆放又很直白，那么别人从你手牌露出的顺序，很容易反推出：

1. 下段筒子还留了很多
2. 真正本命多半就是 `1p`、`4p`
3. 甚至连 `2p` 或边 `3p` 这种次候选都能看得八九不离十

这就属于把自己的等待变成“玻璃听牌”。

所以这页最后给出的提醒非常实用：

1. 如果你准备做吃后延展、双面子处理这类鸣牌
2. 应该在鸣牌发生前，就先把手牌摆得更自然
3. 或者在吃牌瞬间快速把将要亮出的两张牌右移，减少信息暴露

这不是花活，而是很实际的隐藏信息能力。

## 这一页的结论

1. 吃后延展是通过一次吃牌，把拥挤结构直接延展成两组可用面子
2. 对已经够贵的机会手，副露后的满贯速度往往比门清上限更重要
3. 受入重叠若能靠鸣牌直接解消，应尽量解消，让手牌做轻
4. 这类技巧的副作用是容易暴露剩余结构，因此理牌和亮牌顺序必须提前考虑
5. 好的鸣牌技巧，不只是“会吃”，还包括“吃完以后别人看不出什么”
