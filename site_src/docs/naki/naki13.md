# 鸣牌技巧（1）

接下来分 3 页介绍一些实战里很有用的副露技巧。

这类小技巧，对熟手来说往往是自然就会用的东西；

但不熟的人，常常根本想不到“这里其实可以鸣”，或者即使想到了，也会本能地抗拒鸣牌。

把这些东西记住，不会吃亏。

尤其在带赤牌的麻将里，这些技巧会是很大的武器。

这是一种通过吃牌直接增加一组面子的技巧。

在混一色里尤其常见。

## 双面子处理

这是把连对形通过吃牌直接拆成两组面子的技巧。

### 例 3

![3m](../hai/man3.gif)![3m](../hai/man3.gif)![4m](../hai/man4.gif)![4m](../hai/man4.gif)![5p](../hai/pin5.gif)![7p](../hai/pin7.gif)![7p](../hai/pin7.gif)![8p](../hai/pin8.gif)![5s](../hai/sou5.gif)![6s](../hai/sou6.gif)![发](../hai/hatu.gif)![发](../hai/hatu.gif)![发](../hai/hatu.gif)

役牌暗刻在手时，人很容易产生“太可惜了，还是想门清”的心理。

所以这种手，很多人都会想门清去做三暗刻，或者顺便等万子的一杯口。

像 ![4s](../hai/sou4.gif)、![7s](../hai/sou7.gif) 这类牌，当然可以先放过。

但真正的关键牌，还是该鸣。

这手的 key 牌，是 ![6p](../hai/pin6.gif) 和 ![2m](../hai/man2.gif) 到 ![5m](../hai/man5.gif) 这一带。只要上家打出，就应该鸣。

之前在牌理里已经讲过，像 `3344m` 和 `5778p` 这样的两度受，本质上都属于愚形。

这里的思路，就是用鸣牌把这种愚形直接解消掉。

鸣完以后，整手会一下子变得很轻。

变化 2：![3m](../hai/man3.gif)![4m](../hai/man4.gif)![4m](../hai/man4.gif)![7p](../hai/pin7.gif)![8p](../hai/pin8.gif)![5s](../hai/sou5.gif)![6s](../hai/sou6.gif)![发](../hai/hatu.gif)![发](../hai/hatu.gif)![发](../hai/hatu.gif)　　![6p](../hai/ypin6.gif)![5p](../hai/pin5.gif)![7p](../hai/pin7.gif)

这不只是把手牌做成了可以吃听的一向听。

变化 1 里，如果上家之后又摸到 ![2m](../hai/man2.gif) 到 ![5m](../hai/man5.gif) 这些牌，再打出来的概率也会高很多。

变化 2 同理，![6p](../hai/pin6.gif) 到 ![9p](../hai/pin9.gif) 一带，都会明显比原来更好鸣。

所以，只要两度受能靠鸣牌直接解消，就应该尽量去解消。

但要注意：**理牌必须小心。**

这种牌一旦散成这种样子，别人一眼就能看出：你剩下的是筒子下段。

对高手来说，这就是玻璃听牌。

他们会马上意识到：

**`1p-4p` 是绝对本命，别的最多也就是 `2p` 或边 `3p`。**

所以，在吃 ![1p](../hai/pin1.gif) 之前，你就该先把理牌做一点调整。

例如可以事先把牌理成这样：

例：![5p](../hai/pin5.gif)![7p](../hai/pin7.gif)![7p](../hai/pin7.gif)![2p](../hai/pin2.gif)![2p](../hai/pin2.gif)![3p](../hai/pin3.gif)![3p](../hai/pin3.gif)![9p](../hai/pin9.gif)![9p](../hai/pin9.gif)![9p](../hai/pin9.gif)

这次介绍的吃后延展和两度受解消，都会让手牌以不太自然的方式残留下来。

所以等你熟练之后，最好要么：

1. 在鸣牌发生前，就预先按“等会儿要鸣”的方式去理牌
2. 要么在喊出“吃”之前，迅速把待会儿要亮出的两张牌往右移

这样才能尽量减少信息暴露。
