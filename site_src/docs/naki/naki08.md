# 食断

断幺九之所以是最常见的副露役之一，正是因为它**鸣了也成立**。

这让它在很多对局里都像一条“随时可切入的快攻路线”。但问题也正出在这里：

1. 太容易做
2. 看起来太容易和
3. 很多人因此把食断当成默认动作

这一页要解决的不是“食断能不能做”，而是：

**什么情况下该做，什么情况下宁可继续门清。**

## 第一部分：好手牌不要随便为了 1000 点去开食断

### 例 1：本来就是机会手，过早鸣牌只会把上限砍掉

第一例里，手牌本身形状优良，还有宝牌因素，明显属于值得认真做大的手。

这种牌如果从序盘开始就为了食断去鸣，问题不在于“能不能和”，而在于你把一手本来有很高上限、还能继续改良的机会手，主动压成了便宜副露手。

所以食断的第一条原则很简单：

**牌本来就漂亮、上限本来就高时，不要急着为 1000 点把门清价值卖掉。**

## 第二部分：有两张以上宝牌或赤牌时，食断就很值得积极出手

### 例 2：食断一旦带上 2 张以上宝牌，已经不是廉价快手

第二例里，自摸到第三张 `7s`，而 `7s` 本身又是宝牌。此时如果先切 `1s` 确定断幺九，再对 `456m`、`345p` 这类中张牌积极吃牌，整手的价值已经完全不同。

因为食断最怕的，是：

1. 打点太低
2. 为了速度牺牲太多
3. 和出来也不值得

但当你已经握有两张以上宝牌或赤牌时，上述问题会大幅缓和。此时“鸣了尽快把和牌兑现”往往比执着门清更合理。

原页这里的判断可以直接转成实战语言：

**食断 + 2 宝牌以上，就该把它当成可靠进攻手，而不是廉价凑合手。**

## 第三部分：只要和牌就够的局面，食断要比平时积极得多

### 例 4：终局抢名次时，1000 点本身就可能值一整局

在南四、争一位、争保二之类局面中，评价标准会和东一局完全不同。

第四例就是这种思路：平时你可能会舍不得把这样一副还算漂亮的牌开掉，但如果当前局面只要求“先和就行”，那食断 1000 点完全够用。

这时不该被“这手牌好可惜”牵着走，而应先看局况：

1. 只要和牌就能逆转或守住名次
2. 那么速度就是唯一核心指标
3. 食断因此会从“可选路线”变成“优先路线”

所以这里就算需要连续吃 `345p`、`678s` 之类的中张块，也应果断出手。

## 第四部分：门清节奏已经赶不上时，食断是追上牌局速度的手段

### 例 5：图里的重点不是“有图”，而是图说明了你已经没时间慢慢做大手

第五例是这页最重要的图示局面，不能只看图片本身，而要看它说明的局势。

手牌虽然还是满贯候选，但当前仍处在两向听，离门清立直还有一段距离。与此同时，场上已经出现了明显的速度信号：

1. 其他两家已经开始副露
2. 牌局节奏明显变快
3. 如果你还继续坚持门清，很可能还没听牌，别人就先和了

<img src="../images/36.gif" width="432" height="461" />

因此这张图的真正结论是：

1. 此时应当对 `2s` 果断碰牌
2. 目的不是“便宜也无所谓”，而是先把自己从落后节奏里拉回来
3. 如果你是亲家，更该比子家更早决定鸣牌，因为亲番的连庄价值更高，不能把巡目白白拖掉

也就是说，这张图不是单纯的配图，而是在强调一个实战判断：

**当场上速度已经快到门清路线大概率来不及时，食断就是把和牌率重新拉回来的应急方案。**

## 第五部分：但烂手不能因为“反正能食断”就乱开

### 例 6：牌又烂又便宜时，食断不是救命药，只会把自己锁进危险状态

最后一个例子是对常见误区的反击。

有些人会觉得：“反正门清也难和，不如吃碰做食断碰碰运气。” 这类想法通常最危险。

因为这种垃圾手一旦硬开食断，会同时出现几个问题：

1. 牌形依旧很苦
2. 打点仍旧很低
3. 一旦有人立直，你几乎没有安全回头空间
4. 最终甚至会变成“食断都未必和得出”的尴尬局面

所以这种牌通常应继续门清，能改良就改良，改不动就准备收手。只有像原页说的那种极端局况，例如南场亲番、此局亲落几乎等于确定垫底，才会被逼到用这种几乎是赌博式的食断。

## 这一页的结论

1. 好手牌、有上限的手，不要过早为了 1000 点去开食断
2. 食断只要带上 2 张以上宝牌或赤牌，就值得积极鸣牌兑现速度
3. 终局只求和牌的场况里，食断的优先级会显著上升
4. 当场上已经明显加速、门清大概率来不及时，食断是追上节奏的重要手段
5. 图中的 `36.gif` 说明的正是这种“因为牌局速度而被迫转快”的判断
6. 又慢又烂又便宜的手，不能因为“还能食断”就乱鸣，否则风险远大于收益
