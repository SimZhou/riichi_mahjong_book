# 第5章：鸣牌

<img src="../images/top-image8.jpg" width="610" height="300" alt="鸣牌章节主图" />

1. [鸣牌的优点与缺点](./naki01.md)
   要想把鸣牌用好，先理解它的长处和短处很重要。也常叫作“副露”或“吃碰仕挂”。

2. [副露判断的基础](./naki02.md)
   该鸣还是不该鸣，要怎样判断。

3. [碰与吃（1）](./naki03.md)
   碰和吃都能利用他家打出的牌，但这两者之间其实有区别。

4. [碰与吃（2）](./naki04.md)
   当手里有多个可以碰的对子，或有几种不同的吃法时，该怎么处理。

5. [杠（1）](./naki05.md)
   和碰、吃一样，杠也同样有优点和缺点。

6. [杠（2）](./naki06.md)
   明明手里有杠材，却不该去杠的时候。

7. [鸣牌与向听数](./naki07.md)
   鸣牌时要注意向听数。这一页也会讲到“向听数不变的鸣牌”。

8. [食断](./naki08.md)
   什么时候应该从门清手里主动鸣出去做食断。

9. [对对和](./naki09.md)
   这是初学者特别喜欢的役种，但当然也有不该去做对对和的时候。

10. [混一色](./naki10.md)
   做混一色时，也有不该鸣出去的牌。那会是什么牌呢？

11. [后付（1）](./naki11.md)
   在役还没有确定的状态下，先把别的部分鸣出去，这就叫“后付”。

12. [后付（2）](./naki12.md)
   有些手牌和场面，比起门清推进，反而更适合后付副露。

13. [鸣牌技巧（1）](./naki13.md)
   来记一些能在实战里用上的鸣牌小技巧。鸣牌技术会直接影响和牌率。

14. [鸣牌技巧（2）](./naki14.md)
   介绍红牌麻将里很常用的食断技巧。

15. [鸣牌技巧（3）](./naki15.md)
   虽然真正用到的场面不算很多，但记住了也不会吃亏。
