# 鸣牌

鸣牌是麻将里最容易拉开水平差距的技术之一。

很多人会在两端犯错：

1. 该鸣的不鸣，白白把能和的牌拖成和不到
2. 不该鸣的乱鸣，把本来能做成门清高质量手的牌活活做成廉价副露手

所以这一章的重点，不是教你“多鸣”或“少鸣”，而是建立一整套副露判断框架：

1. 鸣牌会换来什么速度收益
2. 你为此牺牲了多少打点和守备
3. 当前局面到底需不需要这份速度
4. 不同副露方式会怎样改变手牌结构和待牌质量

## 本栏内容

1. [鸣牌的优缺点](./naki01.md)
: 先把副露的收益和代价拆清楚，明确它本质上是“速度换打点与守备”的技术。

2. [副露判断基础](./naki02.md)
: 把鸣或不鸣整理成一套可执行的比较方法，从速度、打点、守备和局况四方面称重。

3. [碰与吃（1）](./naki03.md)
: 先讲清碰和吃的速度差、消一发时的取法，以及同牌可碰可吃时的基本比较。

4. [碰与吃（2）](./naki04.md)
: 继续比较碰材优劣与多种吃法，重点看鸣完以后剩下的牌形质量。

5. [杠（1）](./naki05.md)
: 先把杠的收益与风险拆开，说明为什么杠从来不是“能杠就杠”。

6. [杠（2）](./naki06.md)
: 进一步整理哪些局面明明能杠却不该杠，重点看受入、待牌、立直与局况风险。

7. [鸣牌与向听数](./naki07.md)
: 用向听数衡量鸣牌是否真的让手前进，并给出二副露、三副露时的实战下限。

8. [食断](./naki08.md)
: 说明什么情况下该用食断抢速度，什么情况下又不能为了便宜快手把高质量门清牌开坏。

9. [对对和](./naki09.md)
: 先分清哪些牌是真能走对对和，哪些只是对子稍多却在硬凑。

10. [混一色](./naki10.md)
: 重点讲混一色副露时如何避免自断手牌，以及什么时候该坚持门清高打点。

11. [后付（1）](./naki11.md)
: 从速度、守备和隐蔽性三方面重估后付，不再把它简单视为“勉强有役”。

12. [后付（2）](./naki12.md)
: 继续处理急所牌已现、双路线后付，以及南场只求和牌时的全鸣抢和判断。

13. [鸣牌技巧（1）](./naki13.md)
: 讲食い伸ばし、双面子处理，以及这些鸣牌前后的理牌与信息隐藏。

14. [鸣牌技巧（2）](./naki14.md)
: 进入食断进阶，重点是放弃最高形幻想、把高宝牌与赤牌手更快兑现成和牌。

15. [鸣牌技巧（3）](./naki15.md)
: 收尾于干扰型鸣牌，包括一发消除、干扰碰、海底挪位与技术型杠的局况运用。
