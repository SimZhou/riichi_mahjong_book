# 第1章：麻将基础

![麻将基础主图](../images/top-image4.jpg)

## 本栏目录

1. [麻将是什么](./kihon01.md)
: 麻将到底是一种什么性质的游戏？这里先做一段包含作者麻将观在内的引言。

2. [麻将与概率](./kihon02.md)
: 介绍麻将这项游戏里“概率”这一问题的基本思考方式。

3. [听牌与向听](./kihon03.md)
: 在以“和牌”为目标时，先确认“听牌”和“向听”这两个绕不过去的概念。

4. [面子与搭子](./kihon04.md)
: 说明面子的基本想法，也会涉及边张与嵌张的比较。

5. [数牌与字牌](./kihon05.md)
: 说明 1 到 9 的数牌各自带有怎样的意义，以及字牌所包含的役等内容。

6. [对子（1）](./kihon06.md)
: 讲解作为雀头或面子候补的两张一组配对，也就是“对子”的基础思路。

7. [对子（2）](./kihon07.md)
: 即使同样是对子，优劣也并不相同。这里介绍做牌时很重要的一部分判断。

8. [基本形与复合形](./kihon08.md)
: 麻将的目标是完成 4 面子 + 1 雀头，但走向和牌的过程并不只有一种。这里开始说明该如何把手牌推进到和牌。

9. [复合形（1）](./kihon09.md)
: 介绍三张复合形的思考方式，也会谈到两嵌的处理。

10. [复合形（2）](./kihon10.md)
: 介绍四张复合形。这是理解复杂多面听基础时很重要的一页。

11. [麻将的听牌（1）](./kihon11.md)
: 介绍两面、单骑、嵌张等最具代表性的麻将待牌形。

12. [麻将的听牌（2）](./kihon12.md)
: 进一步介绍多种待牌交织而成的变则复合形。
