# 麻将是什么

这套教程会以“牌理”和“实战判断”为主轴，系统讲清楚麻将里那些真正影响成绩的东西。

我会默认读者已经知道麻将规则和常见术语。如果还不熟，最好先补基础规则，再回来看这一套内容。

## 麻将与运气

“麻将里到底有没有运气、手气、气流？”

这是讨论很多年的老问题。先把结论说清楚：

1. 麻将当然非常吃运气
2. 但“手气”并不是一个可以拿来指导打法的可靠理论

从长期看，水平高低最终还是会体现在成绩上。但现实里，一个人不可能连续打上千个半庄，把短期波动完全抹平。多数时候，你感受到的胜负，很大程度上就是当天牌运的结果。

一晚打两三个半庄，或者一场牌局里连续几手摸得很好、连续几手什么都做不了，这都很正常。问题在于：

**这种波动存在，不代表你能靠“顺势”“借流”之类的说法把它利用起来。**

如果只是因为“我今天状态很好”“这把感觉气在我这边”，就去硬顶危险牌，那本质上不是读懂了运势，只是给自己找了一个冲动进攻的理由。

在这种点棒状况下，合理选择通常不是逞强，而是老老实实退守。像把发的暗刻拆出来当安全牌，这种处理才符合牌局利益。

所以这里最重要的一点是：

**不要用“手气”“气流”给不合理的打牌找借口。**

麻将当然有运气，而且运气经常决定短期结果；但它并不会稳定地给你提示“现在该拼”还是“现在该撤”。

## 麻将的目的

来学麻将的人，通常都会有某种“想变强”的目标。比如：

1. 提高实战收支
2. 提高线上平台评分
3. 冲更高的段位、称号或排名

但这并不意味着所有人打麻将时的价值取向都完全一样。

有人最在意长期成绩，有人很看重“能不能拿第一”，也有人就是喜欢做大牌、做役满。就算从效率角度看不是最优，只要是规则允许范围内的个人选择，本来就没有必要拿自己的价值观去压别人。

这套教程的立场会比较明确：

**我写这些内容，目标是帮助你提高长期成绩。**

所以后面会不断强调：

1. 什么打法长期更赚
2. 什么判断更稳定
3. 什么选择只是看起来帅，实际却容易亏

但这不等于我在说“除了这里写的打法，其他都不许打”。它只是告诉你：如果目标是把成绩打上去，那么哪些选择更值得优先学习。

## 这一页的结论

1. 麻将非常吃运气，但“手气理论”不适合作为实战依据
2. 短期胜负波动很大，不能用单场结果倒推打法对错
3. 这套教程以“提高长期成绩”为主要目标
