# 立直还是默听（3）

前一页讲的是好形听牌几乎都应立直。这一页开始处理更难的一类：

**愚形听牌要不要立直。**

这里不能一句话概括，因为愚形听牌之间差异很大：

1. 有没有役
2. 有没有宝牌
3. 手变多不多
4. 巡目早不早

这些因素都会直接改变答案。

## 第一部分：无役愚形听牌，只要带宝牌，通常也应立直

### 例 1：只剩单一路手变时，不如直接立直把即时报酬拿到手

第一例看似是很想继续等手变的牌，因为当前是愚形听牌。但如果把这手真正能改善的路线拆开看，会发现最像样的手变几乎只剩 `6m` 这一种。

这时继续默听的问题在于：

1. 你并没有明确的荣和资格，必须靠未来手变把牌做顺
2. 另一方面，立直已经能立刻解决无役问题
3. 又因为有宝牌在，立直后的得点收益并不低

所以原页的意思非常直接：与其期待唯一那一路手变，不如直接立直。即使将来真摸到 `5m` 之类牌，也未必比现在立直更划算。

### 例 2：无役愚形 + 宝牌，依然是“愚形立直上等”

第二例继续补强这个原则。虽然这里等待仍然不好，但只要是无役愚形听牌，而且手里有 `1` 张以上宝牌或红牌，立直通常就占优。

原因不是说愚形会突然变好，而是：

1. 立直后打点抬升幅度大
2. 一发、里宝牌的附加收益也更值钱
3. 愚形仍然可能靠后引挂、单张见切、筋判断失误而荣出

所以无役愚形听牌的默认答案，不是“先忍着”，而是“只要带点数材料，多数直接立直”。

## 第二部分：但极早巡、手变很多的愚形听牌，可以默听

### `25.gif` 这张图说明的，是“巡目 + 手变数量”足以压过即立直收益

这一页唯一的独立示意图，就是例外场面。

<img src="../images/25.gif" width="383" height="406" />

这张图展示的是非常早的巡目，才第 `2` 巡就已摸成宝牌 `2` 的愚形听牌。若放到普通场面，很多人会觉得“有宝牌，那就立直吧”。但原页在这里明确反对。

关键不在图本身，而在这副牌同时满足了两个条件：

1. 巡目极早，后面还有大量摸牌回合
2. 手变非常丰富，万子部分有多条路能改成两面，甚至还能碰宝牌改良受理

在这种局面里，立直并不是不能打，而是太急。因为未来还有很多机会把它从“难和愚形”变成“更宽更高的听牌”。哪怕最后只是直接自摸当前愚形，也还有 `1300/2600` 的兑现，不算太亏。

所以 `25.gif` 要表达的重点是：

**愚形听牌若巡目极早、手变又显著丰富，默听才是真正值得讨论的例外。**

## 第三部分：无役愚形时，甚至还可能选择不取听

### 例 3：与其立直 `1300`，不如先把听牌拆掉继续养牌

第三例是这一页最重要的进阶判断。

很多人以为“愚形听牌了，就只剩立直或默听两种答案”。原页专门强调并非如此，还有第三个选项：**不取听**。

这手在平场里，若只为一个立直 `1300` 去强行保住当前听牌，其实非常亏。因为整副牌仍有大量自然伸展空间：

1. 索子仍能继续做宽
2. 筒子形也不差
3. 甚至连宝牌附近的万子都有成长空间

因此这里更好的处理不是立直，也不是勉强默听，而是干脆切 `1m`，先把听牌拆掉，回到一向听继续追更高质量的听牌。

这就是所谓的“听牌不取”。它不是保守，而是拒绝把一副本来能长大的牌，硬锁进低质量小和路线。

## 第四部分：有役愚形听牌时，按打点分层判断

### 例 4：有役但手变很少时，除非当前已经特别高打点，否则仍可立直

第四例是役有愚形听牌。这里虽然默听已经能荣和，但如果真正能改善的手变只剩非常少的几种，那么多数情况下仍可直接立直。

原页这里甚至给出近似判断：除非宝牌本身就是刻子、默听打点已经非常高，否则还是立直更自然。因为你虽然已经有役，但愚形带来的和牌困难并没有消失，立直至少还能把点数和自摸收益继续抬高。

### 例 5：有役愚形 + 手变尚可时，可以先默听再看

第五例是有役愚形里更偏向默听的一型。因为这里的手变并不只是一张两张，而是有一组相对成体系的改良路线。

所以这时可以先默听，等：

1. 摸进关键改良牌再转立直
2. 或者场况变化后再重新判断

但原页同时补了一条重要限制：如果这种手一变张，原有手役反而会消失，那么判断方式就要重新切换。此时可以粗略记一条经验线：

1. 默听已有 `5200` 以上，愚形也可以先默听
2. 若只有 `2600` 左右，则多数还是应立即立直

原因很简单。愚形听牌本来和牌率就有限，若当前点数又低，再不立直去争附加收益，整手的兑现价值就太差了。

## 这一页的结论

1. 无役愚形听牌只要带宝牌或红牌，多数都应立即立直
2. `25.gif` 说明的例外是：极早巡、手变丰富的愚形听牌，可以先默听养牌
3. 愚形听牌不只有“立直 / 默听”两种选择，很多牌还应直接不取听
4. 有役愚形听牌若手变很少，除非当前已是高打点，否则仍可立直
5. 有役愚形听牌若手变成体系，可先默听；粗略经验线是 `5200` 可偏默听，`2600` 则偏立直
