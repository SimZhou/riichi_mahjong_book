# 立直理论（1）

前面几页把“立直还是默听”的基本牌姿判断拆开讲完了。这一页开始转向更通用的立直理论。

第一条理论，其实很朴素：

**该立直就立直，而且要在听牌当下立直，不要拖几巡再说。**

很多初学者最大的问题，不是不知道立直有好处，而是总觉得“再看看”“也许下一巡更好”。原页这一页就是在专门纠正这种拖延型判断。

## 第一部分：如果已经决定立直，就必须当场立直

### 例 1：为了两种手变而先默听几巡，通常是亏的

第一例里，手里确实存在 `4p`、`1m` 这类手变，看上去似乎值得等等三色或更好形。但原页直接否定这种思路。

原因很简单：

1. 这些手变并不多
2. 在你等待手变的几巡里，本来可能被和掉的牌会直接流走
3. 更糟的是，别人可能打出你现在还不能和的牌，等你以后再立直时，这些机会已经永久损失

所以这里正确答案不是“先默听数巡，再找机会立直”，而是听牌当下就把立直判断做完。若结论是立直，就立即立直。

原页里还特地嘲讽了那种“我在找一发时机”的想法，因为你每多等一巡，一发本身就更不可能发生。想靠拖几巡来提升一发率，本身就是自相矛盾。

### 例 2：高低目差也不是拖立直的理由

第二例也是同样的病灶。有人会想：先默听着，如果先出低目就见逃，过几巡再立直，或者等高目更稳再宣言。

但这类操作基本都是亏的，因为：

1. 你把能和的窗口主动缩短了
2. 低目如果现在就出了，你原本是能直接兑现的
3. 立直的威压、一发、里宝收益也被你自己往后拖没了

所以这页第一条总原则就是：

**立直与否只在听牌那一刻判断一次。判断为立直，就立即执行。**

## 第二部分：不要太执着“这张牌看起来不容易出”

### 例 3：立直判断的基准，首先是自摸，不是别人会不会正好打出来

很多初学者不敢立直，核心原因是“这张不会出的”。

第三例就是针对这种心理。哪怕是跨宝牌、看起来容易被回避的两面待，正确想法也不是“反正荣不到，不如不立”，而是：

1. 只要仍是足够枚数的良形
2. 你就本来有很高自摸可能
3. 别人出不出，只是附加收益，不是唯一收益来源

这也是立直理论里非常重要的一点：

**立直判断首先按自摸期望算，再把荣和当额外增益，而不是倒过来。**

### 例 4：就算是非常难出的边张，仍然应先想“能不能自摸到”

第四例更极端，是看上去几乎荣不到的宝牌边张。

如果你只用“会不会出”来判断，结论很容易变成默听甚至放弃。但原页强调，这种牌并不是没价值，因为：

1. 自摸仍然存在
2. 立直以后和牌值会显著提高
3. 只要自摸一次就足够把整手兑现到很高收益

所以除了单骑这种极特殊情况，原页认为“待牌是否容易被别人打出”不应成为立直判断里的第一优先级。更可靠的基准，仍然是：

1. 这副牌的剩张数
2. 这副牌立直后的总体期望
3. 以及在剩余巡目里，自摸兑现的概率

## 这一页的结论

1. 立直与否只在听牌当下判断一次；若决定立直，就必须立即立直
2. 先默听几巡再立直，多数只是把原本的和牌窗口白白让掉
3. “找一发时机”“先看看低目会不会出”这类拖延思路，通常都是亏的
4. 立直判断的基准首先是自摸期望，而不是别人会不会刚好打出这张牌
5. 除单骑等特殊场面外，“牌好不好出”不应压过枚数与自摸率本身
