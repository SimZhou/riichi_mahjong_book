# 立直理论（2）

这一页继续讲立直的通用原则，但重点不再是“立直还是默听”，而是：

**一旦准备立直，听牌该怎么整理得更适合立直。**

原页给了三条很实用的经验：

1. 立直时，挂内不挂外，伸外不伸内
2. 没有平和时，应优先保留暗刻可能
3. 舍牌最好尽量朴素，少暴露信息

## 第一部分：挂内不挂外，伸外不伸内

### 例 1：若已经决定立直，切牌思路要从“最好进张”转成“最好荣和”

平时牌理里，像 `124s` 这种结构，大家都知道应切 `1s`，去争以后摸 `5s` 的两面改良。

但原页说得很明确：一旦这副牌本来就大概率要立直，思路就要变。

第一例里，正确的立直取舍反而是切 `4s`。原因不是牌效率突然反了，而是：

1. 立直以后，更希望留下外侧延展出来的等待
2. 若场上先出现现物 `4s`，`3s` 这类牌会更容易被人打出
3. 也就是说，立直时要更多考虑“将来哪种形更容易荣和”

原页把它浓缩成一句话：

**挂内，伸外。**

当然它也补了一条务实限制：这个差距并没有大到压过安全度。若 `4s` 明显更危险，那切 `1s` 立直也完全可以。

## 第二部分：没有平和时，要优先保留暗刻机会

### 例 2：能做暗刻、能暗杠的路线，比看上去平整的平和外形更值钱

第二例里，`2p` 切和 `3p` 切都能立直，但原页推荐保留暗刻可能。

原因是这手本来就没有平和，说明你并不靠“符平番数低但速度高”的模式吃饭，而更应该考虑：

1. 若再摸到同牌，可形成暗刻
2. 有时甚至还能暗杠
3. 符数会直接抬高，杠宝牌也有额外上限

所以这类手立直时，不该只看“哪种形看起来顺”，而要先看：

**哪一边留下来后，未来更容易让这手涨点。**

## 第三部分：立直河最怕信息太多，最好让别人读不清你的等待

### 例 3：与其做花哨迷彩，不如让舍牌本身尽量没有特征

原页这一段其实是在反对旧式“迷彩舍牌”思路。与其故意做一些花活让别人猜错，不如直接把立直河做得朴素、难读。

第三例里，正确处理是把雀头从 `北` 换成 `4p`，然后切 `北` 立直，而不是反过来切 `4p`。

因为若你先切 `4p`，等到立直时，别人会立刻得到很多额外信息：

1. `1p-4p-7p` 一带更容易被排除或反推
2. 某些两面、搭子形会被明显缩窄
3. 整体等待更容易被猜中

而如果切的是一张字牌，别人通常只能知道“你大概不是字牌单骑”，剩下的信息非常少。

原页最后甚至把这种“全是一九字牌、没什么特征”的立直河称为强河。它比刻意做出来的迷彩更实用，因为它真能逼别人在信息不足的情况下继续硬打。

## 这一页的结论

1. 一旦准备立直，取舍标准要从单纯牌效率，部分转向“哪种等待更容易荣和”
2. 所谓“挂内伸外”，就是立直时更偏向保留外侧延展、让等待更好出的结构
3. 没有平和的手，立直整理时应优先考虑暗刻、暗杠与符数成长空间
4. 立直河应尽量减少可读信息，不必追求花哨迷彩，朴素强河通常更实用
5. 若结构差异不大但某张牌明显更危险，安全度仍然优先于这些细微立直技巧
