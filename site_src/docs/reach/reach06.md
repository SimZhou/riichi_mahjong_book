# 立直理论（2）

## 挂内，伸外

例如手里有 ![1s](../hai/sou1.gif)![2s](../hai/sou2.gif)![4s](../hai/sou4.gif) 这种结构时，

平时当然会去考虑摸 ![5s](../hai/sou5.gif) 后变成两面，所以切 ![1s](../hai/sou1.gif) 是很正常的打法。

但如果这手大概率是要立直的，那思路就会变。

### 例 1

![1m](../hai/man1.gif)![2m](../hai/man2.gif)![2m](../hai/man2.gif)![2m](../hai/man2.gif)![3m](../hai/man3.gif)![1s](../hai/sou1.gif)![2s](../hai/sou2.gif)![4s](../hai/sou4.gif)![6s](../hai/sou6.gif)![8s](../hai/sou8.gif)![西](../hai/sha.gif)![西](../hai/sha.gif)![西](../hai/sha.gif)　　自摸![7s](../hai/sou7.gif)　　宝牌![6s](../hai/sou6.gif)

这种牌大多数都会去立直。

而一旦决定立直，定式就是切 ![4s](../hai/sou4.gif)。

因为如果场上先打出一张现物 ![4s](../hai/sou4.gif)，接下来 ![3s](../hai/sou3.gif) 就会更容易被打出来。

当然，这个效果也没有强到压过一切。

如果 ![4s](../hai/sou4.gif) 明显更危险，那切 ![1s](../hai/sou1.gif) 立直也完全没问题。

## 暗刻优先

### 例 2

![6m](../hai/man6.gif)![6m](../hai/man6.gif)![6m](../hai/man6.gif)![1p](../hai/pin1.gif)![1p](../hai/pin1.gif)![1p](../hai/pin1.gif)![2p](../hai/pin2.gif)![2p](../hai/pin2.gif)![3p](../hai/pin3.gif)![5s](../hai/sou5.gif)![5s](../hai/sou5.gif)![6s](../hai/sou6.gif)![7s](../hai/sou7.gif)　　自摸![4s](../hai/sou4.gif)　　宝牌![3s](../hai/sou3.gif)

这题无论切 ![2p](../hai/pin2.gif) 还是切 ![3p](../hai/pin3.gif) 都能立直。

但像这种本来就没有平和的手，应该优先保留能做暗刻的路线。

如果之后再摸进 ![1p](../hai/pin1.gif)，就能暗杠。

这样不只会符跳，还可能顺手中杠宝牌。

## 信息越少越好

过去一度很流行所谓“迷彩”，也就是故意在舍牌里设下陷阱来误导对手。

但现在对此持否定意见的人占压倒多数。

**比起花哨迷彩，舍牌本身越没有特征越好。**

### 例 3

![5m](../hai/man5.gif)![6m](../hai/man6.gif)![4p](../hai/pin4.gif)![6p](../hai/pin6.gif)![7p](../hai/pin7.gif)![8p](../hai/pin8.gif)![2s](../hai/sou2.gif)![3s](../hai/sou3.gif)![7s](../hai/sou7.gif)![8s](../hai/sou8.gif)![9s](../hai/sou9.gif)![北](../hai/pei.gif)![北](../hai/pei.gif)　自摸![4p](../hai/pin4.gif)　　宝牌![9s](../hai/sou9.gif)

这是一手非常典型的立直牌。

这里应该把雀头从 ![北](../hai/pei.gif) 挪到 ![4p](../hai/pin4.gif)，然后切 ![北](../hai/pei.gif) 立直。

如果这里把 ![4p](../hai/pin4.gif) 切掉，等你立直以后，

对手就会更容易把 ![1p](../hai/pin1.gif) ![2p](../hai/pin2.gif) ![3p](../hai/pin3.gif) ![4p](../hai/pin4.gif) ![7p](../hai/pin7.gif) 打出来。

如果切 ![北](../hai/pei.gif)，那对手顶多只知道你不是单骑待，再多就读不出来了。

理想的立直河，应该是那种一九字牌排得很自然、没有明显特征的河。

比如像这样：

![南](../hai/nan.gif)![北](../hai/pei.gif)![中](../hai/tyun.gif)![1m](../hai/yman1.gif)

碰到这种立直河，对手往往就只能硬着头皮继续打。

比起刻意做迷彩，这种**强河**反而更容易让你荣和。
