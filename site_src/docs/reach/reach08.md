# 立直取听（2）

上一页先讲了最基础的三步比较：枚数、打点、出牌难度。

这一页继续往下处理更麻烦的题：

**如果是“枚数少但打点高”对上“枚数多但打点低”，到底该取哪边。**

原页给出的答案不是绝对公式，而是一组很实用的经验线：

1. 小分差时，宽度压倒打点
2. 一番差时，仍然通常是宽度更强
3. 两番差开始进入灰区，要看有没有宝牌
4. 三番以上差距，才多数转向高打点优先

## 第一部分：点数差很小时，宽度明显更重要

### 例 1：`3900` 对 `5200` 这种差距，不足以换掉明显更多的待牌张数

第一例里，一边是较宽的待牌，一边是立直一杯口等稍高打点路线。但原页判断得很清楚：只要只是 `3900` 和 `5200` 这种微差，当然选更宽那一边。

因为这类差距远不足以弥补：

1. 和牌张数变少
2. 自摸率下降
3. 被别人先和的概率上升

所以微差局面里，宽度几乎总是绝对优先。

## 第二部分：只差一番时，通常仍是宽度优先

### 例 2：即便点数翻倍，也未必比得过宽度翻倍

第二例看似打点差已经变大，从 `2600` 到 `5200`，几乎翻倍。但原页仍然判定是枚数更多的一边占优。

原因有两个：

1. 宽度的差距本身往往已经接近甚至超过一倍
2. 立直手还有里宝牌与满贯天井效应，高打点边的实际收益并没有纸面差距那么夸张

原页还额外提醒了一个常被忽略的问题：待牌一旦缩窄，你让别人先和的概率也会上升。这是负向期望，很多人算点数时会忘掉。

### 例 3：追立时，宽度的重要性会再上升一档

第三例是在场上已有先行立直者时的判断。此时若还为了多一番去主动把待牌压窄，就更亏。

因为追立本来就在拼“谁先摸到”。在这种比翻牌速度的局面里，枚数越多越重要，过度压缩待牌往往等于主动把拼摸牌的胜率送掉。

### 例 4：一番差的灰区里，仍是“宽度优先，少数薄牌例外”

第四例更接近实战灰区：

1. 宽边是两面平和立直
2. 窄边是坎张三色立直或三色默听

这类题原页没有说只能选一种，而是给出更现实的态度：默认还是取宽边；只有当高打点那边的关键牌仍然很厚、很好出时，才勉强可以偏向打点。

也就是说，一番差不是完全不看打点，但宽度仍是基准答案。

## 第三部分：两番差时，开始真正进入看宝牌的区域

### 例 5：无宝牌时，两番差通常该取高打点；有宝牌时，宽边会重新变强

第五例是这一页真正的分水岭。

原页把“两番差”拆成了三种情况：

1. **没有宝牌**：例如好形 `1300` 对愚形 `5200`，这时高打点边通常更优
2. **有 `1` 张宝牌或等价附加番**：例如好形 `3900` 对愚形 `5200`，这时两边会比较接近，属于典型灰区
3. **有 `2` 张以上宝牌**：这时好形立直往往又重新占优，因为宽边本身已经不低分，且立直后还有额外上限

这套分层很重要，因为它说明：

**两番差不是固定答案，要看宽边那一侧是否已经有足够分值托底。**

## 第四部分：三番以上差距，才多数转向高打点优先

### 例 6：宽边若只是普通手，而窄边能直接跳到大手，就应优先取高打点

第六例就属于典型“三番以上差”。这时原页判断很干脆：绝大多数情况下都应优先高打点。

因为这已经不是“多一点少一点”，而是整局价值层级不同。若宽边只是普通手，而窄边一旦和出就是明确的大手，那么缩窄等待通常是值得的。

### 例 7：但如果宽边本身已经很高，就别再贪到最差等待上去

最后一个例子又把这个原则往回拉了一下。

有时候虽然高打点边更高，但宽边本身也已经足够值钱。这时若你为了再多几番，硬把牌取成极难出的宝牌附近单薄等待，那就过线了。

原页在这里的态度很务实：

1. 宽边若已经够高，可以直接默听收和
2. 或者用较好的宽边立直去争自摸
3. 没必要为极限打点把整副牌碰成最差等待

也就是说，高打点优先也有底线，不是无条件越高越好。

## 这一页的结论

1. 点数微差时，宽度压倒打点
2. 一番差时，默认仍是宽度优先，尤其在追立场面更如此
3. 两番差进入灰区，关键看宽边这侧有没有宝牌或等价加分托底
4. 三番以上差距时，通常应转向高打点优先
5. 但若宽边本身已足够高，就不要为了极限打点把等待压到过差
