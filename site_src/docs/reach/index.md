# 立直

立直是现代立直麻将里最特殊、也最核心的手役之一。

它的强，不只因为“门清听牌就能多 1 番”，还因为它同时附带了几种极其重要的效果：

1. 直接解决无役问题
2. 带来一发、里宝牌等额外上限
3. 通过立直宣言限制全桌的出牌自由
4. 改变自己和他家对速度、风险与收益的判断

所以这一章不会只讲“听牌就立直”，而是逐步拆开：

1. 立直的收益和代价
2. 什么时候该立直，什么时候该默听
3. 牌姿、巡目、打点与局况如何共同影响立直判断

1. [立直的优缺点](./reach01.md)

2. [立直还是默听（1）](./reach02.md)
: 从手牌本身出发，先讲哪些高打点、差等待手更适合默听，哪些则仍应立直。

3. [立直还是默听（2）](./reach03.md)
: 继续处理好形听牌，重点是为什么大多数手变都不值得为了它放弃即立直。

5. [立直理论（1）](./reach05.md)
: 建立立直的第一条总原则：若决定立直，就要在听牌当下立即立直，不拖巡。
