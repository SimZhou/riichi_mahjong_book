# 立直还是默听（2）

上一页先讲了高打点手里，哪些该默听、哪些仍应立直。这一页继续往下走，专门讨论：

**如果听牌本身已经是好形，还要不要为了手变继续默听。**

原页的答案非常鲜明：除非极端例外，否则好形听牌就该立直，不要被“最高形”这种想象绑住。

## 第一部分：好形听牌在平场里，默认应立即立直

### 例 1：双碰字牌、双碰宝牌这类“易出牌”也算好形

这一页先把“好等待”讲得更具体。好形不只是两面、多面张，也包括一些实际非常容易荣和的双碰、字牌单骑之类等待。

例如第一例，`1p` 与 `北` 的双碰虽然枚数只有 `4` 张，但两边都是场上很容易被打出的牌，实际荣和率往往不输一些普通两面。

所以这里首先要建立的不是“几面张才算好”，而是：

**只要这副听牌本身已经很容易和，就该把它视为立直候选。**

## 第二部分：平和只要听牌，即使只有 1000 点也应即立直

### 例 2：现代立直麻将里，“平和 only 默听”已经是过时观念

过去常见的老说法是：平和只有 `1000` 点，拿 `1000` 点去押立直棒不划算，所以应默听。

但现在对立直收益的理解已经变了。因为“立直平和”从来不只是 `2000` 点，它平均还能吃到：

1. 一发
2. 里宝牌
3. 额外的自摸收益

综合期望下来，立直平和的平均和牌值明显高于单纯 `1000` 点默听。因此像例 2 这种牌，不管切 `1s` 还是 `4s` 成听，正确答案都是立即立直。

## 第三部分：为了单一路手变去默听，多数得不偿失

### 例 3：只差一种牌才能变出三色，不值得放弃立直

第三例是典型的“我再等等三色”心理。

问题在于，这类手变虽然看起来漂亮，但往往只有很少的进张，例如这里只靠一张 `5m` 才能把三色做实。实战里，和牌张先出来的概率通常远高于你刚好在那之前摸进唯一手变牌。

而立直这边的收益却是立即生效的：

1. 直接多 1 番
2. 一发、里宝牌马上进入结算池
3. 还会让全桌对你产生压制效果

所以像这种“为了一个单路高打点手变而默听”的做法，通常是亏的。

### 例 4：就算有两种三色手变，仍然多半不值得等

第四例比上一题稍微强一点，因为它同时存在 `234` 和 `345` 的三色变化，看上去手变比前一题丰富。

但原页的结论仍然是：即使如此，立直依然更优。

原因有两层：

1. 这些手变本身仍不稳定，而且就算摸到了，也未必最终按高目和牌
2. 平和宝牌 `1`、断幺平和这类 `2` 番手，本来就是立直增值效率最高的一类手

也就是说，越是这种“当前手本来就不差、立直后涨点效率又高”的牌，越不应执着未来不确定的再升级。

## 第四部分：极端多手变的好形听牌，默听才勉强能讨论

### 例 5：手变非常丰富时，默听可以作为少数例外，但即立直也完全没问题

第五例才是这一页真正意义上的例外。这里手变确实很多，`3m`、`6m`、`7m`、`4p`、`8p` 都可能把牌继续做大。这类牌默听就不再荒谬，因为未来确实有不少现实可见的强化路线。

但原页即便在这里也没有给出“必须默听”的结论，而是说：

1. 这才算能认真讨论默听的极端例外
2. 即便直接立直，也完全没有问题

因为绝大多数所谓“手变很多”的牌，其实都远没到这个程度。很多人只是被“最终最漂亮会变成什么样”吸走了注意力，却忽略了实战里往往在那之前就已经和牌或流失和牌机会。

所以原页最后直接否定了“最终形”“最高形”这种旧概念。麻将不是比谁想象到最美的成牌，而是比谁更快把高期望路线兑现出来。

## 这一页的结论

1. 平场里的好形听牌，默认就是立即立直
2. 平和只有 `1000` 点也不该默听，现代定式是平和 only 即立直
3. 只为单一路、双路手变去默听，多数都不划算
4. 平和宝牌 `1`、断幺平和这类 `2` 番手，本来就是立直增值效率最高的牌
5. 只有手变极端丰富的好形听牌，默听才勉强能讨论，但即立直通常也完全成立
