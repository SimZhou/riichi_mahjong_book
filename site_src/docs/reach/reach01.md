# 立直的优缺点

立直是非常特殊的手役。  
它没有牌形限制，还附带一发、里宝牌这些额外收益。  
实际上几乎可以看作有 1.5 番左右的价值，而且又是麻将里最容易做成的手役。

所以到现在为止，这套教程一直都在建议：

<span class="red b">“不要太拘泥手役，要尽量让自己更快听牌。”</span>

这就是现代流速度优先的打法。  
虽然这样一来没有手役的便宜手会变多，  
但那部分打点，就靠<span class="b">立直来弥补。</span>

先来确认一下立直到底意味着什么。

## 立直的优点

<span class="red b">・可以很轻松地补上役。</span>

麻将没有役就不能和，也就是所谓“1 番缚”。  
而立直可以很轻松地跨过这道门槛。毕竟你只要不鸣牌就行。  
像图 1 这种手，就算已经有 3 张宝牌，不立直也还是只能自摸和牌。  
所以当然要立直。

<span class="b">・有一发、里宝牌的可能。</span>

就算只是没有役也没有宝牌的便宜手，虽然得看运气，但只要立直了，就始终有满贯的可能。  
一发自摸再翻出 1 张里宝就是满贯；  
就算没有一发，只要里宝牌正好打在雀头上，也一样能变成满贯。  
现在很多自由桌雀庄里，一发和里宝牌还会附带祝仪，这个收益非常大。

<span class="b">・不让对手自由打牌</span>

立直的好处，不只是提高点数。  
只要有人立直，大家通常都会谨慎起来。  
会转守，也会有人因此错过和牌。  
这样一来，自己自摸的可能性就会上升。  
让对手不能自由推进，这也是立直的一项优点。

最典型的例子，大概就是自摸四暗刻时去立直吧。  
点数上就算别人打出来也已经够了，但还是故意立直，  
目的就是把巡目拉长，尽量提高自摸和牌的可能性。

<span class="red b">・有时反而能通过公开听牌来狙击荣和</span>

一般来说，立直以后和牌张会变得更难出来。  
但有时根据舍牌，反而会更容易荣和。像筋引挂、字牌待就是典型。  
如果是七对子去等场上已经切了 2 张的字牌，也就是所谓“地狱待”，就应该立直。

## 立直的缺点

<span class="blue b">・必须先拿出 1000 点立直棒。</span>

你也许会觉得 1000 点很便宜，  
但比如南 4 局只领先 2 位 700 点的时候，这就已经是极大的点数。  
立直棒一放出去，当场就会掉到 2 位。  
结果甚至会出现：之后就算 3 位或 4 位和牌，自己也只能以 2 位结束。

<span class="blue b">・向其他三家表明自己已经听牌，荣和会更难期待。</span>

如果同桌的人多少会打一点，他们面对立直，不会轻易切太危险的牌。  
所以在“无论如何都想荣和”的时候，默听会更好。

<span class="blue b">・立直后，除了和牌张和能暗杠的牌以外，其他摸进来都必须摸切</span>

有句话叫“立直就是暴牌宣言”。  
立直以后，除了和牌张和能暗杠的牌以外，其他摸进来的牌都绝对只能摸切。
