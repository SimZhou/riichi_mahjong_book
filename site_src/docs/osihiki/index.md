# 押引

这一章处理的是麻将里最难、也最容易被结果论污染的一块：

**到底什么时候该继续押，什么时候该立刻收手。**

押引之所以难，不在于概念复杂，而在于它永远要同时面对三种不确定：

1. 自己这手到底和不和得了
2. 对手那手到底值多少钱
3. 现在这张牌真正的放铳风险有多高

所以这一章不会从“某种牌型一律押、某种牌型一律退”开始，而是先把押引最底层的思考方式搭起来，再逐步列出真正有用的判断材料。

## 本栏内容

1. [押引铁则](./osihiki01.md)
: 先建立押引的根本观念，重点是不要用结果论评价决策、要有明确目标、并为自己建立可复盘的判断基准。

2. [判断依据](./osihiki02.md)
: 把押引时最基础的五类材料摊开：自手和牌率、打点、危险度、对手推定打点，以及弃和成功率。
