# 麻将入门教程

欢迎来到网络麻将战术书《Beginners Luck》中文整理版。

这套内容面向想学会麻将规则、并一步步变强的读者。就算不一口气读完，也可以按自己的节奏慢慢往下看。

你可以自由使用这套教程：

1. 从第 1 章开始按顺序细读
2. 只挑自己薄弱的部分重点看
3. 把感兴趣的条目先快速翻一遍

这里整理的是各种麻将环境里都通用的技术。无论是线上麻将、街机麻将，还是自由桌，都能用来理解牌形、牌效率、手役、防守和局势判断。

## 教程章节

### 第1章：麻将基础

![麻将基础](./images/top1_off.jpg)

运气与技术都包含在麻将里，但它到底是一种什么性质的游戏？想要赢，目标又该放在什么地方？这一章从等待与牌张性质开始，讲最基础的思考方式。

前往：[第1章：麻将基础](./kihon/index.md)

### 第2章：牌理与牌效率

![牌理与牌效率](./images/top2_off.jpg)

牌理与牌效率，是一种尽量把自己有效牌枚数最大化的“拼图式”思维。它是最基础的技术之一，值得优先掌握。

前往：[第2章：牌理与牌效率](./pairi/index.md)

### 第3章：麻将役种

![麻将役种](./images/top3_off.jpg)

麻将的役很多，但可惜的是，难度和分数并不总是成正比。不要被役种的华丽感和吸引力带偏，先看清真正重要的取舍。

前往：[第3章：麻将役种](./teyaku/index.md)

### 第4章：宝牌与红宝牌

![宝牌与红宝牌](./images/top4_off.jpg)

现代自由桌与线上麻将里，红宝牌与宝牌已经是无法回避的要素。它们既容易做高打点，也会显著改变整场游戏的平衡。

前往：[第4章：宝牌与红宝牌](./dora/index.md)

### 第5章：鸣牌

![鸣牌](./images/top5_off.jpg)

最能拉开“会不会打”差距的部分之一就是鸣牌。现代麻将速度越来越快，必须理解碰、吃在实战里的真正意义。

前往：[第5章：鸣牌](./naki/index.md)

### 第6章：立直

![立直](./images/top6_off.jpg)

立直不只是一个宣言动作。它会同时带来收益和代价，因此需要判断这一次立直到底是否真的必要。

前往：[第6章：立直](./reach/index.md)

### 第7章：防守

![防守](./images/top7_off.jpg)

有些人不重视麻将里的防守力，但防守其实直接影响长期成绩。减少不必要的放铳，成绩会有明显改善。

前往：[第7章：防守](./mamori/index.md)

### 第8章：局势判断

![局势判断](./images/top8_off.jpg)

就算手牌一样，最佳打法也常常会因为对手舍牌、分数状况而不同。关键是判断此时此刻，为了让自己赢下整场，最需要做什么。

前往：[第8章：局势判断](./joukyou/index.md)

### 第9章：押引

![押引](./images/top9_off.jpg)

麻将不是每一局都能一路猛攻，但如果完全不进攻，点棒也不会增加。该押还是该退，这种判断贯穿整场对局。

前往：[第9章：押引](./osihiki/index.md)
