# 麻将入门教程

欢迎来到网络麻将战术书《Beginners Luck》中文整理版。

这套内容面向想学会麻将规则、并一步步变强的读者，细致而清楚地整理了各种麻将战术。就算不一口气读完，也可以按自己的节奏慢慢往下看。

你可以自由使用这套教程：

1. 从第 1 章开始按顺序细读
2. 只挑自己薄弱的部分重点看
3. 把感兴趣的条目先快速翻一遍

读得越多，麻将水平就越能稳步提高。

这里整理的是各种麻将环境里都通用的技术。无论是 Hangame、雀龙门、天凤、东风庄、桃色大战ぱいろん、Maru-Jan 这样的线上麻将，还是街机上的《麻将格斗俱乐部》与 SEGA MJ，都可以用来理解牌形、牌效率、手役、防守和局势判断。
