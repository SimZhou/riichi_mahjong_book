# Tile Theory at One-Shanten (5)

Continuing from the previous page, this one also examines headless shapes.

By now it should be clear that turning the hand into **taatsu + taatsu** is usually the widest choice.  
But what about a shape like Example 1, where only one taatsu remains on the outside?

Even so, drawing <img src="../../hai/man3.gif" width="24" height="34" /> produces Pinfu only if the manzu side is kept.  
And the wait on <img src="../../hai/man1.gif" width="24" height="34" /><img src="../../hai/man3.gif" width="24" height="34" /> is also better than <img src="../../hai/pin1.gif" width="24" height="34" />-<img src="../../hai/pin4.gif" width="24" height="34" /> or <img src="../../hai/sou5.gif" width="24" height="34" />-<img src="../../hai/sou8.gif" width="24" height="34" />.

Then which is better, discarding <img src="../../hai/pin1.gif" width="24" height="34" /> or discarding <img src="../../hai/sou5.gif" width="24" height="34" />?  
When the acceptance count is the same, the basic rule is to compare **good-shape improvements**.

If you keep the inner <img src="../../hai/sou5.gif" width="24" height="34" />, then improvements like Change A and Change B become available.  
In other words, for a pseudo-ryanmen shape, keeping the inside tile is better.

In headless shapes, good-shape improvements are a very important factor.

Many players would casually discard <img src="../../hai/man7.gif" width="24" height="34" /> from a hand like Example 2.  
But a one-tile difference in the current acceptance is not a big deal at all.  
You need to think in terms of continuing to build **taatsu + taatsu** out of the connected manzu shape.

If you discard <img src="../../hai/man7.gif" width="24" height="34" /> in Example 2, the current acceptance is only 5 types, 17 tiles.  
It should be clear how painful it is to miss improvements like Change A and Change B below.

### Theory Summary

<span class="green">The basic comparison is still acceptance count, but when the difference is tiny, you also need to include the quality of the tenpai wait and the value of good-shape improvements.</span>
