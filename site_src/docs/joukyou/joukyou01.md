# 局势判断基础

## 什么是局势判断

初学者往往只会看自己的手牌。  
但随着巡目推进，可以从牌桌上得到各种信息。  
把这些信息也考虑进去，再去决定切牌、鸣牌、立直与否，这就叫做局势判断。

如果一直只盯着自己的手牌，就很容易漏掉重要信息。  
说得尖刻一点，那就只是“配对麻将”而已。

麻将当然需要很多种技术，但其中尤其重要的一项，我认为就是局势判断。

连做牌这件事，也会随着局势而变化。

例如，边张和坎张相比，通常麻将里都会嫌弃边张。  
但即便这一点，也会因为局势不同而反过来。

比如：

场上 ![3p](../hai/pin3.gif) **已经见了三张**

这种情况下，就会选择把边张留下。

更进一步说，从这手里去切 ![3m](../hai/man3.gif) 或 ![4s](../hai/sou4.gif)，也都完全可能。

如果已经到了海底，那更不用说，当然是切最安全的那张牌。  
如果 ![3m](../hai/man3.gif) 是确定能过的牌，那就直接切 ![3m](../hai/man3.gif)。

麻将正是因为有局势判断，才真正成立。

在局势判断里，最重要的一块其实是：面对对手手牌时，到底该攻还是该守，  
也就是所谓的押引。  
不过这一点放到最后一章再说。  
这一章先来考虑：**结合局势做牌。**

## 两种局势

所谓局势，其实并不难。

只有两种：

<span class="red">场况</span> 和 <span class="red">点棒状况</span>

所谓场况，就是牌桌上的局面。  
宝牌是什么、四家的舍牌、谁在副露、自己是亲家还是子家、到了第几巡……  
这些全部都包含在内。

另一个同样重要的，就是点棒状况。  
因为麻将的胜负，是以整场结束时的点数来决定的，  
所以四家的点棒必须一直放在脑子里。

## 局势与战略

真正重要的，是根据这两种局势先立出战略再去打牌。  
你平时打牌时，会这样去想吗？

例如，如果场上已经供托了 `4` 根立直棒，  
那就会形成“尽快和牌把这些立直棒拿到手”的战略。

如果头名是亲家，那么去做高打点的自摸手就是一种有效战略。  
因为一旦成功，还能靠亲被自摸进一步缩小分差。

也就是说，比起速度、效率这些局部问题，  
<span class="red">首先该有的是战略</span>。

即使是这种配牌，打法也完全可能有很多种。  

- 如果是东场亲家，可以用九种九牌流局
- 如果是断崖垫底，就同时看国士和混一色，先从 ![4s](../hai/sou4.gif) 这类牌开始处理
- 如果是南四头名，就把国士骨架保留下来，做成随时可以退守的形

也就是说，这样的战略都是可能成立的。
