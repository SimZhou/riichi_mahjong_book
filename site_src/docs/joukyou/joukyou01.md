# 局势判断基础

这一章原页开头先强调了一件事：

**麻将不是只看自己手牌的游戏。**

初学时大家都很容易把注意力全部放在自己手里的形状上，但随着巡目推进，牌桌上会不断出现新的信息。是否有人鸣牌、宝牌是什么、谁是亲家、现在是第几巡、四家的点棒差距怎样，这些都会改变同一副手牌的答案。

这就是原页所说的“局势判断”。

## 第一部分：只看自己手牌，很容易把麻将打成机械配对

原页说得很直接：如果始终只盯着自己手里的搭子和向听数，很多重要信息就会被自动忽略。那样打出来的麻将，往往只是在做静态配牌题，而不是在打真正的对局。

局势判断之所以重要，是因为它会直接改写手作的标准。原页举了一个很有代表性的例子：通常我们都会嫌弃边张，优先保留坎张或两面；但如果某一门的关键牌已经在场上见了三张，那么原本理论上更好的那一块，实战价值就可能已经塌掉。

所以同一手牌，在下面这些局面里，答案可能完全不同：

1. 某张关键牌已经三枚见牌
2. 当前已到海底前后
3. 某家已经鸣牌逼近听牌
4. 你是亲家还是子家

原页的意思很明确：**牌理不是废掉，而是必须放到场况里面重新计算。**

## 第二部分：局势判断主要看两类信息

原页把复杂的局面信息先收束成两大类，这个整理很实用。

### `场况`

所谓场况，就是牌桌当下正在发生的事情，包括：

1. 宝牌是什么
2. 四家的舍牌内容
3. 谁已经鸣牌
4. 亲家是谁
5. 当前巡目
6. 场上某门哪些牌已经薄掉

这些信息会改变某副手牌的速度、打点、危险度和可实现性。

### `点棒状况`

另一类是点棒状况，也就是：

1. 四家当前分差
2. 自己现在是第几名
3. 这一局最该争的目标名次是什么
4. 终局时哪些结果对自己有利、哪些不能接受

原页强调，麻将最终比的不是局中某一手打得多漂亮，而是整场结束后的结果。所以你不能只算这手牌能不能多打两千点，还得算：

1. 这两千点到底能不能改变名次
2. 失误放铳会不会直接掉位
3. 此时应该抢速度，还是应该守住现有位置

## 第三部分：真正先决定打法的，是战略，不是局部效率

原页后半段最重要的一句，其实就是：

**先有战略，再谈速度和效率。**

比如场上已经积了很多立直棒，那么这一局的战略就可能优先变成：

1. 尽快和牌
2. 先把供托拿到手

如果头名是亲家，那么另一个可能的战略就会变成：

1. 追求更高打点
2. 尽量靠自摸一次性把分差缩大
3. 利用亲被自摸的连带损失去拉近差距

也就是说，战略目标一旦不同，手牌处理的顺序也会跟着变。原页想强调的是：不要先盯着“哪一打最像标准牌理”，而要先问“这局我到底想靠什么结果获利”。

## 第四部分：再差的配牌，也会因为局势不同而有不同打法

原页最后给了一副看起来非常差的起手，意思不是教你死记其中某一打，而是说明：

**局势不同，连烂牌的战略都不一样。**

比如同一副烂起手：

1. 东场亲家时，可以优先考虑九种九牌流局
2. 若自己是断崖垫底，就可以同时保留国士、混一色这类逆转路线
3. 若到了南四你是头名，则甚至可以保留国士骨架，把这手变成“随时可退可守”的安全结构

这就是局势判断和普通手作最大的区别。普通手作只会问“这手离听牌还有多远”，而局势判断会先问：

1. 这局我最需要什么结果
2. 什么结果绝对不能发生
3. 在这个目标下，这副牌该朝哪条路线走

## 这一页的结论

1. 局势判断，就是把场上信息和点棒状况一起纳入打牌决策
2. 只看自己手牌，会把麻将打成静态配牌题，忽略真正重要的桌面信息
3. 局势判断主要看两类内容：场况与点棒状况
4. 同一副手牌在不同局面下，牌理答案可能完全不同
5. 实战中应先决定战略目标，再回头选择最适合该战略的手作路线
