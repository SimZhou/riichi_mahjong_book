# 关注名次

这一页原页要讲清楚的核心很直接：

**麻将本质上是顺位游戏。**

如果只盯着局中的素点，很容易做出“看起来赚了，实际上输掉整场”的选择。真正该一直放在脑子里的，是这手牌会不会改变最终名次。

## 第一部分：为什么点棒状况这么重要

原页先解释，点棒状况之所以重要，不是因为数字本身好看，而是因为麻将结算时还有额外的顺位点。也就是原页说的：

1. 顺位奖惩分，也就是不同名次之间额外结算的奖惩
2. 头名奖励，也就是头名独享的起始点差奖金

这意味着，牌桌上的 `1000`、`3900`、`8000`，并不能直接等同于整场的输赢价值。很多时候，真正拉开成绩差距的不是你这局多和了多少，而是最后你排第几。

## 第二部分：乌马会把“名次变化”放大

原页列了几种常见乌马：

1. `5-10`
2. `10-20`
3. `10-30`

不管采用哪一种，意思都一样：排名高的人，会在终局时再额外拿一笔；排名低的人，则要再额外付一笔。

所以从实战角度看：

1. 第二和第三之间，看起来可能只差一点素点
2. 但只要名次交换，整场收支就会被再放大一次
3. 第一和第二之间更是如此

这也是为什么很多南场判断，不能只问“这手和了值不值”，而是要问“这手和了以后，名次会不会变”。

## 第三部分：飞鸟位会让头名价值被进一步抬高

原页接着解释头名奖励。常见规则是：

1. 原点 `30000`
2. 起始点 `25000`
3. 中间差的 `5000 x 4 = 20000`
4. 最后全部给头名

因此头名拿到的，不只是“比别人多一点”，而是额外再吃下一整笔奖金。

这一点原页配了一张图：

<img src="../images/40.gif" width="451" height="276" />

这张图要表达的意思非常重要：

1. 游戏中的素点，只是结算前的一部分
2. 真正决定收支的，是加上乌马与飞鸟位后的最终名次点
3. 因此越接近终局，越必须把“自己该争第几”摆到牌理前面

也正因为如此，原页才会说：头名的顺位价值特别大，很多规则下几乎可以理解成“除了第一，其余名次的价值都会被压低一截”。这话虽然有点夸张，但在实战判断上非常有用。

## 第四部分：南场开始后，必须一边打牌一边算名次

原页后半段给出的要求非常明确：一旦进南场、四家的点差开始拉开，就不能再只顾自己手牌成不成形，而要持续问自己：

1. 我现在该争的是第一、第二，还是先保第三
2. 这手若改走快攻，能不能真正换到名次提升
3. 这手若放铳，会不会直接掉位
4. 现在有没有必要为了逆转，主动提高波动

这其实就是顺位麻将和普通配牌题最大的差别。很多局面下：

1. 追一手高打点，是为了逆转名次
2. 改成全鸣快和，是为了先锁住当前顺位
3. 放弃薄和，是因为那点素点根本不改变结算结果

如果不把名次放进判断里，这些选择看起来都像“不合牌理”；但一旦放进整场目标，它们反而才是正确打法。

## 第五部分：不同名次的人，本来就该有不同打法

原页最后一句话可以直接当这一页的总纲：

**头名有头名的打法，垫底也有垫底的打法。**

意思不是让你每一手都过度算分，而是提醒你：

1. 头名更该重视保位、控制风险
2. 中位常要先看自己离上家、下家各差多少
3. 末位若只做普通和牌，可能根本没有意义，必须提高逆转路线比重

所以“关注名次”不是终盘特技，而是从中盘开始就该持续存在的背景计算。

## 这一页的结论

1. 麻将是顺位游戏，不是单纯的素点游戏
2. 乌马会放大名次交换带来的收益与损失
3. 飞鸟位会进一步抬高头名价值
4. `40.gif` 表达的重点是：真正决定收支的是素点加上顺位点后的最终结果
5. 到南场以后，必须持续计算自己该争哪一名，并据此决定速度、打点与风险取舍
