# 关注名次

为什么点棒状况重要？  
因为麻将里还有 **乌马** 和 **冈** 这两种顺位点。

**`乌马`**

所谓 **乌马**，就是半庄结束时，根据名次高低所支付的点数。  
大多数规则，基本采用下面三种之一：

- `5-10`（第四给第一 `10000` 点，第三给第二 `5000` 点）
- `10-20`（同理，变成 `20000` 点和 `10000` 点）
- `10-30`（同理，变成 `30000` 点和 `10000` 点）

**`冈`**

所谓 **冈**，就是麻将中每一场给头名的奖金点。  
麻将的原点是 `30000` 点，但一般并不是从 `30000` 点起打，而是从较少的 `25000` 点开始。  
这中间相差的 `5000` 点，全部都归头名所有。  
也就是说，只要拿到头名，就会得到 `5000 × 4 = 20000` 点。

也就是说，就会变成下面这样：

就像图里那样。  
在现在常见的规则下，麻将的最终收支，与其说更看对局中的素点，  
不如说更看 **对局结束后，按照名次加减的点数**。

所以，对局中必须一直意识到四家的点棒。  
一旦进入南场，四家的持点开始拉开差距，  
就请务必一边打牌一边思考：

**“自己这局应该争第几名？”**

尤其头名的顺位点非常大，  
几乎可以说“除了第一，其他名次都不值钱”。  
所以请尽可能朝着头名去打。

网络麻将的 Rating 等系统，本质上采用的也是顺位战。  
也就是说，评价标准只看最终名次。

别忘了麻将是名次游戏。  
请始终根据局势去选择与之相应的打法。

头名有头名的打法，末位也有末位的打法。
