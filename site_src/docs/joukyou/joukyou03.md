# 状况与做牌（1）

这一页来谈谈：**根据巡目来做牌。**

巡目可以以舍牌所在的行数为参考，大致分成下面 3 种状况：

- 序盘（`1` 到 `6` 巡）
- 中盘（`7` 到 `12` 巡）
- 终盘（`13` 到 `18` 巡）

分成这三段来思考会比较清楚。

**`基本思路`**

- 序盘要追手役可能，重视好形
- 中盘以后，第一目标是把牌做进听牌；有些手还要开始准备应对他家进攻
- 终盘如果已经不是能战斗的牌形，就该提高安全性权重

下面用具体例子来说明。

做出即立直路线，会是最没破绽的打法。

但如果还是序盘，也可以追求打点，  
把 ![9s](../hai/sou9m.gif) 的对子拆掉，这样打也完全可以。

**`例2`**  
![5m](../hai/man5.gif)![6m](../hai/man6.gif)![7m](../hai/man7.gif)![6p](../hai/pin6.gif)![6p](../hai/pin6.gif)![7p](../hai/pin7.gif)![8p](../hai/pin8.gif)![2s](../hai/sou2.gif)![2s](../hai/sou2.gif)![3s](../hai/sou3.gif)![4s](../hai/sou4.gif)![5s](../hai/sou5.gif)![6s](../hai/sou6.gif)![8s](../hai/sou8.gif) 宝牌 ![8m](../hai/man8.gif)

例 2 的本手应该是**不取听牌**，切 ![9s](../hai/sou9m.gif)。

因为筒子从 ![5p](../hai/pin5m.gif) 到 ![9p](../hai/pin9m.gif) 摸到什么都会变成好形听牌，  
索子的手变也有很多。

但到了中盘以后，筒子可能已经变薄，  
这时手变就未必还值得期待。

那种情况下，就该改成切 ![6p](../hai/pin6m.gif) 先取听牌。

之后再期待索子变化，或摸到宝牌后做出三色。  
若索子的舍牌状况不妙，有时甚至该直接立直。

**`例3`**  
![5m](../hai/man5.gif)![7m](../hai/man7.gif)![1p](../hai/pin1.gif)![1p](../hai/pin1.gif)![2p](../hai/pin2.gif)![8p](../hai/pin8.gif)![8p](../hai/pin8.gif)![2s](../hai/sou2.gif)![3s](../hai/sou3.gif)![6s](../hai/sou6.gif)![7s](../hai/sou7.gif)![8s](../hai/sou8.gif)![北](../hai/pei.gif)![北](../hai/pei.gif) 宝牌 ![2m](../hai/man2.gif)

如果只看牌效率，例 3 的最佳手当然是拆 ![北](../hai/peim.gif) 对子。

损失的只是 ![北](../hai/peim.gif) 本身，这点应该没有什么异议。

但如果已经是中盘以后，这副牌形就该把守备放在牌效率之前。  
没有宝牌的两向听，还去盲目扩宽，这件事本身就很危险。

这时应当把安全牌 ![北](../hai/peim.gif) 留下，转而处理筒子部分，这样更稳妥。

顺带一提，终盘一个重要主题就是**形式听牌**。  
当舍牌已经进入第三行时，与其追求和牌，不如优先保住形式听牌会更好。

**`例4`**  
![2m](../hai/man2.gif)![2m](../hai/man2.gif)![7m](../hai/man7.gif)![8m](../hai/man8.gif)![9m](../hai/man9.gif)![1p](../hai/pin1.gif)![3p](../hai/pin3.gif)![4p](../hai/pin4.gif)![5p](../hai/pin5.gif)![5p](../hai/pin5.gif)![6p](../hai/pin6.gif)![2s](../hai/sou2.gif)![2s](../hai/sou2.gif)![3s](../hai/sou3.gif)

如果按平和路线来做，理论上应切 ![2s](../hai/sou2m.gif)。

但若目标只是形式听牌，那就该切 ![1p](../hai/pin1m.gif)。

吃牌只能从上家来，  
可 ![2m](../hai/man2m.gif) 和 ![2s](../hai/sou2m.gif) 的碰却能从三家得到。

所以，如果目的是尽快听牌，  
答案就是切 ![1p](../hai/pin1m.gif)。

**`例5`**

<img src="../images/h007.png" width="500" height="500" />

来看一个终盘里稍微进阶一点的技巧。  
例 5 是在他家已经立直的情况下，终盘取到形式听牌的场面。

上家切出了 ![4s](../hai/sou4m.gif)，而 ![中](../hai/tyunm.gif) 在场上已经见了两张。  
当然不能荣和，但你会不会选择见逃，把手伸向下一张自摸？

这里应该吃掉这张牌，再切 ![中](../hai/tyunm.gif)。

这样一来，就算下一巡摸到危险牌，  
也还能再切一张 ![中](../hai/tyunm.gif)，把听牌继续维持下去。
