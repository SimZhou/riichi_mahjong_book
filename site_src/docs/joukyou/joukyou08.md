# 最终局战术（2）

这一页原页专门讨论最终局的立直判断。平场里“能立多立”基本成立，但到了南四，

**立直棒本身就会改变名次。**

所以这时不能只问“立直期望值高不高”，还要问：

1. 这一根立直棒丢出去后，自己会不会直接掉位
2. 默听是否已经足够守住或逆转
3. 立直多出来的收益，到底值不值得冒这层顺位风险

## 第一部分：`h036.gif` 说明，头名最终局的有役听牌默认先默听

第一张图是最终局头名最典型的场面。

<img src="../images/h036.gif" width="500" height="500" />

平场里，这种牌很多人会自然往满贯立直上靠；但原页在这里给出的答案非常硬：

1. 你已经是头名
2. 这手本来就有役
3. 现在最重要的是别因为多丢一根立直棒，反而把原本稳住的顺位打乱

所以这里不只要默听，而且还要把等待改成更容易出和的一边。原页想表达的是：

**最终局头名的有役听牌，默认先想“怎样最稳结束”，而不是“怎样再多拿一点”。**

它还给了一个补充边界：只有在领先非常多，哪怕被满贯直击也不会被翻掉时，才可以考虑“锦上添花”的役有立直。但这种局面本来就不多。

## 第二部分：`h037.gif` 说明，若不立直就拿不到关键荣和，那头名也得攻

第二张图则是头名必须主动出手的例外。

<img src="../images/h037.gif" width="500" height="500" />

当前虽然你是第一，但第二名已经非常接近，而且：

1. 就算你现在默听
2. 对方也已经满足逆转条件或极接近逆转条件
3. 你的某张更容易出的牌，只有立直后才有荣和资格
4. 牌本身也几乎没有值得继续等的手变

这时再因为“我是头名”就死守默听，反而是把唯一主动得分机会让掉。原页因此明确认为这里应立直。

也就是说：头名不是永远缩。若不立直就拿不到关键和牌，而对手本来也快反超，那就应把立直视为主动封盘手段。

## 第三部分：`h038.gif` 说明，对垫底家立直时，头名更该优先保住位置

第三张图是另一种很常见、也最容易乱冲的场面。

<img src="../images/h038.gif" width="500" height="500" />

末位家已经立直，你自己此时也做成听牌。平场里你也许会觉得“不如对攻”；但原页这里明确主张默听。

原因很简单：

1. 末位无论摸到什么，都不影响你头名
2. 你当前大体已经稳住第一
3. 若立直，就要先交出一根立直棒
4. 一旦被击回、甚至发生别家自摸，顺位反而可能变坏

所以这时最正确的做法不是全面对攻，而是：

1. 先默听
2. 摸到真正高价值进张再考虑
3. 若接下来摸进危险牌，就迅速弃和

原页对这种场面下乱立直的评价很重，因为这本质上是在用不必要的波动，去破坏本来已经快到手的头名。

## 第四部分：最终局里，所谓“等手变”多数不如立直附加收益现实

原页最后补了一条非常重要的经验：最终局当然也有“等手变”的牌，但实际成立的情况比很多人想象得少。因为只要：

1. 一发能逆转
2. 里宝牌能逆转
3. 自摸加一番就够

那立直的附加收益，往往比去等某一种低概率手变更现实。

所以原页给的最终判断其实很清楚：

### 有役且守位优先时

先默听，别白白交出立直棒。

### 无役或必须靠立直增益逆转时

立直多数优于等待手变。

### 只有当某一种手变是明确且高收益的唯一路线时

才值得在最终局继续埋着等。

## 这一页的结论

1. 最终局的立直判断，必须把立直棒对顺位的直接影响算进去
2. `h036.gif` 说明：头名最终局的有役听牌，默认先默听收官
3. `h037.gif` 说明：若不立直就拿不到关键荣和，即使是头名也该主动立直
4. `h038.gif` 说明：对末位家的立直，头名很多时候更应保住现状，而不是无谓对攻
5. 最终局里真正值得等待手变的场面不多，很多时候立直带来的一发、里宝牌、自摸增益更现实
