# 局势判断

这一章讲的不是单纯“牌效率是不是最优”，而是：

**同一手牌，在不同巡目、不同分差、不同对手状态下，最优打法会变。**

很多人学了牌理之后，容易把麻将打成固定公式题。但实战并不是只看自己手里能不能多一张受入，而是要不断回答下面这些问题：

1. 现在更该追速度，还是追打点
2. 这一局我真正该争的是第一、第二，还是只要别掉到第四
3. 亲家、供托、鸣牌家、终盘巡目，这些场况会不会改变原本的牌理答案

所以这一章的目标，是把“场况”和“点棒状况”都纳入决策，让打牌不只停留在手牌内部。

## 本栏内容

1. [局势判断基础](./joukyou01.md)
: 先建立局势判断的两大来源：场况与点棒状况，并说明为什么战略优先于局部牌效率。

2. [关注名次](./joukyou02.md)
: 讲清麻将是顺位游戏，乌马与飞鸟位会让“名次变化”比局中素点本身更重要。

3. [状况与做牌（1）](./joukyou03.md)
: 先按序盘、中盘、终盘拆解手作重心，说明巡目为何会改变打点、速度与形式听牌的优先级。

4. [状况与做牌（2）](./joukyou04.md)
: 把场上见牌数、颜色强弱和他家副露路线折回自己的手牌判断。

5. [状况与做牌（3）](./joukyou05.md)
: 进入点棒状况，比较落后时追打点与领先时追速度的不同手作策略。

6. [局势与立直判断](./joukyou06.md)
: 处理山越字牌宝牌、现物待与南场顺位条件，说明何时该压住立直、何时反而必须立直。

7. [最终局战术（1）](./joukyou07.md)
: 先算最终局条件，再根据逆转或守位目标重排手作与鸣牌优先级。

8. [最终局战术（2）](./joukyou08.md)
: 专讲最终局立直判断，重点是立直棒对顺位的直接影响。

9. [最终局战术（3）](./joukyou09.md)
: 通过拆听、见逃、诺听宣言与差入，补充最终局最特殊的顺位导向打法。
