# 第8章：局势判断

![局势判断栏目主图](../images/top-image11.jpg)

1. [局势判断基础](./joukyou01.md)
   麻将不是一个人独自进行的游戏，打牌时必须始终留意局势。

2. [关注名次](./joukyou02.md)
   无论是看收支还是打顺位战，都必须用“关注名次”的打法来应对。

3. [状况与做牌（1）](./joukyou03.md)
   所谓“按巡目做牌”，即使是同一手牌，到了不同巡目，应该切的牌也会改变。

4. [状况与做牌（2）](./joukyou04.md)
   这是上一页的延续，这次会进一步谈到“根据局势来做牌”。

5. [状况与做牌（3）](./joukyou05.md)
   要理解：有些场合该追打点，有些场合则该优先追速度。

6. [局势与立直判断](./joukyou06.md)
   在考虑点棒状况和他家舍牌之后，再来判断这一手该不该立直。

7. [最终局战术（1）](./joukyou07.md)
   讲解在最考验局势判断能力的最终局里，需要特别注意什么。

8. [最终局战术（2）](./joukyou08.md)
   解释即使是最终局暂居第一，也仍然有必须进攻的场合。

9. [最终局战术（3）](./joukyou09.md)
   介绍差入、诺听宣言、见逃等特殊选择在什么情况下会变得有效。
