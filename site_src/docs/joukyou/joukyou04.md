# 状况与做牌（2）

接着来看：**按照场况来做牌。**  
观察场上已经打出的牌，以及他家已经鸣掉的牌，  
有时就必须根据这些信息来调整手作。

## 对变薄的牌做出反应

看舍牌时，先去确认自己需要的牌在场上已经打掉了多少。  
这是局势判断里最基本的基本。

**`例1`**

<img src="../images/h004.gif" width="500" height="500" />

这一题是在比较：索子的两边两面，到底该嫌哪一边。

答案就在场上。  
如果 ![1s](../hai/sou1m.gif) 已经切了 2 张，那当然就该打 ![3s](../hai/sou3m.gif)。

## 留意哪一门颜色更强

**`例2`**

<img src="../images/h005.gif" width="500" height="500" />

亲家已经副露了客风 ![西](../hai/sham.gif)，这说明他在做索子混一色。

而自己这边是宝牌面子已经完成的一向听。  
由于面子超载，现在要在索子坎张和筒子坎张之间选择拆哪一边。

两边在场上都各只见了一张，  
但真正要看的是整个场上的筒子和索子分别被切了多少。

筒子被切了这么多，说明他家并不需要筒子。  
反过来，索子很少出现在场上，就很容易推测它们大多还被用在手里。

像这样，某一门牌在场上大量被切出的状态，叫做 **“场上便宜”**；  
而几乎看不到的状态，就叫 **“场上高”**。

一般来说，等在“场上便宜”的颜色上，会更容易和出。

就算坎 ![3p](../hai/pin3m.gif) 最多只剩 3 张，也仍然可以算是好待。  
因为这种“场上便宜”的颜色，不仅他家不太会留在手里，  
也更容易变成一机会、无机会，实际和牌率往往会高于账面枚数。

所以这里应该拆掉：

![1s](../hai/sou1m.gif)![3s](../hai/sou3m.gif)

如果因为亲家染索子，就过度警惕而去动筒子，  
那反而会显得太软弱。

要是以坎 ![2s](../hai/sou2m.gif) 待去立直，由于和亲家的混一色撞色，  
基本不可能从两边荣和，反而更危险，也更容易吃到亲家的反击。
