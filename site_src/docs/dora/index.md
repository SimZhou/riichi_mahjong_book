# 宝牌与红宝牌

这一栏处理的是麻将里最直接的打点来源：宝牌、红宝牌，以及围绕它们产生的取舍。

但这章的重点不是“宝牌很值钱”这种废话，而是更具体的几件事：

1. 打牌时要不要保留宝牌受入
2. 手牌里已经有宝牌时，怎样把它们真正用到和牌里
3. 什么时候该为了宝牌改变速度判断，什么时候又不能被宝牌绑架
4. 红宝牌和祝仪规则会怎样改写常规牌理

换句话说，这一章讲的是：**宝牌如何改变你对速度、打点和牌形的权衡。**

## 本栏内容

1. [宝牌应对（1）](./dora01.md)
: 先建立最基础的“宝牌受入”意识，避免在拆牌时把未来摸进宝牌的路线自己打掉。

2. [宝牌应对（2）](./dora02.md)
: 继续看手里已经有宝牌时的切牌，重点是怎样把现有宝牌尽量用到底，而不是听牌后白白打出去。

3. [宝牌 2 张以上](./dora03.md)
: 讲清手里已有多张宝牌时，为什么应更重视速度、和牌率与副露变现。

4. [切宝牌时机](./dora04.md)
: 处理最容易失误的“宝牌该什么时候放”，区分役牌宝牌与普通数牌宝牌。

5. [宝牌与手役选择](./dora05.md)
: 总结当宝牌与手役冲突时，怎样按受入、和牌率和实际打点差来选边。

6. [红宝牌麻将基础](./dora06.md)
: 说明红宝牌规则为何会整体抬高速度、手广和副露的优先级。

7. [红宝牌与祝仪](./dora07.md)
: 进入祝仪麻将，重新估值红牌、立直张数、自摸率和红牌副露的收益。
