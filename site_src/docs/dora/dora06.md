# 红宝牌麻将基础

红宝牌规则会直接改变做牌逻辑。

因为一旦牌桌上有红宝牌，很多原本只是“普通可和”的手，都会突然变成：

1. 不必特别做役也有不错打点
2. 即使副露也仍然有收入
3. 中张价值进一步上升
4. 某些传统役种的优先级反而下降

所以红宝牌麻将的核心，不是“多了一张值钱牌”，而是：

**整套速度、手广、鸣牌和危险度判断，都会被它重新加权。**

## 第一部分：有红宝牌时，速度和手广比役种更重要

### 例 1：有红的手，不必为了三色之类硬做窄

第一例里，普通麻将看起来很像三色手，但在红宝牌规则下，继续抱着三色幻想，反而不如直接拆掉更差的坎张，让整副牌更快、更宽。

原页要强调的是：

1. 红宝牌已经天然提供了打点
2. 因此你没必要再为了一点额外役种，主动把手牌做窄
3. 这和无红规则下“有时值得多看一眼手役”不同

所以红宝牌麻将的第一条实战规则就是：

**红牌在手时，默认更偏向速度与手广。**

## 第二部分：别死守门清，红牌手要更敢副露

### 例 2：有红宝牌时，副露收益会显著抬高

第二例里，手里同时有红五筒和红五索。若继续坚持门清，当然也能慢慢打；但如果直接按断幺九路线去鸣，`3900` 左右的副露手会比门清慢推更现实。

这题最重要的不是具体能吃哪些牌，而是思维切换：

1. 红牌已经替你垫了打点
2. 因此鸣牌后的“便宜手”其实并不便宜
3. 一些平时不愿意做的平移副露、同向听副露，在红牌桌上都更值得考虑

这就是为什么原页会强调：**红宝牌麻将里，鸣牌技术比无红环境更重要。**

## 第三部分：带红的手，通常不适合做纯幺九型役种

### 例 3：混老头、纯全带幺九这类路线，和红宝牌天然冲突

红宝牌都在 `5`，这意味着：

1. 凡是强迫你远离 `4-6` 中张区的役种
2. 在红牌桌上都天然失去一部分吸引力

原页特别点名全带幺九、纯全带幺九、混老头这类手。因为你一旦往这些方向走，不光放弃了红牌本身，还会让自己在终盘必须去处理原本最危险的 `4-6` 牌。

所以即使手牌确实有幺九役种的影子，也要先问一句：

**这个方向是不是在和红牌桌的高价值中张区对着干。**

## 第四部分：拆牌时也要保留“红受入”

### 例 4：红五能不能接进来，要单独计算

最后一段最实用。原页用一个典型一向听比较说明：在无红麻将里，通常会保留更安全、也更传统的那组牌；但在红牌桌上，若某种切法能让你未来摸到红五时直接完成替换，那这条路线就应上浮优先级。

也就是说，红宝牌要像普通宝牌一样单独算“受入”，但它比普通宝牌更常见、也更值钱，因此权重还要更高一些。

这会带来一个非常稳定的习惯：

1. 看切牌时，不只问“哪张现在最没用”
2. 还要问“摸到红五时，我有没有自然的替换空间”

## 这一页的结论

1. 红宝牌麻将会整体抬高速度、手广和副露的优先级
2. 有红牌时，不必为了中小手役把牌做窄
3. 红牌已经垫好打点后，应更敢副露加速
4. 幺九系手役与红牌天然冲突，通常优先级下降
5. 拆牌时要额外保留“红五受入”和未来替换空间
