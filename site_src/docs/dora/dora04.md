# 切宝牌时机

宝牌最让人头疼的，往往不是它值不值钱，而是：**什么时候该切。**

切得太早不行，切得太晚也不行。

## 役牌是宝牌的情况

最麻烦的，往往就是役牌宝牌。

如果问这种牌的最佳切出时机是什么，原页给出的答案非常明确：

**第一打，或者听牌时。**

为什么第一打反而好？因为切出去时，最不容易被碰、也最不容易被荣和的时点，本来就是第一打。

比如在南四局自己处于头名，拿到这样一手配牌，只要和出就够了，那么第一打就该把 `白` 宝牌切掉。

因为这手本来就很像会自然收束成平和之类的牌，既然局况只要求先和，那么就没有必要抱着役牌宝牌不放。

当然，这种“只求和牌”的场面其实并不多。大多数时候，宝牌一旦重成雀头，打点会大幅上升，所以通常还是会先留着。

像这种一向听就把宝牌打出去，是完全不成立的。

因为一旦宝牌重上，直接就能拿来做雀头，甚至是满贯手。所谓“宝牌抱太久会危险”并不能成为理由，这个想法本身就是错的：你若真把宝牌碰给别人，全场的危险度只会更高。

所以这手正确的切牌时机，是听牌时。最理想的情况，就是把宝牌拿来当立直宣言牌。

### 理论小结

<span class="green">有明确不要牌时，先切不要牌，不要先对宝牌下手。</span>

不过，也不是说从一向听阶段绝对不能切役牌宝牌。

比如下面两种变化：

- 摸到 `9p`
- 或摸到 `4m / 5m / 7s / 8s`，让牌形加上双碰受入

这时若一向听已经宽到：

<img src="../hai/man4m.gif" width="19" height="26"><img src="../hai/man5m.gif" width="19" height="26"><img src="../hai/sou7m.gif" width="19" height="26"><img src="../hai/sou8m.gif" width="19" height="26"> 这样整体很顺，役牌宝牌提前切出也可以讨论。

原页给出的适用条件大致是：

1. 亲家
2. 序盘还没人副露
3. 场上已有同种宝牌，或整体危险度较低

在这种情况下，一向听提前切宝牌可以成立。

但前提仍然没变：**不能在明明还有废牌可切时，先去乱切宝牌。**

## 其它情况

作者自己就明确说过，他并不是那种会特别死保宝牌的打手。若为了宝牌而明显缩窄受入，他会直接放掉。

这一题虽然不是完全不能把两张宝牌都用进去，但如果以和牌优先来打，最宽的切法仍然是切宝牌。

如果摸进 `4p / 6p / 5s` 这类改良牌，就能直接变成平和一向听；而且从 4 家持点看，当下也不是那种非要做超高打点不可的局面，因此这里就该优先手广，直接切宝牌。

大多数一向听场面里，优先手广而不是死抱宝牌，才是更常见的正解。

当然，若这张宝牌切出去本身就极容易放铳，那又另当别论。

## 这一页的结论

1. 役牌宝牌最特殊，通常最好的切牌时机是第一打或听牌时
2. 有明确不要牌时，应先切不要牌，而不是因为焦虑提前动宝牌
3. 某些高质量一向听局面，在条件合适时也可以提前切宝牌
4. 普通数牌宝牌不值得为它明显伤害手广
5. 切宝牌时机的本质，是在风险、速度、打点之间选最划算的窗口
