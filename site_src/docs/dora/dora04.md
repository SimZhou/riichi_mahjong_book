# 切宝牌时机

总是让人头疼的，恐怕就是宝牌该什么时候切。  
切得太早不行，切得太晚也不行。

## 役牌是宝牌的情况

最麻烦的，就是役牌宝牌。  
如果问最佳切牌时机是什么，

我会回答：<span class="blue b">第一打</span>，或者<span class="blue b">听牌时</span>。

为什么第一打好？  
因为切出去时，**最不容易被碰、也最不容易被荣和的，就是第一打**。

比如，如果南 4 局自己在头名，配牌是这样：

这种局面只要和出来就够了，而且这手看起来也像会收束成平和，没必要抱着宝牌不放。

不过，只要和牌就行的情况其实很少。  
大多数时候，宝牌只要重合成雀头，分数就会大幅上升，  
所以通常还是会先留下来。

像这种一向听就把宝牌切出去，根本不成立。  
宝牌一旦重上，不但能拿来做雀头，甚至还能去看满贯。

“因为宝牌拿太久会危险……”这种说法不能当理由。  
这是很明确的错误想法。  
因为一旦宝牌被碰，全场所有牌都会变危险。

所以这手正确的切宝牌时机，是听牌时。  
宝牌拿来当立直宣言牌才是正着。

如果宝牌比较容易切，或者这手无论如何都想和，

就可以在这里放出宝牌。  
具体来说，比如下面这些情况：

- 亲家
- 序盘还没人副露
- 场上已经有宝牌见出

像这种情况，如果条件合适，一向听切宝牌也可以。  
不过，明明还有废牌，却先去切宝牌，这种随便的打法还是不行。

## 其他情况

就我自己来说，也不是特别珍惜宝牌的打手。  
如果要靠缩窄受入才能保宝牌，我一般不会保。
