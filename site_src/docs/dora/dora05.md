# 宝牌与手役选择

很多时候，宝牌和手役必须放弃一边。  
那种时候，该怎么决定切牌，就来想一想。

原则上，还是选待牌更多的一边。

这是面子过载时的搭子选择。  
就算去看 234 三色，也非得一路高目、高目地摸进来，最后否则还是只有立直。

拆掉带宝牌的搭子，是非常亏的选择。

像切 <img src="../hai/pin8.gif" width="24" height="34"> 或 <img src="../hai/pin9.gif" width="24" height="34"> 这种会让手变慢的打法，更是论外。

### 理论小结

<span class="green">
不确定的 2 番，通常不如确定的 1 番。
</span>

这一题反而就是切宝牌的一手。  
因为这手已经可以去看断幺九、平和，还有 345 三色，宝牌反而是不需要的。

到底宝牌和手役哪边更优秀，并不能一概而论。  
有时该优先手役，有时该优先宝牌。  
到底选哪边，还是比较受入宽度和和牌容易度来决定。

### 理论小结

<span class="green">
宝牌和手役之间的选择，优先选受入更宽的一边，失误会更少。
</span>

## 序盘时的选择

如果是序盘，压倒性更多的时候，还是应该把宝牌和手役两边的可能都留下来。

<span class="b">例 7</span>
<img src="../hai/man1.gif" width="24" height="34"><img src="../hai/man3.gif" width="24" height="34"><img src="../hai/man3.gif" width="24" height="34"><img src="../hai/man4.gif" width="24" height="34"><img src="../hai/man5.gif" width="24" height="34"><img src="../hai/man7.gif" width="24" height="34"><img src="../hai/man8.gif" width="24" height="34"><img src="../hai/pin5.gif" width="24" height="34"><img src="../hai/pin7.gif" width="24" height="34"><img src="../hai/sou5.gif" width="24" height="34"><img src="../hai/haku.gif" width="24" height="34"><img src="../hai/haku.gif" width="24" height="34"><img src="../hai/ton.gif" width="24" height="34"><img src="../hai/tyun.gif" width="24" height="34">　　宝牌<img src="../hai/sou5.gif" width="24" height="34">

★东 1 局第 2 巡，南家的手牌

这是一手可以去看万子混一色，同时宝牌还浮着的牌。

这时就把宝牌切掉，或者先打出 <img src="../hai/ton.gif" width="24" height="34"><img src="../hai/tyun.gif" width="24" height="34"> ，都不是值得称道的打法。

因为字牌如果重起来，正好能补出混一色所需的搭子，

所以这手如果把 <img src="../hai/ton.gif" width="24" height="34"><img src="../hai/tyun.gif" width="24" height="34"> 先叠在舍牌里，损失太大了。

切 <img src="../hai/sou5.gif" width="24" height="34"> 也太早。  
宝牌一旦重上，打点并不比混一色差；

而如果摸到 <img src="../hai/sou4.gif" width="24" height="34"> 、<img src="../hai/sou6.gif" width="24" height="34"> ，那反而应当放弃混一色。

总的来说，我认为序盘这里比较妥当的选择，是先处理 <img src="../hai/pin5.gif" width="24" height="34"><img src="../hai/pin7.gif" width="24" height="34"> 。

俗话说“两兔相逐，一兔不得”，  
但这句话不一定适用于麻将。  
尤其在序盘，更应该用那种能保留各种可能性的打法。
