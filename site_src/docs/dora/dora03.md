# 宝牌 2 张以上

自己手里有 2 张以上宝牌时，就是机会手。  
应该朝着和牌的最短距离前进。

## 以鸣牌为前提做牌

如果手里是宝牌 2、宝牌 3，那就算鸣牌也还是高打点，  
带着一种<span class="red">“本来就该鸣”</span>的心态去打很重要。
