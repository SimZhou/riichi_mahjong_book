# 牌理与牌效率

这一栏开始从“牌形是什么”进入“具体该怎么切牌”。

如果说第一章是在搭基础，那么这一章处理的就是实战里最常用的核心能力：

1. 找出候选打牌
2. 比较每种选择的受入
3. 理解哪些牌只是看起来多，实际上功能重复

也就是通常所说的`牌理`和`牌效率`。

## 本栏内容

1. [按牌理打牌](./pairi01.md)
: 先建立这整章的总方法，不被役种幻想带偏，而是先从结构上比较“哪张最不需要”。

2. [有效牌与枚数](./pairi02.md)
: 把“枚数越多越有利”这个最基础原则讲清楚，并区分真正的受入和好形变化。

3. [浮牌理论](./pairi03.md)
: 进入更细的单牌功能分析，理解孤张的强弱、筋牌重叠，以及为什么有些牌会互相拖累。

4. [搭子理论（1）](./pairi04.md)
: 进入实战搭子比较，先按受入和好形变化给搭子排优先级。

5. [搭子理论（2）](./pairi05.md)
: 专门讨论两嵌，以及它和双碰、坎张取法之间的取舍。

6. [搭子理论（3）](./pairi06.md)
: 继续看复杂搭子，重点识别“两面 + 坎张”之类容易漏看的复合形。

后续会继续补入：

1. 打牌选择思路
2. 初期牌理
3. 两向听与一向听阶段的牌理
