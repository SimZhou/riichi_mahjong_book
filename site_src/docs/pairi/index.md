# 牌理与牌效率

这一栏开始从“牌形是什么”进入“具体该怎么切牌”。

如果说第一章是在搭基础，那么这一章处理的就是实战里最常用的核心能力：

1. 找出候选打牌
2. 比较每种选择的受入
3. 理解哪些牌只是看起来多，实际上功能重复

也就是通常所说的`牌理`和`牌效率`。

## 本栏内容

1. [按牌理打牌](./pairi01.md)
: 先建立这整章的总方法，不被役种幻想带偏，而是先从结构上比较“哪张最不需要”。

2. [有效牌与枚数](./pairi02.md)
: 把“枚数越多越有利”这个最基础原则讲清楚，并区分真正的受入和好形变化。

3. [浮牌理论](./pairi03.md)
: 进入更细的单牌功能分析，理解孤张的强弱、筋牌重叠，以及为什么有些牌会互相拖累。

4. [搭子理论（1）](./pairi04.md)
: 进入实战搭子比较，先按受入和好形变化给搭子排优先级。

5. [搭子理论（2）](./pairi05.md)
: 专门讨论两嵌，以及它和双碰、坎张取法之间的取舍。

6. [搭子理论（3）](./pairi06.md)
: 继续看复杂搭子，重点识别“两面 + 坎张”之类容易漏看的复合形。

7. [打牌选择思路](./pairi07.md)
: 不再只靠背答案，而是总结出不同向听阶段应该怎样比较牌形、怎样组织判断顺序。

8. [初期牌理（1）](./pairi08.md)
: 进入早巡实战，先处理浮牌较多的手牌，核心是比较“切掉哪张之后损失最小”。

9. [初期牌理（2）](./pairi09.md)
: 继续看早巡的六搭子、五搭子与浮牌并存局面，建立更稳定的初期拆牌习惯。

10. [两向听牌理（1）](./pairi10.md)
: 进入两向听阶段，先处理“一面子一雀头完成，剩三块外加浮牌”的高频形。

11. [两向听牌理（2）](./pairi11.md)
: 处理“一面子完成 + 四搭子”的两向听，重点是弱搭子比较与是否保留判断空间。

12. [两向听牌理（3）](./pairi12.md)
: 继续看四搭子加浮牌、无头形和七对子兼容形，把两向听阶段的主要分支补齐。

13. [一向听牌理（1）](./pairi13.md)
: 进入一向听阶段，先建立‘受入是否合格、是否需要退回两向听’的判断顺序。

14. [一向听牌理（2）](./pairi14.md)
: 处理面子超载形，比较弱搭子、受入重叠与好形变化。

15. [一向听牌理（3）](./pairi15.md)
: 继续看剩余两组搭子的典型一向听，重点是双碰受入该留在哪一侧。

16. [一向听牌理（4）](./pairi16.md)
: 进入无头一向听，说明为什么默认应保留雀头不确定的‘搭子 + 搭子’结构。

17. [一向听牌理（5）](./pairi17.md)
: 继续看无头一向听在受入接近时，如何比较好形变化与最终等待。

18. [一向听牌理（6）](./pairi18.md)
: 处理黏连形一向听，重点转向‘好形听牌枚数’而不是单纯总受入。

19. [听牌时牌理](./pairi19.md)
: 进入听牌后的最终选择，先比剩张，再比改良空间、场况和鸣牌便利性。

20. [向听回退](./pairi20.md)
: 总结什么时候该主动不取听，或为了更好打点与牌型退回两向听甚至更远。
