# 第2章：牌效率与牌理

![牌效率与牌理栏目主图](../images/top-image5.jpg)

1. [按牌理打牌](./pairi01.md)
   牌效率到底是怎样一种思考方式？其中的要诀，在于反复列出可切候选并逐一比较。

2. [有效牌与枚数](./pairi02.md)
   来确认一下麻将的基本原则: “枚数越多越有利”。

3. [浮牌理论](./pairi03.md)
   有时会因为持牌方式不同，导致牌的功能发生重叠。把那些不必要的牌舍掉吧。

4. [搭子理论（1）](./pairi04.md)
   从受入枚数与向好形变化的角度，比较搭子的功能。

5. [搭子理论（2）](./pairi05.md)
   以两嵌的思路为中心，确认两嵌的各种变化形。

6. [搭子理论（3）](./pairi06.md)
   说明实战中也很常见的复合搭子，以及该如何处理它们。

7. [打牌选择的思考方法](./pairi07.md)
   牌效率的练习方法、思考方式、高手到底哪里不同？来接触一下牌理的精髓吧。

8. [初期牌理（1）](./pairi08.md)
   配牌阶段，或者牌局初期，应该如何推进自己的手牌？

9. [初期牌理（2）](./pairi09.md)
   来看看那些搭子很多、还无法清楚想象最终完成形的情况。

10. [两向听的牌理（1）](./pairi10.md)
    掌握接近完成时的两向听牌效率。

11. [两向听的牌理（2）](./pairi11.md)
    继续上次内容。请别忘记始终带着“先处理功能最低的搭子”这个视角。

12. [两向听的牌理（3）](./pairi12.md)
    来确认一些例外发生的牌姿，例如七对子可能性，或没有雀头的情况。

13. [一向听的牌理（1）](./pairi13.md)
    一向听时的牌理，会大幅影响和牌率，我认为这是做牌中最重要的技术。

14. [一向听的牌理（2）](./pairi14.md)
    一向听时的牌效率。把那些频繁出现的形状当作固定模式记下来吧。

15. [一向听的牌理（3）](./pairi15.md)
    养成这样的比较意识: 到底选哪一边，才能更接近和牌？

16. [一向听的牌理（4）](./pairi16.md)
    继续讲一向听的牌理，这一页会涉及无头形的一向听。

17. [一向听的牌理（5）](./pairi17.md)
    接着上一页，继续检验无头形。

18. [一向听的牌理（6）](./pairi18.md)
    这里会讲到黏连形一向听。一向听的牌理非常重要，所以篇幅也比较多。

19. [听牌时的牌理](./pairi19.md)
    听牌时多数情况并没有太多选择，不过还是来看看一些具体案例。

20. [向听回退](./pairi20.md)
    只看效率的话，有时也会出现“为了离和牌更远一步”的向听回退。
