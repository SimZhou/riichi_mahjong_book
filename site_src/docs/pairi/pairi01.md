# 按牌理打牌

所谓牌理，就是为了高效率地完成四面子一雀头而形成的理论。  
这是麻将做手时的根本思路。  
和它意思相近的词，还有“牌效率”。

如果以单局来理解麻将这个游戏，那么它本质上就是:

**“4 个人里，谁能最快完成四面子一雀头”**

的竞争。

因此，基本原则就是按照牌理来选择要切的牌。

麻将手牌有 `速度`、`打点`、`守备力` 这 3 个要素。  
其中最重要的是速度。  
因为速度可以覆盖打点和守备力。

手再高，如果和不了牌，就不可能增加点棒。  
而更快和牌，等于阻止了将来本来可能由对手和掉的那一手，所以可以理解为实际和到的点数之外，还有额外收益。

另外，如果能在对手听牌之前就先和掉，自然也不会放铳。速度正是“手牌的生命”。请务必记住: 被手役吸引了注意力，结果错过和牌，是最不该做的事。

**“表面上的华丽，只不过是在拖慢真正和牌的巡目。”**

这是漫画《阿布雷门》里的一句台词。的确可以说是名言。

前言就先到这里，下面来说说“按牌理打牌”到底是什么意思。

像这样思考之后，麻将里“该切什么牌”其实往往并不会太难得出答案。  
先分析手牌，用排除法找出应该切的候选牌，  
之后再把这些牌互相比较就行了。

最重要的是**“比较”**。只要会比较，切错牌的次数就会大幅减少。

对于这副手牌来说，没有被面子和搭子使用到的  
<img src="../hai/man9.gif" width="24" height="34" /><img src="../hai/sou5.gif" width="24" height="34" /><img src="../hai/pin3.gif" width="24" height="34" /><img src="../hai/tyun.gif" width="24" height="34" />  
就是要切的候选牌。

把握住这些牌各自有什么功能，再去思考哪一张最不需要。

**反复列出弃牌候选，再逐一比较。**  
这就是所谓按牌理打牌，也是麻将的基本。
