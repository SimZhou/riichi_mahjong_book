# 两向听牌理（1）

先从“比较浮牌”的模式讲起。

适用于这种手牌：
已经完成 1 组面子和 1 个雀头，另外还有 3 个搭子块（包含对子块）。  
完成面子当然不能拆；而如果雀头只有 1 个，去拆它同样会让手牌大幅变慢。  
因为这两种都会直接退回三向听。

所以，基本做法是先把浮牌列成候选，再在这些浮牌里比较。

### 理论

<div class="green" markdown="1">
“一面子完成 + 一雀头完成 + 三搭子”的两向听，  
基本上还是从剩下的浮牌里选择最不要的那一张。  
不过有些手牌，反而应该把 3 张搭子块拆回单搭子，  
从而把所有浮牌都保留下来。  
其中绝大多数，都出现在 3 对子形里，用来决定保留哪组两面。
</div>
