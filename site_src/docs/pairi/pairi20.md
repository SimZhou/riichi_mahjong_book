# 向听回退

先说结论。  
把向听数打回去之后，最后反而更快和牌的情况，其实并不算多。

其中大多数，都是为了去看断幺九、混一色之类的手役。  
因为之后可以鸣牌，所以把副露考虑进去以后，结果上反而会更快。  
另外，还有一种情况是：当前待牌几乎已经没剩多少了，  
比如坎张待又被碰掉之类，这时也会需要把向听打回去。

所以基本上可以这样理解：

**向听回退，本质上是一种为了提高打点而使用的技术。**

当然，追求到听牌的最短距离，本来就是牌理的基础。  
但把和牌难度和打点也一起考虑进去的话，  
并不是每一次“向听数推进”，就一定是正解。  
一边在速度和高度之间取平衡，一边打牌，才是更重要的事。
