# 附录

这一部分整理的是原站首页之外的附加内容，包括作者介绍、麻将链接集，以及原站首页侧栏里提到的书目与外部资源。

## 附录目录

1. [作者介绍](./prof.md)
   原站 `prof/` 页的翻译，整理了作者和站点管理者的基本信息。

2. [麻将链接集](./link.md)
   原站 `link/` 页的翻译，包含麻将战术站、博客、职业团体和免费游戏等外部链接。

3. [站外推荐资源](./resources.md)
   把原站首页侧栏里的书单、讲座、杂志、雀庄和线上麻将资源集中整理成一页。

4. [全站目录](../sitemap/index.md)
   对应原站 `sitemap/`，按栏目列出整套教程页面。

## 使用建议

如果你是第一次来，建议先从[首页](../index.md)进入正文教程。

如果你想找的是：

1. 作者和站点背景，去看[作者介绍](./prof.md)
2. 外部麻将站点索引，去看[麻将链接集](./link.md)
3. 原站推荐书单和站外资源，去看[站外推荐资源](./resources.md)
