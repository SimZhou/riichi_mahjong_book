    一向听牌理（2）

    先来看面子过剩形。
    如果前面那些基础内容已经掌握到位，这一页的正着打原则上都应该能推出来。

    例 1
    摸

    => 切 1m

    两面待比嵌张待更有利，这是常识。
    而且摸进 时还会出现三面张变化，所以切牌顺序的基本原则是 1m -> 3m。

    例 2
    摸

    => 切 2s

    这是嵌张和嵌张之间的比较。
    的双重受入不利，所以这里应打 。

    例 3
    摸

    => 切 3p

    去追一杯口，受入枚数会变少；就算把打点算进去，也还是不利。
    这里的正解，仍然是嫌弃 的双重受入。

    例 4
    摸

    => 切 1m

    两边的受入枚数同为 12 张，最后留下的也都是嵌张，差别不在这里。
    既然如此，就该比较好形变化有多少。

    万子的好形变化，只有摸进 形成两嵌这一种。
    而筒子的变化则有 ，明显丰富得多。
    （双碰变化两边相同，所以这里不算进去）

    虽然坎 和坎 的和牌难度多少有点差别，
    但从正统牌理来说，还是应当看重变化数量，把万子落掉。

    例 5
    摸

    => 切 2s

    这是两组两面搭子的选择。

    不能因为自己已经用掉了 1 张 ，就误以为 那边比较差。
    那样看，会漏掉牌形本身的变化。打 之后，索子可以变成“两面 + 嵌张”的复合形。

    ☆ 摸 即听牌

    例 6
    摸

    => 切 4m（或 5m）

    乍看之下，万子、筒子、索子似乎都能切。
    但如果落的是万子，那么“反向摸进”的 依然会成为有效牌。

    ☆ 摸

    如果场况上没有特别差别，正解就是先落 。
    这是比较细的点，看漏了也不至于造成大问题；
    但“对反向摸进的补救”这种想法，最好还是记在脑子里。
