    打牌选择思路

    到这一页，第二章前半的零件已经讲得差不多了。接下来要回答一个更实际的问题：

    高手为什么能很快选出最优切牌？

    原页给出的答案很准，主要就两点。

    原因 1：高频牌形已经模式化

    高手并不是每次都从零开始算。

    很多高频形，尤其是：

    1. 一向听常见形
    2. 面子超载常见形
    3. 二度受け常见形
    4. 复合搭子常见形

    他们已经在脑中模式化了，看到就能直接排除明显错误。

    这件事的价值主要在于：

    1. 减少失误
    2. 提高决策速度

    但这不是全部。

    原因 2：他们有更好的比较方法

    真正更重要的，是高手会比较。

    因为现实里不可能每一手都真的数出：

    1. 切 A 有几种多少枚
    2. 切 B 有几种多少枚
    3. 切 C 又有多少好形变化

    可即便不逐张死数，依然可以靠一套稳定的比较顺序，快速筛出最好的答案。

    这也是这整章真正想训练你的东西：

    不是把答案死背完，而是让你在陌生牌形里也能自己推到答案。

    不同向听阶段，比较重点不同

    原页把牌理拆成四个阶段，这个划分很实用：

    1. 三向听及以下
    2. 两向听
    3. 一向听
    4. 听牌

    之所以这样分，是因为不同阶段最该优先看的量不一样。

    三向听及以下：先看“切掉后损失什么”

    在很远的阶段，手牌还没定型，孤张、浮牌、弱搭子会同时存在。

    这时不必执着于把“下一步一定降向听”算得特别细，而更适合用一种更快的比较法：

    切这张后，会损失什么？

    如果损失极小，就说明它大概率该先走。

    这个阶段里：

    1. 细微效率差距往往没那么关键
    2. 留住役种可能、留住将来守备，也会开始有意义

    所以它不是“完全不看效率”，而是没必要为一两枚差距过度用脑。

    两向听：开始看下一步，也要看下一阶段

    到两向听时，不能只看“谁更快进一向听”，还要看：

    进了一向听之后，牌会不会很薄。

    原页强调的一点很重要：

    1. 越接近听牌，受入通常越少
    2. 真正最难的一关，常常是“一向听 -> 听牌”

    所以两向听时，如果某条路线虽然少一两枚一向听受入，却能换来更厚的一向听质量，这种交换经常是值得的。

    一向听：听牌受入最重要

    到一向听时，优先级会收得非常明确：

    1. 先比听牌受入
    2. 若相同，再比好形变化
    3. 若仍接近，再看打点和最终待型

    这里一两枚差距就已经很关键，因为你离听牌只差一步，少掉的每一枚都会更真实地影响和牌速度。

    听牌：先看枚数，再看好不好和

    听牌阶段最重要的当然是和牌牌数，但还要再补一层现实判断：

    1. 这张牌是不是容易被打出来
    2. 单骑有没有明显手变
    3. 同枚数时，哪边更容易形成高打点或更好待张

    不过顺序不能乱。默认仍然是：

    先看枚数，再考虑“出不出来”和“手变多少”。

    这一页的结论

    1. 高手判断快，主要靠“高频形模式化”和“比较方法稳定”
    2. 牌理训练的目标不是死背答案，而是学会自己比较
    3. 三向听及以下，适合先看“切掉后损失什么”
    4. 两向听时，要同时看“进一向听的速度”和“一向听的厚度”
    5. 一向听时，听牌受入最重要；听牌时，则先看和牌枚数
