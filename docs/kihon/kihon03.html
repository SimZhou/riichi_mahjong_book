    听牌与向听

    要把麻将打明白，先得弄清楚两个最常见的词：

    1. 听牌
    2. 向听

    它们描述的都是你离和牌还有多远。看不清这一点，后面的牌效率就没法判断。

    麻将的和牌形

    标准和牌形主要有三类：

    1. 四组面子加一组雀头
    2. 七对子
    3. 国士无双

    真正打牌时，绝大多数情况都在处理第一类，也就是“四面子一雀头”。

    面子又分成两种：

    1. 顺子：连续三张数牌组成的一组
      　　……连续三张数牌
    2. 刻子：三张相同的牌组成的一组
      　　……三张相同的牌

    把这件事说透，后面很多概念就顺了：麻将不是“随便凑一副看起来差不多的牌”，而是一步一步把手牌推进到固定的和牌结构。

    和牌形有哪些样子

    四面子一雀头看起来固定，但内部组合并不一样：

    1. 四面子一雀头

    1. 1-a 四组顺子
    2. 1-b 三组顺子加一组刻子
    3. 1-c 两组顺子加两组刻子
    4. 1-d 一组顺子加三组刻子
    5. 1-e 四组刻子

    特殊和牌形也要知道：

    2. 七对子

    3. 国士无双

    这些和牌形的成形难度并不一样。

    其中最难做成的当然是国士无双，而它的回报也最大，对应的是役满这个最高打点级别。

    另外，严格来说面子里还包括四张同牌组成的杠子；这里只先不把它和刻子分开讲，后面再单独处理。

    七对子和国士无双都属于特殊和牌形。学习牌效率时，默认仍然先用普通手的“四面子一雀头”思路来理解。

    什么叫听牌

    只差最后一张牌就能和牌，叫做听牌。

    例 1：

    这手只要再来 或 就和牌。

    听牌是实战里的关键节点：

    1. 没听牌就不可能和牌
    2. 门清听牌后才能立直
    3. 越早进入听牌，通常越容易抢到先手

    什么叫向听

    离听牌还差几步，用向听数表示。

    例 2：

    这手只要摸到 、 或 就能进入听牌，所以它是一向听。

    也就是说：

    1. 0 向听 = 已经听牌
    2. 1 向听 = 再进一步就听牌
    3. 2 向听 = 还要两步

    后面依次类推。

    能把手牌当前是几向听看清，是麻将里最基础的判断能力。

    同样是一向听，质量也差很多

    例 3：

    这一手也是一向听，但受入非常多，原页给出的结论是 13 种 42 枚。和上面的例 2 相比，进入听牌的速度差距非常大。

    所以实战里不能只说“我是一向听”，还得继续问：

    1. 这手一向听的受入有多少
    2. 是不是容易进入听牌
    3. 进入听牌后会是什么牌型

    最远会差到多少

    例 4：

    这种杂乱起手，最短也要六步才能到七对子听牌，因此是六向听。这也是常见手牌里距离听牌最远的级别。

    这个知识点本身不是实战重点，但能帮助你建立一个整体感觉：向听数越大，离成牌越远，做牌空间也越大。

    打牌时要优先降低向听

    例 5： 自摸

    如果只因为 在边张、看起来孤立，就先把它切掉，手牌仍然停留在两向听。

    正确方向是先找能把手牌推进到一向听的切法，也就是切 或 。

    麻将正常做牌就是这样一步一步推进：

    1. 两向听推进到一向听
    2. 一向听推进到听牌
    3. 听牌再去争取和牌

    有些局面会为了打点、防守或手役而暂时不降向听，但那是后续的进阶判断。基础阶段必须先把“优先推进向听”这个主线养成。

    这一页的结论

    1. 麻将的主流和牌结构是四面子一雀头
    2. 听牌就是只差最后一张
    3. 向听数表示离听牌还有几步
    4. 同样是一向听，受入多寡会让质量差很多
    5. 基础做牌时，优先让手牌往更低向听推进
