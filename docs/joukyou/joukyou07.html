    最终局战术（1）

    最终局是最考验局势判断力的一局。

    首先，必须准确掌握“和多少点能到第几名”。

    自摸时会缩小的点差

    如果是亲家，和牌以后再看“咦，翻过去了吗”也还来得及；

    但如果是子家，就必须在听牌那一刻先知道这手是不是逆转手。

    和了点 对方是子家 对方是亲家
    300 / 500 1400 1600
    400 / 700 1900 2200
    500 / 1000 2500 3000
    700 / 1300 3400 4000
    1000 / 2000 5000 6000
    1300 / 2600 6500 7800
    2000 / 4000 10000 12000
    3000 / 6000 15000 18000

    尤其是 1300 / 2600 这种条件很常见，

    最好把亲家和子家各自能缩小多少点差直接背下来。

    另外，还有一点很重要，就是满贯自摸时能缩小多少点差。

    “如果满贯自摸就能追上，那就应该以逆转为目标。”

    这条最好当作定理记住。

    满贯直击也好、跳满自摸也好，都不是那么容易主动做出来的。

    “满贯自摸条件”，可以看成现实可争的逆转边界。

    反过来说，关键就在于要把最终局打成自己处在满贯自摸圈内，如果是头名则要打成满贯自摸圈外。

    逆转的手作

    虽然是最终局，也没必要莫名其妙地过度紧张；

    但如果逆转手已经来了，就不能白白浪费机会。

    例 1

    如果是平场，我觉得切 、重视速度也可以。

    但如果眼下自己是末位，条件又是“满贯自摸到二位，运气好甚至跳满能到头名”，

    那就只能打 。

    因为这手已经具备了足以去争的手材料。

    不过，也不是说它一定能顺利做成三色。

    如果最后只是听成断幺九，那就该即立直，

    去赌一发、赤牌、里宝牌这些现实增益。

    例 2

    这一题很简单。

    和头名只差 4300 点。

    再加上场上有立直棒和一本场，3900 点从谁那里和都能逆转。

    因此就该把宝牌切出去，

    有能鸣的牌就鸣，直接取听。

    这里不是去考虑断平三色一杯口之类的时候。

    鸣牌判断

    如果是和了就头名的局面，那就只重视速度即可。

    后付也好，食断也好，什么都可以。

    例 3 　　打出

    比如说，最终局里自己和二位只差一点点，无论如何都想守住领先。

    像这种场面，就应该不顾牌形，直接奔着和牌去。

    所以上家打出 时，就是鸣牌一手。

    例 4

    最终局和头名相差 4400 点。

    条件是 1000 / 2000 自摸，或者 3900 直击。

    因此，这手的主题就是做出 3900，

    更准确地说，是做成白宝牌 2。

    这样一来，这张 就可以看成急所牌。

    在所有人都参与的最终局里，就算只是役牌后付去副露，被完全停住的可能性也不高。

    而且下家也已经副露，时间上没有余裕。

    虽然出和 3900 还不够，

    但也不能轻视摸进赤牌，或对家、亲家再放出立直棒这类偶发因素。

    这里我认为果断吃牌更好。
