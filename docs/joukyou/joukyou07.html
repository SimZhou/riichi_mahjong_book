    最终局战术（1）

    最终局最考验局势判断。

    第一步必须先算清：和多少点能到第几名。

    自摸时会缩小多少点差

    子家时，不能等和了以后再看“有没有翻过去”，而是必须在听牌时就知道这手是不是逆转手。

    下面这张表，就是原页给出的子家自摸缩点表：

    和了点 对方是子 对方是亲
    300 / 500 1400 1600
    400 / 700 1900 2200
    500 / 1000 2500 3000
    700 / 1300 3400 4000
    1000 / 2000 5000 6000
    1300 / 2600 6500 7800
    2000 / 4000 10000 12000
    3000 / 6000 15000 18000

    尤其是 1300 / 2600 这种很常见的条件，最好直接背下来。

    另外还有一条很重要的经验线：

    若满贯自摸就能逆转，就应该把它视为现实可争的逆转局。

    因为满贯直击、跳满自摸之类条件，并不是每次都能主动设计出来；“满贯自摸圈”通常就是最现实的逆转边界。

    反过来说，自己若是头名，也应尽量以“不进入满贯自摸圈”为目标来打最终局。

    逆转的手作

    最终局并不需要莫名其妙紧张，但若逆转手真的来了，就绝不能白白浪费。

    例 1

    若是平场，这题切 5s 走速度线也没问题。

    但现在自己垫底，而且条件是：

    1. 满贯自摸能升到二位
    2. 运气好甚至跳满可以争头名

    这种场面当然只能打 2p 3p，去追大路线。

    因为这手已经有足够的材料可以争。

    当然，也不代表一定能顺利做成三色。

    若最后只是听成断幺九，那就应即立直，去争一发、赤牌、里宝牌这些现实增益。

    例 2

    这题就简单得多。

    和头名只差 4300，加上立直棒和一本场，3900 从任何人那里和到都够逆转。

    因此这里就是：

    1. 直接把宝牌切飞
    2. 尽量扩大可鸣范围
    3. 只求最快听牌、最快和牌

    像断幺九、平和、三色、一杯口这种平时会去想的路线，这里都不重要。

    鸣牌判断

    若是和了就头名、和了就守位的局面，鸣牌判断就只该服从速度。

    后付也好，食断也好，只要能更快和牌，都可以接受。

    例 3

    4m 5m 1p 2p 3p 5p 5pr 8p 8p 9p 6s 7s 8s 出 4p

    若你是二位且只比三位高一点点，目标就是别被反超，那么这时很多平时不会鸣的牌，都应该鸣。形状是否好看、役是否漂亮，都不再重要，重要的是：

    1. 尽快成牌
    2. 尽快结束牌局
    3. 别给后家更多摸牌回合

    所以若上家打出 4p，这里就是鸣牌一手。

    例 4

    最终局里和头名差 4400。

    条件是：

    1. 1000 / 2000 自摸
    2. 或 3900 直击

    因此这手的主题，就是做出 3900，更准确地说，是做 白 + 宝牌 2。

    那么这张 1s 就可以视作急所牌。

    全员都在拼的最终局里，就算只是后付役牌，别人也未必真能把它完全停住。

    而且下家也已经副露，时间并不宽裕。

    虽然表面上看，出和 3900 还不够，但也不能忽视这些偶发要素：

    1. 摸进赤牌
    2. 对家或亲家再放一根立直棒

    这些都可能让条件刚好达成。

    所以这里应果断吃牌。

    这一页的结论

    1. 最终局的第一步永远是先算条件。
    2. 子家要熟悉自摸缩点表，至少把常见条件背熟。
    3. 若满贯自摸就能逆转，应把它当成现实可争的边界。
    4. 逆转手来了，就别再按平场速度线把它压扁。
    5. 若任何人 3900 都够逆转，就应把速度放到绝对优先级。
    6. 最终局的鸣牌判断，也必须完全服从逆转或守位条件。
