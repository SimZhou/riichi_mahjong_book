    状况与做牌（1）

    这一页原页先从“巡目”切入。意思很明确：

    同一副手牌，序盘、中盘、终盘的目标并不一样。

    若还用同一套固定牌理打到底，就会在该追打点时只顾速度，在该保听牌时还贪改良，在该顾安全时还硬把牌往最宽处凑。

    第一部分：先按巡目，把手作目标分成三段

    原页把巡目简单分成三段，这个框架很实用：

    1. 序盘 1-6 巡：优先保留役种可能与好形变化
    2. 中盘 7-12 巡：开始把“尽快进听”放到更高优先级
    3. 终盘 13-18 巡：若牌姿已不适合正面作战，就应提高安全与形式听牌比重

    这不是死规则，而是一个重心变化：巡目越晚，你能期待的摸牌次数越少，原本那些“以后再变好”的路线，价值就会快速缩水。

    第二部分：序盘可以多追一点打点与手役

    原页第一个例子讲的是：若现在还在序盘，即使有一条更稳、更容易立直的平直路线，也可以适当保留更高打点的变化。

    也就是说，序盘时不一定要一味往“最快立直”收缩。因为：

    1. 后面摸牌轮数还很多
    2. 役种和好形改良仍有实现空间
    3. 过早把牌压成纯速度路线，反而可能错过本来能做大的手

    所以原页在这一段要表达的不是“打点永远比速度重要”，而是：

    序盘时，打点与好形变化还没有失去时间价值。

    第三部分：到了中盘，很多“不取听”要开始收手

    第二个例子是这一页里非常关键的一段。原页承认，从静态牌理看，这副牌的本手仍是先不取当前听牌，保留大量好形变化；但它马上补了一句：若已经进入中盘，或者相关门的关键牌已经明显变薄，这些“手变期待值”就不再像序盘那么高。

    于是判断会转成：

    1. 若改良路线仍很丰富，就继续养牌
    2. 若改良路线已经因为巡目或见牌数而明显缩水，就应先把当前听牌拿住
    3. 有时甚至该直接立直，不再拖延

    也就是说，中盘开始后，你要不断重算：

    1. 这副牌还能不能合理期待改良
    2. 还是该把现成的听牌先兑现

    第四部分：中盘的两向听，守备价值会开始压过单纯广度

    第三个例子讲的是另一种很常见的误区：只看牌效率，会觉得某张安全牌对子显然应该先拆，因为这样受入最宽；但原页提醒，若已经到中盘以后，而且这手又没有宝牌、整体打点也不高，那么盲目把牌做得更广，不一定值得。

    因为这时还要看另外一件事：

    别人随时可能先发动攻击。

    所以原页在这里建议：

    1. 保留现成安全牌
    2. 不必为了多扩一点两向听受入，就把退路也一起拆掉

    这不是单纯保守，而是承认一件事实：巡目变晚后，手牌广度的价值在下降，安全退路的价值在上升。

    第五部分：终盘时，形式听牌的价值会明显上升

    原页后半段转到终盘主题：到了第三排附近，很多时候已经不是“能不能和出”优先，而是“能不能保住听牌流局收益”。

    第四个例子讲得很直接。平时为了做平和，你当然会选更标准的那一打；但若目标已经变成“只要形式听牌就够”，那答案就会改成更容易鸣牌、更容易快速凑听的路线。

    原页这里其实是在说：终盘时，和牌价值下降，形式听牌价值上升，因而：

    1. 更容易被三家碰到的对子
    2. 更容易从上家吃进的搭子
    3. 更容易硬凑成听牌的结构

    都会比平时更有价值。

    第六部分：h007.png 这张图讲的是终盘维持形式听牌的技巧

    这一页唯一的大图就是最后那个终盘例子。

    图里的局面是：

    1. 他家已经立直
    2. 你在终盘取得了形式听牌
    3. 上家打出一张可以鸣的牌
    4. 手牌里又留着场上已见两枚的 中

    原页想强调的不是“这张牌能不能胡”，而是：

    应当先鸣牌，把形式听牌维持得更稳。

    理由在于：

    1. 当前本来就没法荣和
    2. 若直接跳过，下一摸可能被迫切危险牌而掉听
    3. 先吃进去，再切 中，等于多给自己留了一次安全缓冲
    4. 下一巡即使摸进危险牌，仍可以靠剩下的 中 继续维持听牌

    这就是终盘局势判断里非常实战的一点：

    当目标已经从“争和”变成“保听”时，维持听牌结构本身就是收益。

    这一页的结论

    1. 巡目会直接改变同一副手牌的最优路线
    2. 序盘可以更积极保留打点与好形变化
    3. 中盘开始后，要把“现在先取听”放到更高优先级
    4. 终盘若正面和牌希望不大，就应提高安全和形式听牌比重
    5. h007.png 说明的重点是：终盘形式听牌不只要拿到，还要想办法维持住
