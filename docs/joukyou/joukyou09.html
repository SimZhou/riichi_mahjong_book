    最终局战术（3）

    这一页用几个特殊例子收尾。

    当然，实战里不太会刚好碰到和例题完全一样的局面，但重要的是把思路学会。

    原页要表达的是：

    南四最重要的不是“标准打法”，而是根据名次条件去选择最有利的风险结构。

    有些平场里明显该取听、该和牌、该维持听牌的局面，到最终局都可能反过来。

    第一部分：h100.gif 说明，最终局有时要主动拆掉现成听牌

    第一张图讲的是“听牌不取”的最终局应用。

    平场里，这种手很多人会直接切宝牌立直，先争即时胜负。但原页把点差条件一摆出来，结论就变了：

    1. 就算满贯自摸也拿不到第一，必须跳满自摸
    2. 另一方面，若贸然把宝牌打出去，一旦被上家满贯直击，自己还会掉到第三

    也就是说，这里不是“立直了也不错”，而是：

    1. 当前听牌的收益上限不够
    2. 当前听牌的风险却很高

    所以原页的好手是反手拆听，切 5m。

    这样一来，宝牌两张都能真正用上；若索子那边做出雀头，也还保留着跳满自摸的可能。

    这手最关键的不是听牌本身，而是：

    在最终局，现成听牌若既不够逆转、又会显著放大掉位风险，就该敢于主动拆掉。

    第二部分：h101.gif 与 h102.gif 讲的是最终局的见逃

    原页第二组例子讲的是“见逃”，而且是副露手里的战略见逃。

    先看起点图：

    当前你和头名只差 2400，这张 3s 当然要鸣，因为：

    1. 你仍需要尽快成牌
    2. 这手还带着 678 三色希望

    之后演变到第二张图：

    这时你已经听到一手 2000 点，但现在从对面打出的和牌，仍不足以让你夺冠。原页在这里明确说：

    这张牌不能和。

    理由非常直接：

    1. 自摸可以逆转
    2. 直击特定家也可以逆转
    3. 但现在这家点和不够
    4. 若为了“先胡再说”就收下这 2000，反而等于自己放弃第一

    所以最终局的见逃不是炫技，而是纯粹的条件服从：

    只要当前和牌不满足你的目标名次，而继续进行仍有现实逆转空间，就该见逃。

    第三部分：h103.gif 说明，头名甚至可以主动选择“听牌却报未听”

    第三个例子讲的是很多人不熟的最终局技巧：诺听宣言。

    原页给的逻辑很强：若你是头名，而就算流局时自己报未听，也仍然能稳住第一，那么：

    1. 你不必执着维持这手无役听牌
    2. 甚至可以主动在流局时选择未听结束比赛

    原页还给了一条非常实用的经验线：

    1. 流局一人未听时，点差最多缩 4000
    2. 所以只要你领先超过 4000
    3. 就可以较早开始往“流局结束”方向设计

    图里的局面就是这样。虽然你现在勉强维持着无役听牌，但上家已经连续打现物，看起来大概率已经在弃和。此时继续强行维持听牌，风险反而可能大于收益。

    所以原页的判断是：只要上家最后那张不是明显危险牌，就可以直接报未听结束。因为相比“继续打下去被人逆转”，这点流局缩分是更可控的结果。

    第四部分：h104.gif 说明，南四大领先时，甚至可以帮别人和掉亲家

    最后一个例子讲的是最终局里最不像“标准麻将”的一类动作：鸣牌协助与差入。

    原页设定的是大幅领先的头名局面。此时你真正怕的，不是别人用小和追分，而是：

    1. 亲家连庄
    2. 亲家继续多摸多打
    3. 原本稳定的头名被一局局磨掉

    而图中下家的副露，看起来并不是逆转手，更像是一副“只想保住第二”的小和手。若如此，你的最优策略反而可能是：

    1. 主动打出中张牌
    2. 让下家更快把断幺九之类的小副露手做成
    3. 尽快把亲家的庄位打掉

    原页这里的意思不是“见副露就喂”，而是：

    若你已经确定某家小和并不会伤害你的名次，反而能帮你终止最危险的亲连庄，那就可以把他视作临时盟友。

    这类判断非常依赖点差理解，但一旦看懂，能明显提高最终局的收官能力。

    这一页的结论

    1. 最终局有时必须主动拆掉现成听牌，只因为它既不够逆转、又会放大掉位风险
    2. h100.gif 说明：最终局的听牌价值，要同时看上限是否够、风险是否值得
    3. h101.gif 与 h102.gif 说明：若当前和牌不满足目标名次，而继续有现实逆转空间，就应见逃
    4. h103.gif 说明：头名若即使报未听也能守住第一，可以主动选择未听结束比赛
    5. h104.gif 说明：大领先头名在亲家连庄最危险时，甚至可以通过喂给别家小和来帮助终局
