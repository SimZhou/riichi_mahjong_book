    读立直（2）

    上一页讲的是“哪些牌特别危险”。这一页原页开始反过来讲：

    如何从立直前的舍牌顺序里，找出那些比表面看起来更安全的牌。

    这里的重点不再是猜待牌，而是利用牌理反推：若对方真在等某一类形，那他往往不会用这种顺序舍牌。

    第一部分：若手出立直前一张是安全牌，它前后的跨筋往往更安全

    原页先讲的是一个很实用的模式：

    1. 对手立直前最后手打出的是一张安全牌
    2. 那么这张牌前后的某些相关牌，往往并不参与最终待牌

    原因在于比较会打的人，通常不会在听牌前故意额外抱一张无用安全牌，也不会为了留安全牌去主动削减大量有效进张。也就是说：

    1. 若他手打安全牌立直
    2. 往往表示这张安全牌是“自然多出来的”
    3. 那它前一张、前一组相关搭子，通常已经与最终待牌脱钩

    原页第一个例子里，结论是：

    1. 8m 9m 几乎可以当成强安全候选
    2. 1p、9s 也比普通无筋更容易通过

    这背后的逻辑，不是玄学，而是牌理约束：若对手真保留着那些相关的双碰、坎张或两面，他往往不会用这种“先把安全牌手打出去再立直”的方式整理手牌。

    原页还补了一条限制：如果场上同种关键牌已经大量 visible，例如 7m 已经见了两张，那这套推理的可信度会明显下降。因为这时原本应该保留的受理本身就可能已经不成立。

    第二部分：若立直后又有关键中张通过，前一张的跨筋会进一步变成强安全候选

    原页第二个例子是在上一种模式上再加一条信息：

    1. 立直前最后打的是安全牌
    2. 立直后某张相关中张又顺利通过

    这时，前面原本只是“比较安全”的牌，往往会升级成更强的安全候选。

    原页用的就是 8m 这种情况。通常来说，一张牌就算某个中张通过了，仍可能命中双碰或坎张，所以不该直接断成绝对安全。但如果你已经先确认：

    1. 对手是手打安全牌立直
    2. 这意味着他前面的搭子结构大概率不保留相关待牌
    3. 之后又有关键中张通过

    那很多原本还留着的双碰、坎张可能性也会一起被削掉。

    原页还顺手补了一点：像 1m 和 7m 之间的中筋 4m，在这种结构下也会比平时更安全。因为既然能判断相关坎张不太成立，那中间那张自然也跟着降危险度。

    所以这一段真正该记住的是：

    “安全牌手打立直”本身就是高价值信息，别只看现物，要继续往前后结构读。

    第三部分：坎张落立直时，若出现逆切，反向那条筋往往更安全

    原页接着讲的是另一类非常有用的安全读法：

    坎张落立直的逆切筋。

    例子里看似只是普通的 5s -> 7s 切牌后立直，但原页指出，这种顺序会强烈暗示：

    1. 这不是标准两面固定
    2. 更像是某种坎张落处理
    3. 因而反向那条筋，例如 3s，会比普通无筋安全很多

    为什么？

    因为若对方真是普通两面形，例如保有 4557 这类结构，他大概率会先切 7s 保留双碰或更宽的受理，而不是先切 5s 再切 7s。既然切序不是这样，就说明单纯的 3s-6s 两面待被保留的概率不高。

    原页把这一类结论总结得很直接：

    坎张落立直里，逆切出来的那条筋，多数只剩三面张这类少数形会命中，因此安全度会明显提高。

    第四部分：但手打和摸切一定要分开看

    原页在这里专门提醒一个很重要的实战分界：

    1. 如果逆切是手打
    2. 那上面的读法可信度高
    3. 如果关键一张是摸切
    4. 这种安全判断就会立刻变弱

    原因很简单。摸切意味着那张牌未必反映原本的固定整理顺序，它可能只是：

    1. 临时摸来的无用牌
    2. 听牌前刚好摸到就切了
    3. 或者伴随别的受理变化一起发生

    所以原页特地强调：会看手打与摸切的人，才能真正把这种安全读法用到实战。只看牌面、不看切牌方式，这类细节优势基本用不出来。

    第五部分：还可以继续用牌理去否定双碰待与坎张待

    原页最后一段又补了两种“否定法”。核心仍然一样：

    从“正常人不会这样拆两面”这个牌理出发，排除某些危险形。

    否定双碰待

    比如某个立直若想把 5p 或 7p 做成双碰待，就意味着对手曾经得把本来能保留为两面的一组牌，硬拆成对子受理。这不符合常规牌理。

    所以原页给出的判断是：

    1. 5p、7p 仍可能命中坎张
    2. 但作为双碰待，可以先基本排除

    这类判断单独看，未必能直接把牌认成安全牌；但若再叠加：

    1. 现物
    2. 无机会
    3. 场上张数

    就常常能把它进一步推成真正可打的牌。

    否定坎张待

    另一种则是：如果要让某张牌成为坎张待，意味着对手必须先把原本更自然的两面受理主动拆掉。原页认为，在正常手役和正常打点追求下，这种事并不常见。

    所以像例子里那种牌，即使表面看仍是无筋，其实很多常见坎张都已经能靠牌理直接否定。

    第六部分：宝牌附近通常危险，但也能反过来读出例外安全牌

    原页最后举了一个很实战的小技巧：宝牌附近的牌通常危险，这是大原则；但若某张待牌会明显降低打点，尤其是会放弃宝牌周边价值时，也能据此反推出另一张牌反而比较安全。

    原页的意思不是要你和“宝牌附近危险”对着干，而是说：

    1. 多数人不会为了听牌，故意把明显更高打点的待牌路线放掉
    2. 所以有些看起来很危险的牌，实际上反而因为“太不合算”而不常成为真实待牌

    这类读法不能单独当绝对安全依据，但和前面的切序信息叠加后，经常能把原本不敢打的牌，变成相对可靠的退路。

    这一页的结论

    1. 读立直时，除了找危险牌，更重要的是主动找那些被舍牌顺序否定掉的危险形
    2. 若对手是手打安全牌立直，那张安全牌前后的相关跨筋常常会比普通无筋安全得多
    3. 坎张落立直若出现逆切，反向那条筋通常只有少数高阶形会命中，因此安全度更高
    4. 手打和摸切必须分开看，很多安全读法一旦换成摸切，可信度就会明显下降
    5. 双碰待、坎张待都可以用“正常不会主动拆两面”这个牌理去做否定，再结合无机会和场上张数继续缩小危险区
